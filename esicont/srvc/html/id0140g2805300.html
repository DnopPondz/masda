การตรวจสอบเซ็นเซอร์กระแสไฟฟ้า [SKYACTIV-G 2.0, SKYACTIV-G 2.5]

id0140g2805300


การตรวจสอบรหัสวิเคราะห์ปัญหา

1.  ต่อ M-MDS เข้ากับ DLC-2

2.  ทำการตรวจสอบรหัสวิเคราะห์ปัญหาโดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

•  ถ้ารหัสวิเคราะห์ปัญหาใดๆ ที่เกี่ยวข้องกับเซ็นเซอร์กระแสไฟฟ้าปรากฏขึ้น ให้ซ่อมตำแหน่งที่บกพร่องตามการค้นหาสาเหตุปัญหาของรหัสวิเคราะห์ปัญหานั้นๆ (ดูที่ ตารางรหัสวิเคราะห์ปัญหา [SKYACTIV-G 2.0, SKYACTIV-G 2.5])


การตรวจสอบด้วยสายตา

ข้อควรระวัง
•  ระมัดระวังอย่าสัมผัสสายขั้วบวกแบตเตอรี่
•  ถ้านัตของเซ็นเซอร์กระแสไฟฟ้าหลุดหลวมขณะที่สายขั้วลบแบตเตอรี่ต่ออยู่กับแบตเตอรี่ แรงบิดที่มากเกินไปจะจ่ายไปที่สายขั้วลบแบตเตอรี่และขั้วแบตเตอรี่จะไ้ด้รับความเสียหาย เมื่อถอดสายขั้วลบแบตเตอรี่และเซ็นเซอร์กระแสไฟฟ้า ระมัดระวังอย่าให้ขั้วแบตเตอรี่เสียหาย
•  ต่อสายขั้วลบแบตเตอรี่ก่อนต่อขั้วต่อเซ็นเซอร์กระแสไฟฟ้า ถ้าต่อขั้วต่อเซ็นเซอร์กระแสไฟฟ้าก่อน PCM อาจจดจำสัญญาณผิดพลาดจากเซ็นเซอร์กระแสไฟฟ้าและบันทึกสภาวะแบตเตอรี่อย่างไม่ถูกต้อง
•  การตรวจสอบความแน่นในขณะที่จับยึดเซ็นเซอร์กระแสไฟฟ้าอาจจ่ายแรงที่มากเกินไปไปที่ขั้วและขั้วอาจหลุดหลวม ซึ่งทำให้การสตาร์ทเครื่องยนต์ไม่ดี เมื่อตรวจสอบความแน่นของสายขั้วลบแบตเตอรี่ ให้จับยึดบริเวณขั้วเท่านั้น ไม่ใช่้ที่เซ็นเซอร์กระแสไฟฟ้า

1.  ปลดขั้วต่อเซ็นเซอร์กระแสไฟฟ้า

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ถอดเซ็นเซอร์กระแสไฟฟ้า (ดูที่ การถอด / การติดตั้งเซ็นเซอร์กระแสไฟฟ้า [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ตรวจสอบว่าไม่มีความเสียหายต่อเซ็นเซอร์กระแสไฟฟ้า และไม่มีการผุกร่อนและความเสียหายที่ขั้วต่อ

•  ถ้ามีความบกพร่อง ให้เปลี่ยนเซ็นเซอร์กระแสไฟฟ้า (ดูที่ การถอด / การติดตั้งเซ็นเซอร์กระแสไฟฟ้า [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
