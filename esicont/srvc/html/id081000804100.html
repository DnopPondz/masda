การถอด / การติดตั้งสวิตช์ปิดการทำงานของถุงลมนิรภัยด้านผู้โดยสาร (PAD)

1.  บิดสวิตช์สตาร์ทไปที่ OFF

2.  ปลดสายขั้วลบแบตเตอรี่และรอเป็นเวลา 1 นาที หรือมากกว่า (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นกันรอยขีดข่วนด้านหน้า (ด้านผู้โดยสาร) (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(2)  แผงปิดด้านหน้า (ด้านผู้โดยสาร) (ดูที่ การถอด / การติดตั้งแผงปิดด้านหน้า)
(3)  ช่องเก็บของ (ดูที่ การถอด / การติดตั้งช่องเก็บของ)
(4)  แผงตกแต่ง (ดูที่ การถอด / การติดตั้งแผงตกแต่ง)
(5)  แผงด้านล่าง (ด้านผู้โดยสาร) (ดูที่ การถอด / การติดตั้งแผงด้านล่าง)

4.  ถอดสวิตช์ PAD ในทิศทางตามลูกศร (2) ที่แสดงไว้ในรูปภาพและกดแถบยึดในทิศทางตามลูกศร (1)

5.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

6.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน)

7.  ตรวจสอบว่าไฟเตือนระบบถุงลมนิรภัยติดสว่างประมาณ 6 วินาที และดับลง

•  หากไฟเตือนระบบถุงลมนิรภัยทำงานผิดปกติ ให้ดูที่ระบบการวิเคราะห์ปัญหาบนรถ (ระบบถุงลมนิรภัย) และทำการตรวจสอบระบบ
