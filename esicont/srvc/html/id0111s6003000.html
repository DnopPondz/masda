คำเตือน
•  น้ำมันเครื่องและเครื่องยนต์ที่ร้อนจัดสามารถทำให้เกิดไฟลวกรุนแรงได้ ดับเครื่องยนต์และรอจนกระทั่งเครื่องยนต์และน้ำมันเครื่องเย็นตัวลง
•  รถที่ถูกยกขึ้นแต่ไม่ได้รองรับไว้อย่างมั่นคงบนขาตั้งนิรภัยจะเป็นอันตราย ซึ่งสามารถเลื่อนหลุดหรือตกลงมาซึ่งทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตได้ ห้ามปฏิบัติงานรอบๆ หรือใต้รถที่ยกขึ้นหากไม่ได้รองรับไว้อย่างมั่นคงบนขาตั้งนิรภัย
•  จากการทดสอบกับหนูทดลองในห้องทดลอง การสัมผัสน้ำมันเครื่องที่ใช้แล้วต่อเนื่องเป็นเวลานานอาจเป็นสาเหตุให้เกิดมะเร็งผิวหนังได้ ปกป้องผิวหนังของท่านด้วยการล้างด้วยน้ำและฟอกสบู่หลังจากการปฏิบัติงานกับน้ำมันเครื่อง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดฝาครอบใต้เครื่องด้านหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

3.  ถอดแผ่นบังโคลน (ด้านขวา) (ดูที่ การถอด / การติดตั้งแผ่นบังโคลน)

4.  ถอดขายึดอุปกรณ์แปรสภาพก๊าซไอเสียด้านล่าง (ดูที่ การถอด / การติดตั้งระบบไอเสีย [SKYACTIV-D 2.2])

5. ถอดตามลำดับที่ระบุไว้ในตาราง

6. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

7.  สตาร์ทเครื่องยนต์และตรวจยืนยันว่าไม่มีการรั่วของน้ำมันเครื่อง
