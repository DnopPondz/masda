คำเตือน
•  การจัดการโมดูลถุงลมนิรภัยอย่างไม่เหมาะสมอาจทำให้โมดูลถุงลมนิรภัยพองตัว ซึ่งอาจทำให้เกิดการบาดเจ็บร้ายแรง อ่านคำเตือนและข้อควรระวังการซ่อมบำรุงระบบถุงลมนิรภัยก่อนดำเนินการจัดการโมดูลถุงลมนิรภัย (ดูที่ คำเตือนการซ่อมบำรุงระบบถุงลมนิรภัย) (ดูที่ ข้อควรระวังในการซ่อมบำรุงระบบถุงลมนิรภัย)

1.  บิดสวิตช์สตาร์ทไปที่ OFF

2.  ปลดสายขั้วลบแบตเตอรี่และรอเป็นเวลา 1 นาที หรือมากกว่า (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

3.  พลิกยางกันซึมขึ้นบางส่วน

4.  ถอดชิ้นส่วนต่อไปนี้:

(1)  โครงรอยต่อหลังคาซันรูฟ (รุ่นที่มีหลังคาซันรูฟ) (ดูที่ การถอด / การติดตั้งชุดหลังคาซันรูฟ)
(2)  แผ่นปิดเสา A (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา A)
(3)  ไฟส่องเอกสารด้านหน้า (ดูที่ การถอด / การติดตั้งไฟส่องเอกสาร)
(4)  ที่บังแดด (ดูที่ การถอด / การติดตั้งที่บังแดด)
(5)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(6)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(7)  แผ่นปิดเสา B ด้านล่าง (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา B ด้านล่าง)
(8)  ฝาครอบตัวปรับเข็มขัดนิรภัยของที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(9)  ตัวยึดด้านบนของเข็มขัดนิรภัยที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(10)  แผ่นปิดด้านบนเสา B (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านบนเสา B)
(11)  มือจับเสา (ดูที่ การถอด / การติดตั้งมือจับเสา)
(12)  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
(13)  แผงปิดปลายห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดปลายห้องเก็บสัมภาระ)
(14)  แผงปิดประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดประตูท้าย)
(15)  แผ่นปิดเสา D (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา D)
(16)  แผ่นปิดเสา C (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา C)

5.  ถอดแผงปิดใต้หลังคา (ดูที่ การถอด / การติดตั้งแผงปิดใต้หลังคา)

6.  ใช้ไขควงปากแบนยกอุปกรณ์ล็อคขึ้นอย่างระมัดระวัง แต่ยังไม่ต้องนำออก

15.  เมื่อสวิตช์สตาร์ทอยู่ที่ ON (เครื่องยนต์ไม่ทำงานหรือทำงาน)

16.  ตรวจสอบว่าไฟเตือนระบบถุงลมนิรภัยติดสว่างประมาณ 6 วินาที และดับลง
