การถอด / การติดตั้งแร็กพวงมาลัยและก้านต่อ

id061300801700

ข้อควรระวัง
•  การปฏิบัติตามขั้นตอนต่อไปนี้โดยไม่ถอดเซ็นเซอร์ความเร็วล้อระบบ ABS ก่อนอาจทำให้ชุดสายไฟขาดวงจรหากถูกดึงโดยไม่ตั้งใจ ก่อนปฏิบัติตามขั้นตอนต่อไปนี้ ให้ปลดขั้วต่อเซ็นเซอร์ความเร็วล้อระบบ ABS (ด้านดุมล้อ) และยึดชุดสายไฟไว้กับที่ที่เหมาะสมที่จะไม่ถูกดึงโดยไม่ตั้งใจในขณะซ่อมรถ
•  ยึดพวงมาลัยไว้ด้วยเทปหรือสายไฟเพื่อป้องกันแกนพวงมาลัยหมุนหลังจากปลดแกนพวงมาลัย ถ้าพวงมาลัยหมุนหลังจากปลดแกนพวงมาลัย และแร็กพวงมาลัยกับก้านต่อ ชิ้นส่วนภายในของคล็อคสปริงอาจได้รับความเสียหาย

1.  ถอดฝาครอบข้อต่อ (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

2.  ปลดอินเตอร์มีเดียทชาฟออกจากแร็กพวงมาลัยและก้านต่อ (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

3.  ปลดชุดสายไฟเซ็นเซอร์ความเร็วล้อระบบ ABS ด้านหน้าที่ติดตั้งอยู่กับชุดคอม้าและวางไว้ด้านข้าง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ความเร็วล้อหน้าของระบบ ABS)

4.  ถอดฝาครอบด้านล่างหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

5.  ถอดฝาครอบใต้เครื่องด้านหน้า No.1 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.1)

6.  ถอดแผ่นบังโคลน (ดูที่ การถอด / การติดตั้งแผ่นบังโคลน)

7.  ถอดส่วนประกอบของคานยึดชิ้นส่วนช่วงล่างด้านหน้า (ดูที่ การถอด / การติดตั้งคานยึดชิ้นส่วนช่วงล่างด้านหน้า)

8. ถอดตามลำดับที่ระบุไว้ในตาราง

9. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

10.  หลังจากการติดตั้ง ให้ตรวจสอบการตั้งศูนย์ล้อหน้า (ดูที่ การตั้งศูนย์ล้อหน้า)
