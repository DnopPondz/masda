การควบคุมเทอร์โบคู่ [SKYACTIV-D 2.2]

id0140z7987800


รายละเอียด

•  โดยการสับเปลี่ยนการอัดอากาศระหว่างเทอร์โบชาร์จเจอร์แบบใหญ่กับเทอร์โบชาร์จเจอร์แบบเล็ก จึงทำให้สมรรถนะการขับขี่ภายใต้ช่วงสภาวะต่างๆ และประสิทธิภาพการปล่อยมลพิษดียิ่งขึ้น
•  โซลินอยด์วาล์ว 3 ตัวต่อไปนี้จะถูกควบคุมตามสภาวะการทำงานของเครื่องยนต์เพื่อสับเปลี่ยนการอัดอากาศระหว่างเทอร์โบชาร์จเจอร์แบบใหญ่กับเทอร์โบชาร์จเจอร์แบบเล็ก
―  โซลินอยด์วาล์วบายพาสคอมเพรสเซอร์
•  วาล์วบายพาสคอมเพรสเซอร์จะเปิด / ปิดตามสัญญาณ on/off จาก PCM
―  โซลินอยด์วาล์วควบคุม
•  วาล์วควบคุมจะเปิด / ปิดตามสัญญาณ duty จาก PCM
―  เวสเกตโซลินอยด์วาล์ว
•  เวสเกตวาล์วจะเปิด / ปิดตามสัญญาณ duty จาก PCM

*1  : ค่า Duty ที่ต้องการเพื่อปิดวาล์วควบคุมอยู่ที่ 50% อย่างไรก็ตาม ถ้ายากที่จะปิดวาล์วควบคุมเนื่องจากแรงดันก๊าซไอเสีย PCM จะเพิ่มค่า Duty
*2  : ค่า Duty ที่ต้องการเพื่อปิดเวสเกตวาล์วอยู่ที่ 60% อย่างไรก็ตาม ถ้ายากที่จะปิดเวสเกตวาล์วเนื่องจากแรงดันก๊าซไอเสีย PCM จะเพิ่มค่า Duty
•  สำหรับรายละเอียดเกี่ยวกับการควบคุมเทอร์โบคู่ ให้อ้างอิงระบบการอัดอากาศ (ดูที่ ระบบการอัดอากาศ [SKYACTIV-D 2.2])
