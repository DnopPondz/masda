การตรวจสอบปั๊มเชื้อเพลิงแรงดันสูง [SKYACTIV-G 2.0, SKYACTIV-G 2.5]

id0114z9801500

หมายเหตุ
•  ค่าสุญญากาศต่อไปนี้จะแสดงโดยแรงดันสัมพันธ์จากเซ็นเซอร์แรงดันน้ำมันเชื้อเพลิง

1.   ตรวจสอบว่าเซ็นเซอร์แรงดันน้ำมันเชื้อเพลิงปกติหรือไม่ (ดูที่ การตรวจสอบเซ็นเซอร์แรงดันเชื้อเพลิง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

2.  ต่อ M-MDS เข้ากับ DLC-2

3.  อุ่นเครื่องยนต์

4.   เลือก “FUEL_PRES” PID บน M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

5.   ค่า “FUEL_PRES” PID อยู่ที่ 3.0 MPa {31 kgf/cm2, 435 psi} หรือมากกว่าในขณะเดินเบาหรือไม่

―  ถ้าไม่เป็นไปตามที่กำหนด ให้เปลี่ยนปั๊มเชื้อเพลิงแรงดันสูง (ดูที่ การถอด / การติดตั้งปั๊มเชื้อเพลิงแรงดันสูง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
