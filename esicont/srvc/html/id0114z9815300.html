การตรวจสอบวาล์วกันกลับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]

คำเตือน
•  น้ำมันเชื้อเพลิงเป็นของเหลวที่ไวไฟ ถ้าน้ำมันเชื้อเพลิงหกหรือรั่วจากระบบเชื้อเพลิงที่มีแรงดันภายใน จะทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตและทำให้ชิ้นส่วนต่างๆ ที่อยู่ใกล้เคียงเสียหายได้ น้ำมันเชื้อเพลิงสามารถทำให้ระคายเคืองผิวหนังและดวงตาได้อีกด้วย การป้องกันกรณีนี้ ให้ปฏิบัติตาม “ขั้นตอนการป้องกันท่อทางน้ำมันเชื้อเพลิง” ทุกครั้ง ในขณะที่อ้างอิง “ข้อพึงระวังก่อนการซ่อมบำรุง”
•  บุคคลที่มีไฟฟ้าสถิตอาจทำให้เกิดไฟไหม้หรือระเบิดได้ เป็นผลทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตได้ ก่อนถ่ายน้ำมันเชื้อเพลิง ให้แน่ใจว่าได้คายไฟฟ้าสถิตโดยการสัมผัสตัวถังรถ

1.  ให้ปฏิบัติตาม “ข้อพึงระวังก่อนการซ่อมบำรุง” (ดูที่ ข้อพึงระวังก่อนการซ่อมบำรุง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

3.  ใช้ขั้นตอนต่อไปนี้:

แบบแยก 6:4
1.  ถอดเบาะนั่งด้านหลัง (ดูที่ การถอด / การติดตั้งเบาะนั่งด้านหลัง)
แบบแยก 4:2:4
1.  วางเบาะหลังไว้ด้านข้างดังแสดงในภาพ (ดูที่ การถอด / การติดตั้งเบาะหลัง)

4.  ถอดชุดปั๊มเชื้อเพลิง (ดูที่ การถอด / การติดตั้งชุดปั๊มเชื้อเพลิง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

5.  ถ่ายน้ำมันเชื้อเพลิงออกจากถังน้ำมันเชื้อเพลิงไปที่ตำแหน่งที่สามารถมองเห็นวาล์วกันกลับ (ดูที่ ขั้นตอนการถ่ายน้ำมันเชื้อเพลิง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

หมายเหตุ
•  วาล์วกันกลับจะรวมอยู่ในถังน้ำมันเชื้อเพลิง
•  โดยปกติวาล์วกันกลับจะปิดโดยแรงดันสปริง

6.  ตรวจสอบว่าวาล์วกันกลับถูกปิด

•  ถ้าวาล์วกันกลับเปิดค้างและไม่เปิดแม้ว่าจะดึงขึ้นด้วยนิ้วแล้วก็ตาม ให้เปลี่ยนถังน้ำมันเชื้อเพลิง (ดูที่ การถอด / การติดตั้งถังน้ำมันเชื้อเพลิง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
