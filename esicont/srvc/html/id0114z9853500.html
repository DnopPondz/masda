คำเตือน
•  การหกและการรั่วของท่อทางเชื้อเพลิงจากระบบเชื้อเพลิงที่มีแรงดันภายในจะเป็นอันตราย น้ำมันเชื้อเพลิงสามารถลุกติดไฟและเป็นสาเหตุทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิต และเสียหายได้ การป้องกันกรณีนี้ ให้ทำการตรวจสอบดังต่อไปนี้ขณะดับเครื่องยนต์
•  น้ำมันเชื้อเพลิงเป็นของเหลวที่ไวไฟ ถ้าน้ำมันเชื้อเพลิงหกหรือรั่วจากระบบเชื้อเพลิงที่มีแรงดันภายใน จะทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตและทำให้ชิ้นส่วนต่างๆ ที่อยู่ใกล้เคียงเสียหายได้ น้ำมันเชื้อเพลิงสามารถทำให้ระคายเคืองผิวหนังและดวงตาได้อีกด้วย การป้องกันกรณีนี้ ให้ปฏิบัติตาม “ขั้นตอนการป้องกันท่อทางน้ำมันเชื้อเพลิง” ทุกครั้ง ในขณะที่อ้างอิง “ข้อพึงระวังก่อนการซ่อมบำรุง”
•  บุคคลที่มีไฟฟ้าสถิตอาจทำให้เกิดไฟไหม้หรือระเบิดได้ เป็นผลทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตได้ ก่อนปฏิบัติงานกับระบบเชื้อเพลิง ให้คายไฟฟ้าสถิตโดยการสัมผัสตัวถังรถ

ขั้นตอนการดูดน้ำมันเชื้อเพลิงด้วยหลักความกดอากาศโดยใช้ปั๊มเชื้อเพลิง

ข้อควรระวัง
•  การปลด / ต่อข้อต่อแบบปลดเร็วโดยที่ไม่ทำความสะอาดอาจเป็นสาเหตุทำให้ท่อทางเชื้อเพลิงและข้อต่อแบบปลดเร็วเสียหายได้ ให้ทำความสะอาดบริเวณที่ต่อของข้อต่อแบบปลดเร็วก่อนการปลด / ต่อทุกครั้งโดยใช้ผ้าหรือแปรงขนนุ่ม และให้แน่ใจว่าไม่มีสิ่งแปลกปลอม

1.  ปฏิบัติตาม “ขั้นตอนการป้องกันท่อทางน้ำมันเชื้อเพลิง” โดยอ้างอิง “ข้อพึงระวังก่อนการซ่อมบำรุง” (ดูที่ ข้อพึงระวังก่อนการซ่อมบำรุง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

3.  ปลดข้อต่อแบบปลดเร็วดังแสดงในภาพ (ดูที่ การถอด / การติดตั้งข้อต่อแบบปลดเร็ว [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ต่อท่อยาวเข้ากับข้อต่อแบบปลดเร็วที่ปลดออก และถ่ายน้ำมันเชื้อเพลิงลงในภาชนะที่ใช้สำหรับรองน้ำมันเบนซิน

5.  ถ่ายน้ำมันเชื้อเพลิงออกจากถังน้ำมันเชื้อเพลิงโดยใช้ขั้นตอนดังต่อไปนี้:

ใช้ M-MDS
1.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])
2.  ต่อ M-MDS เข้ากับ DLC-2
3.  ใช้ฟังก์ชั่นการจำลองสภาพปัญหา “FP” สั่งให้ปั๊มเชื้อเพลิงทำงาน (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
―  ถ้าปั๊มเชื้อเพลิงไม่สามารถใช้งานได้ ให้ถ่ายน้ำมันเชื้อเพลิงโดยใช้ขั้นตอนที่มีให้ (ดูที่ ขั้นตอนการดูดน้ำมันเชื้อเพลิงด้วยหลักความกดอากาศโดยไม่ใช้ปั๊มเชื้อเพลิง)
ไม่ใช้ M-MDS
1.  ถอดรีเลย์ปั๊มเชื้อเพลิง

ข้อควรระวัง
•  ปั๊มเชื้อเพลิงอาจได้รับความเสียหายถ้าใช้งานปั๊มเชื้อเพลิงในขณะที่ไม่มีน้ำมันเชื้อเพลิงอยู่ในถังน้ำมันเชื้อเพลิง ตรวจสอบว่าน้ำมันเชื้อเพลิงถูกจ่ายออกจากท่อและหยุดการทำงานของปั๊มเชื้อเพลิงเมื่อไม่มีน้ำมันเชื้อเพลิงจ่ายออกมา

6.  หยุดปั๊มเชื้อเพลิงโดยใช้ขั้นตอนต่อไปนี้

ใช้ M-MDS
1.  ใช้ฟังก์ชั่นการจำลองสภาพปัญหา “FP” สั่งให้ปั๊มเชื้อเพลิงหยุดทำงาน (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])
ไม่ใช้ M-MDS
1.  ถอดสายพ่วงเพื่อหยุดการทำงานของปั๊มเชื้อเพลิง
2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])
