รายละเอียด

•  ฟังก์ชั่นการวิเคราะห์ปัญหาบนรถจะประกอบด้วยฟังก์ชั่นต่อไปนี้: ฟังก์ชั่นการตรวจจับความบกพร่อง ซึ่งจะตรวจจับความบกพร่องทั้งหมดในชิ้นส่วนที่เกี่ยวข้องกับโมดูลควบคุมตัวถังด้านหลัง (RBCM) ฟังก์ชั่นหน่วยความจำ ซึ่งจะเก็บบันทึกรหัสวิเคราะห์ปัญหาที่ตรวจพบ ฟังก์ชั่นการแสดงผล ซึ่งจะแสดงตำแหน่งและสถานะความบกพร่องผ่านรหัสวิเคราะห์ปัญหาที่ปรากฏออกมา และฟังก์ชั่นการตรวจสอบ PID / ข้อมูล ซึ่งจะอ่านสัญญาณอินพุท / เอาท์พุทที่กำหนด และตรวจสอบสภาวะสัญญาณอินพุท / เอาท์พุท
•  เมื่อใช้ Mazda Modular Diagnostic System (M-MDS) สามารถอ่านและลบรหัสวิเคราะห์ปัญหา และสามารถเปิดใช้งานฟังก์ชั่นการตรวจสอบ PID/DATA ได้

ฟังก์ชั่นการตรวจจับความบกพร่อง

•  ตรวจสอบความบกพร่องในสัญญาณอินพุทและเอาท์พุท
•  ถ้าความบกพร่องเกิดขึ้น โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะบันทึกความบกพร่องเป็นรหัสวิเคราะห์ปัญหา สามารถอ่านรหัสวิเคราะห์ปัญหาที่บันทึกไว้โดย Mazda Modular Diagnostic System (M-MDS)
