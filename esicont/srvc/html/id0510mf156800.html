ข้อควรระวัง
•  ยึดพวงมาลัยไว้ด้วยเทปหรือสายไฟเพื่อป้องกันแกนพวงมาลัยหมุนหลังจากปลดแกนพวงมาลัย ถ้าพวงมาลัยหมุนหลังจากปลดแกนพวงมาลัย และแร็กพวงมาลัยกับก้านต่อ ชิ้นส่วนภายในของคล็อคสปริงอาจได้รับความเสียหาย
•  อย่าให้น้ำมันคลัตช์โดนสีรถ น้ำมันคลัตช์มีคุณสมบัติซึ่งสามารถทำลายสีได้ ถ้าน้ำมันคลัตช์โดนสีรถ ให้ล้างออกด้วยน้ำทันที และเช็ดออกให้เรียบร้อย

การถอด

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดชิ้นส่วนประกอบกรองอากาศ (กรองอากาศ, ท่อยางอากาศ) (L.H.D.) (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

3.  ถอดชิ้นส่วนประกอบแบตเตอรี่ (แบตเตอรี่ ถาดรองแบตเตอรี่) (L.H.D.) (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-D 2.2])

4. ถอดขายึดออกจากแกนเพลาพวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

5.  ถอดสวิตช์ตำแหน่งแป้นคลัตช์ (ดูที่ การถอด / การติดตั้งสวิตช์ตำแหน่งแป้นคลัตช์ [D66M-R, D66MX-R])

6.  ถอดสวิตช์อินเตอร์ล็อคมอเตอร์สตาร์ท (ดูที่ การถอด / การติดตั้งสวิตช์อินเตอร์ล็อคมอเตอร์สตาร์ท [D66M-R, D66MX-R])

7.  ถอดฝาครอบข้อต่อ (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

8.  ปลดแกนเพลาพวงมาลัยออกจากแร็กพวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

9.  ถอดฝาครอบด้านล่างหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

10.  ถอดฉนวน (R.H.D.) (ดูที่ การถอด / การติดตั้งระบบไอเสีย [SKYACTIV-D 2.2])

11.  ปลดท่อยางและท่อทางน้ำมันคลัตช์ No.1 ออกจากแม่ปั๊มคลัตช์ และอุดไว้เพื่อหลีกเลี่ยงการรั่วของน้ำมันคลัตช์ (ดูที่ การถอด / การติดตั้งท่อยางและท่อทางน้ำมันคลัตช์ [D66M-R, D66MX-R])

12.  ปลดท่อกระปุกน้ำมันคลัตช์ออกจากแม่ปั๊มคลัตช์ และอุดไว้เพื่อหลีกเลี่ยงการรั่วของน้ำมันคลัตช์ (ดูที่ การถอด / การติดตั้งท่อยางและท่อทางน้ำมันคลัตช์ [D66M-R, D66MX-R])

13.  ถอดแม่ปั๊มคลัตช์โดยใช้ขั้นตอนต่อไปนี้:

(1)  ถอดแป้นคลัตช์และแม่ปั๊มคลัตช์ออกเป็นชุด

2.  ต่อท่อกระปุกน้ำมันคลัตช์เข้ากับแม่ปั๊มคลัตช์ (ดูที่ การถอด / การติดตั้งท่อยางและท่อทางน้ำมันคลัตช์ [D66M-R, D66MX-R])

3.  ต่อท่อยางและท่อทางน้ำมันคลัตช์ No.1 เข้ากับแม่ปั๊มคลัตช์ (ดูที่ การถอด / การติดตั้งท่อยางและท่อทางน้ำมันคลัตช์ [D66M-R, D66MX-R])

4.  ต่อแกนเพลาพวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

5.  ติดตั้งฝาครอบข้อต่อ (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

6.  ไล่ลมออกจากระบบคลัตช์ (ดูที่ การเปลี่ยน / การไล่ลมน้ำมันคลัตช์ [D66M-R, D66MX-R])

7.  ตรวจสอบแป้นคลัตช์ (ดูที่ การตรวจสอบแป้นคลัตช์ [D66M-R, D66MX-R])

8.  ติดตั้งสวิตช์อินเตอร์ล็อคมอเตอร์สตาร์ทอันใหม่ (ดูที่ การถอด / การติดตั้งสวิตช์อินเตอร์ล็อคมอเตอร์สตาร์ท [D66M-R, D66MX-R])

9.  ติดตั้งสวิตช์ตำแหน่งแป้นคลัตช์อันใหม่ (ดูที่ การถอด / การติดตั้งสวิตช์ตำแหน่งแป้นคลัตช์ [D66M-R, D66MX-R])

10.  ติดตั้งฉนวน (R.H.D.) (ดูที่ การถอด / การติดตั้งระบบไอเสีย [SKYACTIV-D 2.2])

11.  ติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

12.  ติดตั้งขายึดเข้ากับแกนเพลาพวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)

13.  ติดตั้งชิ้นส่วนประกอบแบตเตอรี่ (แบตเตอรี่ ถาดรองแบตเตอรี่) (L.H.D.) (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-D 2.2])

14.  ติดตั้งชิ้นส่วนประกอบกรองอากาศ (กรองอากาศ, ท่อยางอากาศ) (L.H.D.) (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

15.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])
