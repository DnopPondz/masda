การตรวจสอบชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง [4WD]

SKYACTIV-G 2.0, SKYACTIV-G 2.5


ชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (หลัก)

หมายเหตุ
•  โปรดอ้างอิงการถอด / การติดตั้งชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิงกับการถอด / การติดตั้งปั๊มน้ำมันเชื้อเพลิง เนื่องจากชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิงรวมอยู่กับปั๊มน้ำมัน (ดูที่ การถอด / การติดตั้งชุดปั๊มเชื้อเพลิง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

1.  ตรวจสอบว่าค่าความต้านทานที่ขั้ว D และ C ของชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (หลัก) เป็นไปตามที่ระบุในตารางตามความสูงของลูกลอย

•  ถ้าไม่เป็นไปตามที่ระบุในตาราง ให้เปลี่ยนชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (หลัก)


ชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (รอง)

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])(ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  เบาะนั่งด้านหลัง (เบาะแบบแยก 6:4) (ดูที่ การถอด / การติดตั้งเบาะนั่งด้านหลัง)
(2)  โบลท์ติดตั้งใต้เบาะหลัง (เบาะแบบแยก 4:2:4) (ดูที่ การถอด / การติดตั้งเบาะหลัง)
(3)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(4)  ชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (รอง) (ดูที่ การถอด / การติดตั้งชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง [4WD])

3.  ตรวจสอบว่าค่าความต้านทานที่ขั้ว B และ A ของชุดส่งสัญญาณเกจวัดระดับน้ำมันเชื้อเพลิง (รอง) เป็นไปตามที่ระบุในตารางตามความสูงของลูกลอย
