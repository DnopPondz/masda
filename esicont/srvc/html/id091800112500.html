1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดแผงสวิตช์ (ดูที่ การถอด / การติดตั้งแผงสวิตช์)

3.  ตรวจสอบว่าความต้านทานระหว่างขั้ว B และ C ของสวิตช์ AFS OFF เป็นไปตามที่ระบุไว้ในตาราง

•  ถ้าสามารถตรวจสอบได้ว่าค่าความต้านทานเป็นไปตามที่ระบุในตาราง ให้ไปที่ขั้นตอนถัดไป
•  ถ้าไม่เป็นไปตามที่ระบุในตาราง ให้เปลี่ยนสวิตช์แผงหน้าปัด

4.  จ่ายแรงดันไฟฟ้าขั้วบวกแบตเตอรี่ไปที่ขั้ว J ของสวิตช์ AFS OFF และต่อขั้ว I ลงกราวด์
