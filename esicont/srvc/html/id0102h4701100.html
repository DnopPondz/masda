รหัสวิเคราะห์ปัญหา P0107:00

สัญญาณอินพุทวงจรเซ็นเซอร์ MAP ต่ำกว่าที่กำหนด
สภาวะการตรวจพบ
•  PCM จะคอยตรวจสอบแรงดันไฟฟ้าอินพุทจากเซ็นเซอร์ MAP ถ้าแรงดันไฟฟ้าอินพุทที่ขั้ว 1BW ของ PCM ต่ำกว่า 0.08 V เป็นเวลา 5 วินาที PCM จะตัดสินว่าวงจรเซ็นเซอร์ MAP มีความบกพร่อง
ข้อมูลสนับสนุนการวิเคราะห์ปัญหา
•  กรณีนี้คือ การตรวจสอบต่อเนื่อง (CCM)
•  ไฟเตือนตรวจเช็คเครื่องยนต์จะติดสว่างถ้า PCM ตรวจพบสภาวะความบกพร่องดังกล่าวด้านบนในระหว่างรอบการขับขี่แรก
•  มีข้อมูลสภาพรถยนต์ (FREEZE FRAME DATA) (โหมด 2) / ข้อมูล Snapshot
•  รหัสวิเคราะห์ปัญหาถูกเก็บไว้ในหน่วยความจำ PCM
ฟังก์ชั่น FAIL-SAFE
•  ประมาณค่า MAP โดยใช้เซ็นเซอร์ MAF และความเร็วรอบเครื่องยนต์
•  จำกัดขอบเขตสูงสุดของความเร็วรอบเครื่องยนต์
•  ยับยั้งการควบคุมการดูดไอระเหยน้ำมันเชื้อเพลิง
สาเหตุที่เป็นไปได้
•  ขั้วต่อหรือขั้วของเซ็นเซอร์ MAP / เซ็นเซอร์ IAT No.2 บกพร่อง
•  ชุดสายไฟระหว่างขั้วต่างๆ ต่อไปนี้ลัดวงจรลงกราวด์:
―  ขั้ว C ของเซ็นเซอร์ MAP / เซ็นเซอร์ IAT No.2 — ขั้ว 1BO ของ PCM
―  ขั้ว D ของเซ็นเซอร์ MAP / เซ็นเซอร์ IAT No.2 — ขั้ว 1BW ของ PCM
•  ขั้วต่อหรือขั้วของ PCM บกพร่อง
•  วงจรสัญญาณเซ็นเซอร์ MAP กับวงจรกราวด์ลัดวงจรกัน
•  เซ็นเซอร์ MAP บกพร่อง
•  PCM บกพร่อง
