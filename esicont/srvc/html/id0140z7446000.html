การตรวจสอบเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2]

id0140z7446000


เซ็นเซอร์แรงดันก๊าซไอเสีย No.1

คำเตือน
•  เครื่องยนต์และระบบไอเสียที่ร้อนสามารถทำให้เกิดการบาดเจ็บรุนแรงจากการโดนไหม้ได้ ดับเครื่องยนต์และรอจนกว่าจะเย็นตัวลงก่อนถอดระบบไอเสีย

การตรวจสอบด้วยสายตา

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดฝาครอบเครื่องยนต์ (ดูที่ การถอด / การติดตั้งฝาครอบเครื่องยนต์ [SKYACTIV-D 2.2])

3.  ถอดฉนวน (ดูที่ การถอด / การติดตั้งโซ่ไทม์มิ่ง [SKYACTIV-D 2.2])

4.  ปลดขั้วต่อเซ็นเซอร์แรงดันก๊าซไอเสีย No.1

5.  ถอดโบลท์ติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 ออก (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])

6.  ถอดโบลท์ติดตั้งขายึด (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])

7.  ถอดเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])

8.  ตรวจสอบดูเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 เพื่อหาความบกพร่องดังต่อไปนี้:

•  ความเสียหาย การแตกร้าว ความสกปรก
―  ถ้ามีความบกพร่องใดๆ ให้ซ่อมหรือเปลี่ยนเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])

9.  ตรวจสอบดูท่ออากาศเพื่อหาความบกพร่องดังต่อไปนี้:

•  การเดินท่อไม่ถูกต้อง บิดงอ หรือการรั่วของท่ออากาศ
―  ถ้ามีความบกพร่องใดๆ ให้ซ่อมหรือเปลี่ยนชิ้นส่วนที่บกพร่องตามผลการตรวจสอบ


การตรวจสอบแรงดันไฟฟ้า

1.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

2.  วัดแรงดันไฟฟ้าที่ขั้ว B ของเซ็นเซอร์แรงดันก๊าซไอเสีย No.1

•  ถ้าแรงดันไฟฟ้าอยู่ภายในค่ากำหนด ให้ไปที่ขั้นตอนถัดไป
•  ถ้าแรงดันไฟฟ้าไม่อยู่ภายในค่ากำหนด ให้เปลี่ยนเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])

ประมาณ 0.95 V

3.  ต่อ M-MDS เข้ากับ DLC-2

4.  สตาร์ทเครื่องยนต์

5.  เข้ารายการตรวจสอบ PID/DATA EXHPRES1 โดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

6.  วัดแรงดันไฟฟ้าที่ขั้ว B ของเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 เมื่อ EXHPRES1 PID เพิ่มขึ้น

•  ถ้าแรงดันไฟฟ้าไม่เพิ่มขึ้น ให้เปลี่ยนเซ็นเซอร์แรงดันก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันก๊าซไอเสีย [SKYACTIV-D 2.2])
