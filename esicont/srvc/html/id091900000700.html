การทำงาน


การทำงานที่ความเร็วต่ำแบบต่อเนื่อง

1.  เมื่อเลื่อนสวิตช์ที่ปัดน้ำฝนและที่ฉีดล้างกระจกไปที่ตำแหน่ง LO ขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ (LO) สวิตช์ของที่ปัดน้ำฝนกระจกบังลมหน้า (2)

2.  ชุดเปิด/ปิดการทำงานจะตรวจจับสัญญาณ (LO) สวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้า ทำการส่งสัญญาณ (LO) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้าไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณ (LO) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้า จะทำการจ่ายกระแสไฟอ้างอิงจาก CPU ภายในไปยังทรานซิสเตอร์ A (4) และกระแสคอลเลคเตอร์จะไหลผ่านจากแหล่งจ่ายไฟภายใน (5) เพื่อเปิดใช้งานรีเลย์ต่ำของที่ปัดน้ำฝนกระจกบังลมหน้า (6)

4.  เมื่อเปิดใช้งานรีเลย์ต่ำของที่ปัดน้ำฝนกระจกบังลมหน้า กระแสไฟจะไหลผ่านจากแบตเตอรี่ไปยังมอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้า และที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานแบบต่อเนื่องที่ความเร็วต่ำ (7)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ (HI) สวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้า อุปกรณ์จะทำการส่งสัญญาณ (HI) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้าไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณ (HI) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้า จะทำการจ่ายกระแสไฟอ้างอิงจาก CPU ภายในไปยังทรานซิสเตอร์ A และ B (4) และกระแสคอลเลคเตอร์จะไหลผ่านจากแหล่งจ่ายไฟภายใน (5) เพื่อเปิดใช้งานรีเลย์สูงของที่ปัดน้ำฝนกระจกบังลมหน้า (6)

4.  เมื่อเปิดใช้งานรีเลย์ต่ำของที่ปัดน้ำฝนกระจกบังลมหน้าและรีเลย์สูงของที่ปัดน้ำฝนกระจกบังลมหน้า กระแสไฟจะไหลผ่านจากแบตเตอรี่ไปยังมอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้า และที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานแบบต่อเนื่องที่ความเร็วสูง (7)

การทำงานของระบบหยุดอัตโนมัติ

1.  สวิตช์หยุดอัตโนมัติภายในมอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานในลักษณะต่อไปนี้ตามตำแหน่งการทำงานของที่ปัดน้ำฝนกระจกบังลมหน้า

•  ที่ปัดน้ำฝนกระจกบังลมหน้าหยุดในตำแหน่งจอด: ON
•  ที่ปัดน้ำฝนกระจกบังลมหน้าหยุดในตำแหน่งอื่นนอกเหนือจากตำแหน่งจอด: OFF

2.  เมื่อสวิตช์ที่ปัดน้ำฝนและที่ฉีดน้ำล้างกระจกอยู่ในตำแหน่ง OFF (1) ในระหว่างที่ที่ปัดน้ำฝนกระจกบังลมหน้าทำงาน ที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานต่อไปเนื่องจากสวิตช์หยุดอัตโนมัติอยู่ที่ตำแหน่ง OFF

3.  สวิตช์หยุดอัตโนมัติเปิดใช้งานเมื่อที่ปัดน้ำฝนกระจกบังลมหน้าเลื่อนไปที่ตำแหน่งจอด (2)

4.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ตรวจพบว่าสวิตช์หยุดอัตโนมัติอยู่ที่ตำแหน่ง ON อุปกรณ์จะหยุดการจ่ายกระแสไฟอ้างอิงไปยังทรานซิสเตอร์ (3) และปิดใช้งานรีเลย์ต่ำของที่ปัดน้ำฝนกระจกบังลมหน้า และที่ปัดน้ำฝนกระจกบังลมหน้าจะหยุดที่ตำแหน่งจอด

1.  เมื่อเลื่อนสวิตช์ที่ปัดน้ำฝนและที่ฉีดล้างกระจกไปที่ตำแหน่ง INT ขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ (INT) สวิตช์ของที่ปัดน้ำฝนกระจกบังลมหน้า (2)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ (INT) สวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้า อุปกรณ์จะทำการส่งสัญญาณ (INT) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกบังลมหน้าและสัญญาณระดับ INT ไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

4.  เมื่อเปิดใช้งานรีเลย์ต่ำของที่ปัดน้ำฝนกระจกบังลมหน้า กระแสไฟจะไหลผ่านจากแบตเตอรี่ไปยังมอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้า และที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานแบบต่อเนื่องที่ความเร็วต่ำ (7)

5.  ที่ปัดน้ำฝนกระจกบังลมหน้าจะหยุดที่ตำแหน่งจอดเนื่องจากสวิตช์หยุดอัตโนมัติทำงาน และหลังจากผ่านไปช่วงระยะเวลาหนึ่งซึ่งคำนวณตามสัญญาณระดับ INT โมดูลควบคุมตัวถังด้านหน้า (FBCM) จะสั่งให้ที่ปัดน้ำฝนกระจกบังลมหน้าทำงานที่ความเร็วต่ำ เมื่อทำซ้ำขั้นตอนนี้ ที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานแบบเป็นจังหวะ (8)

การทำงานของที่ปัดน้ำฝนและที่ฉีดน้ำล้างกระจกแบบสอดคล้องกัน

1.  เมื่อเปิดใช้งานสวิตช์ที่ฉีดล้างกระจกบังลมหน้าขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) และที่ปัดน้ำฝนกระจกบังลมหน้าไม่ทำงาน (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ ON ของสวิตช์ที่ฉีดล้างกระจกบังลมหน้า (2)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ ON ของที่ฉีดล้างกระจกบังลมหน้า อุปกรณ์จะทำการส่งสัญญาณขอการทำงานของที่ฉีดล้างกระจกบังลมหน้าไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณขอการทำงานของที่ฉีดล้างกระจกบังลมหน้า จะทำการจ่ายกระแสไฟอ้างอิงจาก CPU ภายในไปยังทรานซิสเตอร์ C (4) และกระแสคอลเลคเตอร์จะไหลผ่านจากแหล่งจ่ายไฟภายใน (5) เพื่อเปิดใช้งานรีเลย์ของที่ฉีดล้างกระจกบังลมหน้า (6)

4.  เมื่อเปิดใช้งานรีเลย์ของที่ฉีดล้างกระจกบังลมหน้าแล้ว กระแสไฟฟ้าจะไหลผ่านจากแหล่งจ่ายไฟภายในโมดูลควบคุมตัวถังด้านหน้า (FBCM) ไปยังมอเตอร์ฉีดน้ำล้างกระจก และมอเตอร์ฉีดน้ำล้างกระจกจะทำงาน จากนั้นน้ำฉีดล้างกระจกจะฉีดออกจากหัวฉีดน้ำล้างกระจกด้านหน้า (7)

5.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณขอการทำงานของที่ฉีดล้างกระจกบังลมหน้าในช่วงระยะเวลาหนึ่ง อุปกรณ์จะสั่งให้ที่ปัดน้ำฝนกระจกบังลมหน้าทำงานที่ความเร็วต่ำ (8)

6.  เมื่อปิดใช้งานสวิตช์ที่ฉีดล้างกระจกบังลมหน้า การทำงานจะหยุดลงหลังจากที่ปัดน้ำฝนกระจกบังลมหน้าทำงานสองครั้ง

หมายเหตุ
•  หากชุดทำความสะอาดไฟหน้าทำงานโดยอัตโนมัติขณะที่ที่ฉีดล้างกระจกบังลมหน้ากำลังทำงาน การทำงานจะหยุดลงหลังจากที่ปัดน้ำฝนกระจกบังลมหน้าทำงานสี่ครั้ง
•  หากสวิตช์ที่ฉีดล้างกระจกบังลมหน้าทำงานขณะที่ที่ปัดน้ำฝนกระจกบังลมหน้ากำลังทำงาน ที่ปัดน้ำฝนกระจกบังลมหน้าจะทำงานต่อไปและที่ฉีดล้างกระจกบังลมหน้าจะทำงานตั้งแต่ขั้นตอน (3) ถึง (7)
