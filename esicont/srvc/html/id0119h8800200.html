การทำงานหลังจากเปลี่ยนมอเตอร์สตาร์ท

หมายเหตุ
•  ถ้าจำนวนครั้งที่มอเตอร์สตาร์ทสามารถทำงานได้เกินขีดจำกัด หรือมีความบกพร่องใดๆ ของชิ้นส่วนที่เกี่ยวข้องกับระบบ i-stop ไฟเตือนระบบ i-stop (สีเหลืองอำพัน) จะกะพริบ นอกจากนี้ รหัสวิเคราะห์ปัญหาจะถูกเก็บบันทึกไว้ในเวลาเดียวกัน
•  เพื่อให้มั่นใจในการทำงานของระบบ i-stop ให้เปลี่ยนมอเตอร์สตาร์ทใหม่ถ้าจำนวนครั้งที่มอเตอร์สตาร์ทสามารถทำงานได้เกินขีดจำกัดสูงสุด
•  เมื่อเปลี่ยนมอเตอร์สตาร์ทใหม่ ให้ปฏิบัติตามขั้นตอนดังต่อไปนี้: เมื่อเปลี่ยน PCM พร้อมกัน ให้ทำการตั้งค่า PCM ก่อน (ดูที่ การถอด / การติดตั้ง PCM [SKYACTIV-D 2.2]) (ดูที่ การตั้งค่า PCM (โดยใช้ฟังก์ชั่นการอ่าน / เขียนข้อมูล) [SKYACTIV-D 2.2]) (ดูที่ การตั้งค่า PCM (โดยใช้ข้อมูลต้นฉบับ) [SKYACTIV-D 2.2])

การถอด / การติดตั้งมอเตอร์สตาร์ท

คำเตือน
•  ถอดและติดตั้งชิ้นส่วนทั้งหมดเมื่อเครื่องยนต์เย็นตัวลงแล้ว มิฉะนั้นอาจทำให้โดนลวกหรือบาดเจ็บรุนแรงได้
•  เมื่อสายไฟแบตเตอรี่ต่ออยู่ การสัมผัสตัวถังรถกับขั้ว B ของมอเตอร์สตาร์ทจะทำให้เกิดประกายไฟ ซึ่งอาจทำให้เกิดการบาดเจ็บ ไฟไหม้ และความเสียหายต่ออุปกรณ์ไฟฟ้าได้ ให้ปลดสายขั้วลบแบตเตอรี่ก่อนปฏิบัติตามการทำงานดังต่อไปนี้ทุกครั้ง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดฝาครอบใต้เครื่องด้านหน้า No.1 และ No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.1) (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

3.  ถอดกรองอากาศ และชิ้นส่วนประกอบท่ออากาศภายนอก (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

4.  ปลดท่อทางออกอินเตอร์คูลเลอร์ออกจากลิ้นตัดอากาศ และวางท่อให้ห่างออกมา (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
