รายละเอียด

•  มีที่ปัดน้ำฝนแบบต่อเนื่อง, แบบหยุดอัตโนมัติ, ที่ปัดน้ำฝนแบบเป็นจังหวะ และการทำงานของที่ปัดน้ำฝนและที่ฉีดน้ำล้างกระจกแบบสอดคล้องกัน
•  โมดูลควบคุมตัวถังด้านหน้า (FBCM) และโมดูลควบคุมตัวถังด้านหลัง (RBCM) จะสั่งงานโหมด Fail-safe ของระบบที่ปัดน้ำฝนกระจกหลังและที่ฉีดน้ำล้างกระจก (ดูที่ โมดูลควบคุมตัวถังด้านหน้า (FBCM)) (ดูที่ โหมดควบคุมตัวถังด้านหลัง (RBCM))

การทำงาน


การทำงานแบบต่อเนื่อง

1.  เมื่อสวิตช์ที่ปัดน้ำฝนกระจกหลังและที่ฉีดล้างกระจกอยู่ที่ตำแหน่ง ON ขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ ON ของสวิตช์ที่ปัดน้ำฝนกระจกหลัง (2)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ ON ของสวิตช์ที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะทำการส่งสัญญาณ (LO) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณ (LO) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะส่งสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) ผ่านแผงหน้าปัดในรูปสัญญาณระบบ CAN (4)

4.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะทำการจ่ายกระแสไฟที่ขาเกตจาก CPU ภายในไปยัง FET B (5) และกระแสไฟฟ้าจะไหลผ่านจากแบตเตอรี่ไปยังมอเตอร์ที่ปัดน้ำฝนกระจกหลังและที่ปัดน้ำฝนกระจกหลังจะทำงานอย่างต่อเนื่อง (6)

หมายเหตุ
•  เมื่อบิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะเปิดใช้งาน FET A

การทำงานของระบบหยุดอัตโนมัติ

1.  สวิตช์หยุดอัตโนมัติภายในมอเตอร์ที่ปัดน้ำฝนกระจกหลังจะทำงานในลักษณะต่อไปนี้ตามตำแหน่งการทำงานของที่ปัดน้ำฝนกระจกหลัง

•  ที่ปัดน้ำฝนกระจกหลังหยุดในตำแหน่งจอด: OFF
•  ที่ปัดน้ำฝนกระจกหลังหยุดในตำแหน่งอื่นนอกเหนือจากตำแหน่งจอด: ON

2.  เมื่อสวิตช์ที่ปัดน้ำฝนกระจกหลังและที่ฉีดล้างกระจกอยู่ในตำแหน่ง OFF (1) ในระหว่างที่ที่ปัดน้ำฝนกระจกหลังทำงาน สัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังที่ส่งไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) ผ่านแผงหน้าปัดจากโมดูลควบคุมตัวถังด้านหน้า (FBCM) ปิดใช้งาน (2)

3.  เมื่อสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังปิดใช้งาน โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะปิดใช้งาน FET B (3)

4.  ที่ปัดน้ำฝนกระจกหลังจะทำงานต่อไปเนื่องจากสวิตช์หยุดอัตโนมัติอยู่ที่ตำแหน่ง ON

5.  สวิตช์หยุดอัตโนมัติในมอเตอร์ที่ปัดน้ำฝนกระจกหลังปิดใช้งานหากที่ปัดน้ำฝนกระจกหลังหยุดในตำแหน่งที่ถูกต้อง และที่ปัดน้ำฝนกระจกหลังจะหยุดทำงาน (4)

การทำงานของที่ปัดน้ำฝนแบบเป็นจังหวะ

1.  เมื่อเลื่อนสวิตช์ที่ปัดน้ำฝนกระจกหลังและที่ฉีดล้างกระจกไปที่ตำแหน่ง INT ขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ (INT) สวิตช์ของที่ปัดน้ำฝนกระจกหลัง (2)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ (INT) ของสวิตช์ที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะทำการส่งสัญญาณ (INT) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณ (INT) ตำแหน่งสวิตช์ที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะส่งสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) ผ่านแผงหน้าปัดในรูปสัญญาณระบบ CAN (4)

4.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะทำการจ่ายกระแสไฟที่ขาเกตจาก CPU ภายในไปยัง FET B (5) และกระแสไฟฟ้าจะไหลผ่านจากแบตเตอรี่ไปยังมอเตอร์ที่ปัดน้ำฝนกระจกหลังและที่ปัดน้ำฝนกระจกหลังจะทำงาน (6)

•  เมื่อบิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะเปิดใช้งาน FET A

5.  โมดูลควบคุมตัวถังด้านหน้า (FBCM) จะส่งสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) ผ่านแผงหน้าปัดอย่างสม่ำเสมอ ในกรณีนี้ ที่ปัดน้ำฝนกระจกหลังจะทำงานเป็นจังหวะ (7)

การทำงานของที่ปัดน้ำฝนและที่ฉีดน้ำล้างกระจกแบบสอดคล้องกัน

1.  เมื่อเปลี่ยนสวิตช์ที่ปัดน้ำฝนกระจกหลังและที่ฉีดล้างกระจกไปที่ตำแหน่งที่ฉีดล้างกระจกหลังขณะที่บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) (1) ชุดเปิด/ปิดการทำงานจะตรวจพบสัญญาณ ON ของสวิตช์ที่ปัดน้ำฝนกระจกหลัง (2)

2.  เมื่อชุดเปิด/ปิดการทำงานตรวจพบสัญญาณ ON ของที่ฉีดล้างกระจกหลัง อุปกรณ์จะทำการส่งสัญญาณขอการทำงานของที่ฉีดล้างกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) ในรูปสัญญาณระบบ CAN (3)

3.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณขอการทำงานของที่ฉีดล้างกระจกหลัง จะทำการจ่ายกระแสไฟอ้างอิงจาก CPU ภายในไปยังทรานซิสเตอร์ (4) และกระแสคอลเลคเตอร์จะไหลผ่านจากแหล่งจ่ายไฟภายใน (5) เพื่อเปิดใช้งานรีเลย์ของที่ฉีดล้างกระจกหลัง (6)

4.  เมื่อเปิดใช้งานรีเลย์ของที่ฉีดล้างกระจกหลังแล้ว กระแสไฟฟ้าจะไหลผ่านไปยังมอเตอร์ฉีดน้ำล้างกระจก และมอเตอร์ฉีดน้ำล้างกระจกจะทำงาน จากนั้นน้ำฉีดล้างกระจกจะฉีดออกจากหัวฉีดน้ำล้างกระจกด้านหลัง (7)

5.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณขอการทำงานของที่ฉีดล้างกระจกหลังในช่วงระยะเวลาหนึ่ง อุปกรณ์จะส่งสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลังไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) ผ่านแผงหน้าปัดในรูปสัญญาณระบบ CAN (8)

6.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณขอการทำงานของที่ปัดน้ำฝนกระจกหลัง อุปกรณ์จะทำให้ที่ปัดน้ำฝนกระจกหลังทำงาน (9)

7.  เมื่อปิดใช้งานสวิตช์ที่ฉีดล้างกระจกหลัง ที่ปัดน้ำฝนกระจกหลังจะหยุดลงหลังจากที่ทำงานประมาณ 2.6 วินาที
