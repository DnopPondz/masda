คำเตือน
•  เครื่องยนต์ที่ร้อนอาจทำให้เกิดการไหม้ที่รุนแรง ดับเครื่องยนต์และรอจนกระทั่งเครื่องยนต์เย็นก่อนการซ่อมบำรุง
ข้อควรระวัง
•  ห้ามถอดแยกแอ็คชิวเอเตอร์ไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า เนื่องจากเป็นชุดอุปกรณ์ที่มีความแม่นยำ

1.  ปลดสายขั้วลบของแบตเตอรี่ออก (ดูที่ การเชื่อมต่อ / การปลดสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การเชื่อมต่อ / การปลดสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (ไม่มีระบบ i-stop)])

2.  ถอดเพลทปิดช่องปลั๊ก (ดูที่ การถอด / การติดตั้งเพลทปิดช่องปลั๊ก [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ถอดมอเตอร์ / ตัวขับไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า (ดูที่ การถอด / การติดตั้งมอเตอร์ / ตัวขับไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ใช้มือหมุนเพลาเยื้องศูนย์กลางของแอ็คชิวเอเตอร์ไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า และตรวจสอบว่าสามารถหมุนได้อย่างราบรื่น

•  หากหมุนได้ไม่ราบรื่น ให้เปลี่ยนแอ็คชิวเอเตอร์ไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า (ดูที่ การถอด / การติดตั้งแอ็คชิวเอเตอร์ไทม์มิ่งวาล์วแปรผันแบบไฟฟ้า, แอ็คชิวเอเตอร์ไทม์มิ่งวาล์วแปรผันแบบไฮดรอลิก [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
หมายเหตุ
•  เกี่ยวนิ้วมือไว้ที่ร่องข้อต่อของเพลาเยื้องศูนย์กลางเพื่อให้หมุนเพลาง่ายขึ้น
•  เพลาเยื้องศูนย์กลางหยุดหมุนที่ตำแหน่งหน่วงสูงสุดเมื่อหมุนทวนเข็มนาฬิกาเมื่อมองจากด้านหน้า และที่ตำแหน่งล่วงหน้าสูงสุดเมื่อหมุนตามเข็มนาฬิกา
•  เพลาเยื้องศูนย์กลางหมุน 15.8 รอบจากตำแหน่งหน่วงสูงสุดจนถึงตำแหน่งล่วงหน้าสูงสุด

5.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
