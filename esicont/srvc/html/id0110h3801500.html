สายพานขับปั๊มน้ำ

หมายเหตุ
•  ไม่จำเป็นต้องตรวจสอบความตึง/การแอ่นตัวของสายพานขับปั๊มน้ำขณะใช้สายพานขับปั๊มน้ำชนิดยืด

1.  ตรวจสอบด้วยสายตาหรือสัมผัสว่าไม่มีน้ำ ความเสียหาย หรือการแตกร้าวที่ด้านในและด้านข้างของสายพานทั้งเส้น

•  หากมีน้ำ ความเสียหาย หรือการแตกร้าวเพียงเล็กน้อย ให้เปลี่ยนสายพานขับปั๊มน้ำ (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

2.  สตาร์ทเครื่องยนต์เพื่อตรวจสอบว่าไม่มีเสียงดังผิดปกติเกิดขึ้น

•  หากเกิดเสียงดังผิดปกติ ให้เปลี่ยนสายพานขับปั๊มน้ำ (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
