การถอด / การติดตั้งสวิตช์ตัวอุ่นที่นั่ง

หมายเหตุ
•  สวิตช์ตัวอุ่นที่นั่งจะรวมอยู่ในชุดควบคุมอุณหภูมิ (รุ่นที่มีระบบปรับอากาศแบบอัตโนมัติ)
•  สวิตช์ตัวอุ่นที่นั่งจะรวมอยู่ในนาฬิกา (รุ่นที่มีระบบปรับอากาศแบบธรรมดา)
•  สำหรับการถอด / การติดตั้งสวิตช์ตัวอุ่นที่นั่ง ให้ดูที่การถอด / การติดตั้งชุดควบคุมอุณหภูมิ (ดูที่ การถอด / การติดตั้งชุดควบคุมอุณหภูมิ [ระบบปรับอากาศแบบอัตโนมัติ]) (รุ่นที่มีระบบปรับอากาศแบบอัตโนมัติ)
•  สำหรับการถอด / การติดตั้งสวิตช์ตัวอุ่นที่นั่ง ให้ดูที่การถอด / การติดตั้งนาฬิกา (ดูที่ การถอด / การติดตั้งนาฬิกา) (รุ่นที่มีระบบปรับอากาศแบบอัตโนมัติ)
