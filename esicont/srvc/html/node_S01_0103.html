• ภาพตำแหน่งระบบถุงลมนิรภัย
  • คำเตือนในการซ่อมบำรุงระบบถุงลมนิรภัย
  • ข้อควรระวังในการซ่อมบำรุงระบบถุงลมนิรภัย
  • การถอด / การติดตั้งโมดูลถุงลมนิรภัยด้านคนขับ
  • การถอด / การติดตั้งโมดูลถุงลมนิรภัยด้านผู้โดยสาร
  • การถอด / การติดตั้งโมดูลถุงลมนิรภัยด้านข้าง
  • การถอด / การติดตั้งโมดูลม่านนิรภัย
  • การถอด / การติดตั้งเซ็นเซอร์ตรวจจับโซนการชน
  • การถอด / การติดตั้งโมดูลควบคุม SAS
  • การกำหนดค่าโมดูลควบคุม SAS (โดยใช้ฟังก์ชั่นการอ่าน/เขียนข้อมูล)
  • การกำหนดค่าโมดูลควบคุม SAS (โดยใช้ข้อมูลต้นฉบับ)
  • การถอด / การติดตั้งเซ็นเซอร์ถุงลมนิรภัยด้านข้าง NO.1
  • การถอด / การติดตั้งเซ็นเซอร์ถุงลมนิรภัยด้านข้าง NO. 2
  • การถอด / การติดตั้งคล็อคสปริง
  • การปรับคล็อคสปริง
  • การตรวจสอบคล็อคสปริง
  • การถอด / การติดตั้งสวิตช์ปิดการทำงานของถุงลมนิรภัยด้านผู้โดยสาร (PAD)
  • การตรวจสอบสวิตช์ปิดการทำงานของถุงลมนิรภัยด้านผู้โดยสาร (PAD)
  • ขั้นตอนการสั่งงานโมดูลถุงลมนิรภัยและเข็มขัดนิรภัยระบบดึงกลับอัตโนมัติ
  • ขั้นตอนการกำจัดทิ้งโมดูลถุงลมนิรภัยและเข็มขัดนิรภัยระบบดึงกลับอัตโนมัติ
  • การตรวจสอบ SST (เครื่องมือการสั่งงาน)
