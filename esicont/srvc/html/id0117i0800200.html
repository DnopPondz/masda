คำเตือน
•  ถอดและติดตั้งชิ้นส่วนทั้งหมดเมื่อเครื่องยนต์เย็นตัวลงแล้ว มิฉะนั้นอาจทำให้โดนลวกหรือบาดเจ็บรุนแรงได้
•  เมื่อสายไฟแบตเตอรี่ต่ออยู่ การสัมผัสตัวถังรถกับขั้ว B ของเจนเนอร์เรเตอร์จะทำให้เกิดประกายไฟ ซึ่งอาจทำให้เกิดการบาดเจ็บ ไฟไหม้ และความเสียหายต่ออุปกรณ์ไฟฟ้าได้ ให้ปลดสายขั้วลบแบตเตอรี่ก่อนปฏิบัติตามการทำงานดังต่อไปนี้ทุกครั้ง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดฝาครอบใต้เครื่องด้านหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

3.  ถอดสายพานขับเจนเนอร์เรเตอร์ (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ถอดคอมเพรสเซอร์เครื่องปรับอากาศโดยที่ท่อสารทำความเย็นยังต่ออยู่และรัดไว้โดยใช้สายไฟหรือเชือกเพื่อไม่ให้กีดขวาง (ดูที่ การถอด / การติดตั้งคอมเพรสเซอร์เครื่องปรับอากาศ)

5. ถอดตามลำดับที่ระบุไว้ในตาราง

6. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

หมายเหตุการถอดเจอเนอร์เรเตอร์

1.  คลายโบลท์ตัวล่างของเจนเนอร์เรเตอร์ออกให้สุดและดึงออกไปจนสัมผัสกับโครงตัวถัง

หมายเหตุ
•  สามารถถอดเจนเนอร์เรเตอร์ออกได้โดยไม่ต้องถอดโบลท์ตัวล่างออกจากเครื่องยนต์

2.  ถอดโบลท์ตัวบนของเจนเนอร์เรเตอร์

3.  ถอดเจนเนอร์เรเตอร์พร้อมกับโบลท์ตัวล่างออกจากเครื่องยนต์

4.  ถอดเจนเนอร์เรเตอร์ออกจากด้านบนห้องเครื่องยนต์

หมายเหตุการติดตั้งตัวปรับความตึงสายพานขับอัตโนมัติ

ข้อควรระวัง
•  ให้ไล่อากาศออกจากตัวปรับความตึงสายพานขับอัตโนมัติโดยใช้ขั้นตอนต่อไปนี้หลังจากประกอบตัวปรับความตึงสายพานขับอัตโนมัติเพื่อให้ตัวปรับความตึงสายพานขับอัตโนมัติทำงานอย่างถูกต้อง

1.  ติดตั้งตัวปรับความตึงสายพานขับอัตโนมัติเข้ากับเครื่องยนต์

ข้อควรระวัง
•  อย่าใช้แรงมากเกินไปหลังจากตัวปรับความตึงเลื่อนไปที่ระยะชักสุดเพื่ือป้องกันไม่ให้ตัวปรับความตึงสายพานขับอัตโนมัติเสียหาย

2.  สวมประแจเข้ากับส่วนที่เป็นหกเหลี่ยมบนตัวปรับความตึงสายพานขับอัตโนมัติ เลื่อนตัวปรับความตึงสายพานขับอัตโนมัติขึ้นและลงที่ระยะชักสุดสามครั้งเพื่อไล่อากาศ
