การตรวจสอบระบบชิฟท์ล็อค

1.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

2.  เลื่อนคันเกียร์ไปที่ตำแหน่ง P

3.  ตรวจสอบว่าคันเกียร์ไม่สามารถเลื่อนจากตำแหน่ง P ไปที่ R ได้เมื่อปล่อยแป้นเบรก

•  ถ้าเป็นไปตามที่กำหนด ให้ไปที่ขั้นตอนถัดไป
•  ถ้าไม่เป็นไปตามที่กำหนด ให้ตรวจสอบชิ้นส่วนต่อไปนี้:
―  โซลินอยด์ชิฟท์ล็อค (ดูที่ การตรวจสอบระบบชิฟท์ล็อค)
―  สวิตช์ไม่ใช่ตำแหน่ง P (ดูที่ การตรวจสอบสวิตช์ไม่ใช่ตำแหน่ง P)
―  สวิตช์ไฟเบรก (ดูที่ การตรวจสอบสวิตช์ไฟเบรก)
―  ชุดเปิด/ปิดการทำงาน (ดูที่ การตรวจสอบชุดเปิด/ปิดการทำงาน)
―  PCM (ดูที่ การตรวจสอบ PCM [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การตรวจสอบ PCM [SKYACTIV-D 2.2])

4.  ตรวจสอบว่าคันเกียร์สามารถเลื่อนจากตำแหน่ง P ไปที่ R ได้เมื่อเหยียบแป้นเบรก

•  ถ้าเป็นไปตามที่กำหนด ให้ไปที่ขั้นตอนถัดไป
•  ถ้าไม่เป็นไปตามที่กำหนด ให้ตรวจสอบชิ้นส่วนต่อไปนี้:
―  โซลินอยด์ชิฟท์ล็อค (ดูที่ การตรวจสอบระบบชิฟท์ล็อค)
―  สวิตช์ไม่ใช่ตำแหน่ง P (ดูที่ การตรวจสอบสวิตช์ไม่ใช่ตำแหน่ง P)
―  สวิตช์ไฟเบรก (ดูที่ การตรวจสอบสวิตช์ไฟเบรก)
―  ชุดเปิด/ปิดการทำงาน (ดูที่ การตรวจสอบชุดเปิด/ปิดการทำงาน)
―  PCM (ดูที่ การตรวจสอบ PCM [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การตรวจสอบ PCM [SKYACTIV-D 2.2])


การตรวจสอบโซลินอยด์ชิฟท์ล็อค

หมายเหตุ
•  โซลินอยด์ชิฟท์ล็อคติดตั้งอยู่ในชิ้นส่วนประกอบคันเกียร์

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดคอนโซลหน้า (ดูที่ การถอด / การติดตั้งคอนโซลหน้า)

3.  ต่อสายขั้วลบแบตเตอรี่กลับเข้าที่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

4.  ตรวจสอบว่าแรงดันไฟฟ้าของแต่ละขั้วของชิ้นส่วนประกอบคันเกียร์เป็นไปตามที่ระบุในตาราง

การตรวจสอบปุ่มปลดชิฟท์ล็อค

1.  บิดสวิตช์สตาร์ท OFF

2.  ตรวจสอบว่าคันเกียร์อยู่ในตำแหน่ง P

3.  ขณะที่ไม่เหยียบแป้นเบรก ให้ตรวจสอบว่าคันเกียร์ไม่สามารถเลื่อนออกจากตำแหน่ง P ได้
