2.  ปลดสายขั้วลบแบตเตอรี่

3.  ถอดชิ้นส่วนต่อไปนี้:

(1)  บังโคลนหน้า (ดูที่ การถอด / การติดตั้งขอบบังโคลนหน้า)
(2)  กันชนหน้า (ดูที่ การถอด / การติดตั้งกันชนหน้า)
(3)  ไฟหน้ารวม (ดูที่ การถอด / การติดตั้งไฟหน้ารวม)
(4)  ตัวเลื่อนกันชนหน้า (ดูที่ การถอด / การติดตั้งตัวเลื่อนกันชนหน้า)
(5)  แผงบังโคลนหน้า (ดูที่ การถอด / การติดตั้งแผงบังโคลนหน้า)

4.  คลายโบลท์และปรับตั้งฝากระโปรงหน้า

5.  ขันโบลท์ให้แน่น (ดูที่ การถอด / การติดตั้งฝากระโปรงหน้า)

6.  ติดตั้งชิ้นส่วนต่อไปนี้:

(1)  แผงบังโคลนหน้า (ดูที่ การถอด / การติดตั้งแผงบังโคลนหน้า)
(2)  ตัวเลื่อนกันชนหน้า (ดูที่ การถอด / การติดตั้งตัวเลื่อนกันชนหน้า)
(3)  ไฟหน้ารวม (ดูที่ การถอด / การติดตั้งไฟหน้ารวม)
(4)  กันชนหน้า (ดูที่ การถอด / การติดตั้งกันชนหน้า)
(5)  บังโคลนหน้า (ดูที่ การถอด / การติดตั้งขอบบังโคลนหน้า)

7.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / การต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

8.  ปรับลำแสงไฟหน้า (ดูที่ การปรับไฟหน้า)

9.  ปรับระดับไฟตัดหมอกหน้า รุ่นที่มีไฟตัดหมอกหน้า) (ดูที่ การปรับไฟตัดหมอกหน้า)
