การถอด / การติดตั้งกล้องจับภาพด้านหลัง

ข้อควรระวัง
•  ชุดเครื่องเสียงจะสื่อสารกับกล้องจับภาพด้านหลังและบันทึกการมีอยู่/การไม่มีอยู่ของกล้องจับภาพด้านหลังเมื่อบิดสวิตช์สตาร์ทไปที่ ACC หรือ ON (เครื่องยนต์ทำงานหรือไม่ทำงาน) หลังจากต่อสายแบตเตอรี่ขั้วลบ
•  ถ้าสวิตช์คำสั่งได้รับการติดตั้งโดยที่สายแบตเตอรี่ขั้วลบต่ออยู่ขณะที่ชุดเครื่องเสียงบันทึกว่าไม่มีกล้องจับภาพด้านหลัง กล้องจับภาพด้านหลังจะไม่ทำงาน
•  สำหรับการติดตั้ง / การถอดกล้องจับภาพด้านหลัง ให้ปลดสายแบตเตอรี่ขั้วลบ บิดสวิตช์สตาร์ทไปที่ ACC หรือ ON (เครื่องยนต์ทำงานหรือไม่ทำงาน) หลังการเข้ารับบริการเสร็จสมบูรณ์ และชุดเครื่องเสียงจะบันทึกการมีอยู่/การไม่มีอยู่ของกล้องจับภาพด้านหลัง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดส่วนบนประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนบนประตูท้าย)
(2)  แผงปิดด้านข้างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างประตูท้าย)
(3)  แผงปิดส่วนล่างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนล่างประตูท้าย)
(4)  แผ่นปิดประตูท้าย (ดูที่ การถอด / การติดตั้งแผ่นปิดประตูท้าย)

3.  กดแถบยึดกล้องจับภาพด้านหลังในทิศทางตามลูกศร (1) ที่แสดงไว้ในรูปภาพ ดึงกล้องจับภาพด้านหลังในทิศทางตามลูกศร (2) ที่แสดงไว้ในรูปภาพ และปลดแถบยึดกล้องจับภาพด้านหลังออกจากแผ่นปิดประตูท้าย

4.  ถอดกล้องจับภาพด้านหลัง

5.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

6.  ทำการตั้งค่าเริ่มต้นระบบช่วยจอด (การปรับเทียบ) (รุ่นสเปคยุโรป (ประเทศอังกฤษ L.H.D.)) (ดูที่ การตั้งค่าเริ่มต้นระบบช่วยจอด (การปรับเทียบ))
