การถอด / การติดตั้งสายปลดล็อคฝากระโปรงหน้า

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  กันชนหน้า (ดูที่ การถอด / การติดตั้งกันชนหน้า)
(2)  ไฟหน้ารวม (ด้านคนขับ) (ดูที่ การถอด / การติดตั้งไฟหน้ารวม)
(3)  ขอบบังโคลนหน้า (ด้านคนขับ) (ดูที่ การถอด / การติดตั้งขอบบังโคลนหน้า)
(4)  ยางกันโคลนด้านหน้า (ด้านคนขับ) (ดูที่ การถอด / การติดตั้งยางกันโคลน)
(5)  ตัวปลดฝากระโปรง (ดูที่ การถอด / การติดตั้งล็อคฝากระโปรงหน้าและตัวปลดฝากระโปรง)

3.  ปลดสายปลดล็อคฝากระโปรงหน้าออกจากล็อคฝากระโปรงหน้า

4.  ขณะกดแถบยึดของคลิปในทิศทางตามลูกศร (1) ที่แสดงไว้ในรูปภาพ ให้ดึงคลิปในทิศทางตามลูกศร (2) เพื่อปลดออกจากตัวถัง

5.  ถอดคลิปออกจากตัวอุปกรณ์

6.  กดปลอกยึดลงในด้านห้องผู้โดยสาร

7.  ถอดสายปลดล็อคฝากระโปรงหน้าออกจากด้านห้องผู้โดยสาีร

8.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
