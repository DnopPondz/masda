การถอด

1.  เลื่อนคันเกียร์ไปที่ตำแหน่งเกียร์ว่าง

2.  ถอดแผ่นปิดช่องหัวเทียน (ดูที่ การถอด / การติดตั้งแผ่นปิดช่องหัวเทียน [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

4.  ถอดกรองอากาศและท่อยางอากาศออกเป็นชุด (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

5.  ถอดแบตเตอรี่และถาดรองแบตเตอรี่ (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
