การตรวจสอบเซ็นเซอร์ A/F

1.  ต่อ M-MDS เข้ากับ DLC-2

2.  สตาร์ทเครื่องยนต์ และอุ่นเครื่องจนถึงอุณหภูมิการทำงาน

3.  เข้า PID ต่อไปนี้โดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

• VSS (ความเร็วรถ)
•  RPM (ความเร็วรอบเครื่องยนต์)
•  O2S11 (กระแสไฟฟ้าเอาท์พุทเซ็นเซอร์ A/F)

4.  ขับรถและลดความเร็วรอบเครื่องยนต์โดยการปล่อยแป้นคันเร่งสุดเมื่อความเร็วรอบเครื่องยนต์อยู่ที่ 3,000 rpm หรือสูงกว่า

5.  ตรวจสอบว่ากระแสไฟฟ้าเอาท์พุทเซ็นเซอร์ A/F (PID: O2S11) อยู่ที่ 0.25 mA หรือมากกว่าในขณะลดความเร็วดังแสดงในภาพหรือไม่

•  ถ้าไม่เป็นไปตามที่กำหนด ให้เปลี่ยนเซ็นเซอร์ A/F (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อัตราส่วนผสมอากาศกับน้ำมันเชื้อเพลิง (A/F) [SKYACTIV-D 2.2])


การตรวจสอบฮีทเตอร์ A/F เซ็นเซอร์

คำเตือน
•  เครื่องยนต์และระบบไอเสียที่ร้อนสามารถทำให้เกิดการบาดเจ็บรุนแรงจากการโดนไหม้ได้ ดับเครื่องยนต์และรอจนกว่าจะเย็นตัวลงก่อนถอดระบบไอเสีย
ข้อควรระวัง
•  อย่าให้วัตถุไวไฟ เช่น สายรัดชุดสายไฟสัมผัสกับอุปกรณ์แปรสภาพก๊าซไอเสีย

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ยกรถขึ้น

3.  ถอดชิ้นส่วนต่อไปนี้:

(1)  ฝาครอบใต้เครื่องด้านหน้า No.2 (2WD) (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)
(2)  แผ่นบังโคลน (ดูที่ การถอด / การติดตั้งแผ่นบังโคลน)
(3)  ซีลยาง (No.1) และฝาครอบซับเสียง (No.1) (ดูที่ การถอด / การติดตั้งโซ่ไทม์มิ่ง [SKYACTIV-D 2.2])

4.  ปลดขั้วต่อเซ็นเซอร์ A/F

5.  วัดค่าความต้านทานระหว่างขั้ว C และ D ของเซ็นเซอร์ A/F
