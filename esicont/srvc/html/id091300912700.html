ข้อควรระวัง
•  เมื่อถอดหรือใส่เบาะนั่งด้านหลังกลับเข้าที่ การสัมผัสกับบริเวณรอบๆ ตัวรถอาจทำให้เกิดรอยขีดข่วนและได้รับความเสียหาย เมื่อต้องถอดหรือใส่เบาะนั่งด้านหลังกลับเข้าที่ ให้ใช้เจ้าหน้าที่สองคนขึ้นไปในการดำเนินการทำงานนี้ทุกครั้งและต้องระวังไม่ให้เกิดรอยขีดข่วนหรือเกิดความเสียหายกับเบาะนั่งด้านหลังและบริเวณรอบๆ ของรถ
•  เมื่อดำเนินการขั้นตอนขณะถอดเบาะนั่งด้านหลังออกจากรถ ให้ดำเนินการขั้นตอนบนผ้าสะอาดเพื่อไม่ให้ที่นั่งเสียหายหรือเปื้อน

แบบแยก 6:4

1.  ถอดเบาะนั่งด้านหลัง (ดูที่ การถอด / การติดตั้งเบาะนั่งด้านหลัง)

2.  เลื่อนตะขอ A ตามลำดับ (1), (2) ที่แสดงไว้ในรูปภาพเพื่อปลดตะขอออกจากแผ่นรองเบาะนั่งด้านหลัง

3.  พลิกขอบปิดเบาะนั่งด้านหลังบางส่วนขึ้นจากแผ่นรองเบาะที่นั่งด้านหลัง ถอดแหวน C ตามลำดับ (3), (4), (5), (6), (7), (8), (9), (10) ที่แสดงไว้ในรูปภาพ

ข้อควรระวัง
•  หากนำแหวน C ที่ได้รับความเสียหายกลับมาใช้ใหม่ อาจไม่สามารถติดตั้งขอบปิดเบาะนั่งด้านหลังและแผ่นรองเบาะนั่งด้านหลังได้อย่างถูกต้อง เมื่อต้องถอดและติดตั้งแหวน C ควรระมัดระวังอย่าทำให้แหวนเสียหาย
•  หากแหวน C ได้รับความเสียหาย ให้เปลี่ยนเป็นอันใหม่

4.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
