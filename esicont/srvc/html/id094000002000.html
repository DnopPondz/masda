การถอด / การติดตั้งโมดูลควบคุมตัวถังด้านหลัง (RBCM)

id094000002000

ข้อควรระวัง
•   ถ้าไม่ทำการตั้งค่าเมื่อเปลี่ยนโมดูลควบคุมตัวถังด้านหลัง (RBCM) ใหม่ ข้อมูลจำเพาะของรถจะไม่ถูกเก็บบันทึกไว้ในโมดูลควบคุมตัวถังด้านหลัง (RBCM) และระบบจะทำงานผิดปกติ
•  เมื่อทำการตั้งค่า จำเป็นต้องอ่านข้อมูลจำเพาะของรถจากโมดูลควบคุมตัวถังด้านหลัง (RBCM) ก่อนการเปลี่ยน ต่อ M-MDS เข้ากับรถ และทำการระบุรถก่อนทำการถอดโมดูลควบคุมตัวถังด้านหลัง (RBCM) ข้อมูลจำเพาะของรถจะถูกเก็บบันทึกไว้ใน M-MDS ชั่วคราว
หมายเหตุ
•  โมดูลควบคุมตัวถังด้านหลัง (RBCM) ก่อนการเปลี่ยนจะเก็บบันทึกข้อมูลจำเพาะของรถ
•  โมดูลควบคุมตัวถังด้านหลัง (RBCM) ตัวใหม่ไม่เก็บบันทึกข้อมูลจำเพาะของรถใดๆ
•  ถ้าไม่สามารถอ่านข้อมูลจำเพาะของรถจากโมดูลควบคุมตัวถังด้านหลัง (RBCM) ก่อนการเปลี่ยนได้ ให้ทำการตั้งค่าโดยใช้ข้อมูลต้นฉบับ

1.  เมื่อเปลี่ยนโมดูลควบคุมตัวถังด้านหลัง (RBCM) ให้ทำการตั้งค่า (ดูที่ การตั้งค่าโมดูลควบคุมตัวถังด้านหลัง (RBCM) (โดยใช้ฟังก์ชั่นการอ่าน / เขียนข้อมูล))

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ถอดชิ้นส่วนต่อไปนี้:

1.  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
2.  แผงปิดแผงตั้งท้าย (ดูที่ การถอด / การติดตั้งแผงปิดแผงตั้งท้าย)
3.  แผ่นกันรอยขีดข่วนด้านหลัง (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
4.  แผงปิดข้างห้องเก็บสัมภาระ (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผงปิดข้างห้องเก็บสัมภาระ)
