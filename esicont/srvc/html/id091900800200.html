1.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน)

2.  เปิดประตูด้านคนขับ

3.  บิดสวิตช์สตาร์ทไปที่ OFF (LOCK)

4.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

5.  ถอดฝาครอบคอพวงมาลัย (ดูที่ การถอด / การติดตั้งฝาครอบคอพวงมาลัย)
