การตรวจสอบเซ็นเซอร์อุณหภูมิก๊าซไอเสีย [SKYACTIV-D 2.2]

id0140z7446300


เซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1

คำเตือน
•  เครื่องยนต์และระบบไอเสียที่ร้อนสามารถทำให้เกิดการบาดเจ็บรุนแรงจากการโดนไหม้ได้ ดับเครื่องยนต์และรอจนกว่าจะเย็นตัวลงก่อนถอดระบบไอเสีย
ข้อควรระวัง
•  ทำการถอด / ติดตั้งเซ็นเซอร์อุณหภูมิก๊าซไอเสียเฉพาะเมื่อจำเป็นต้องตรวจสอบหรือเปลี่ยนใหม่ เช่น เมื่อรหัสวิเคราะห์ปัญหาของเซ็นเซอร์ปรากฏขึ้น เนื่องจากอาจทำให้เกิดก๊าซไอเสียรั่ว
•  ถ้าทำการถอด หรือการถอด / การติดตั้ง ให้ตรวจสอบว่าไม่มีการรั่วของก๊าซไอเสียหลังจากการติดตั้งกลับเข้าที่

การตรวจสอบค่าความต้านทาน

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดแบตเตอรี่ (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ถอดท่ออากาศและท่อยางอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

4.  ปลดขั้วต่อเซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1

5.  ถอดเซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิก๊าซไอเสีย [SKYACTIV-D 2.2])

6.  วางเซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1 ลงในน้ำพร้อมกับเทอร์โมมิเตอร์ และค่อยๆ ให้ความร้อนน้ำ

7.  วัดค่าความต้านทานระหว่างขั้ว A และ B ของเซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1

•  ถ้าไม่เป็นไปตามที่กำหนด ให้เปลี่ยนเซ็นเซอร์อุณหภูมิก๊าซไอเสีย No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิก๊าซไอเสีย [SKYACTIV-D 2.2])
