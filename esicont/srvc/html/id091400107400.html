ระบบกุญแจรีโมทอัจฉริยะ

id091400107400


รายละเอียด

•  ระบบกุญแจรีโมทอัจฉริยะมีการนำมาใช้เพื่อทำการอนุญาตตัวส่งสัญญาณรีโมทที่พกพาและล็อค/ปลดล็อคประตูโดยอัตโนมัติ
•  ชุดเปิด/ปิดการทำงานจะสั่งงานโหมด Fail-safe ของระบบกุญแจรีโมทอัจฉริยะ (ดูที่ ชุดเปิด/ปิดการทำงาน)

การทำงาน

•  ระบบกุญแจรีโมทอัจฉริยะจะทำการอนุญาตตัวส่งสัญญาณรีโมทเมื่อมีการดำเนินการดังต่อไปนี้
―  การล็อค/ปลดล็อคโดยใช้สวิตช์ขอการทำงานบนประตูด้านนอก
―  การล็อคโดยใช้สวิตช์ขอการทำงานบนประตูท้าย
―  การล็อค/ปลดล็อคโดยใช้ตัวส่งสัญญาณรีโมท
―  การเปิดประตูท้าย
•  ระบบกุญแจรีโมทอัจฉริยะจะมีฟังก์ชั่นต่อไปนี้ในกรณีที่ผู้ใช้สั่งการระบบไม่ถูกต้อง หรือเพื่อป้องกันไม่ให้ผู้ใช้ออกจากรถโดยไม่ได้ล็อครถ
―  ฟังก์ชั่นการล็อคซ้ำอัตโนมัติ: หลังจากการปลดล็อค หากประตูใดๆ หรือประตูท้ายไม่ถูกเปิด และผ่านระยะเวลาตามที่ตั้งค่าไว้ของแต่ละบุคคล ประตูทุกบานจะถูกล็อคโดยอัตโนมัติ
― ฟังก์ชั่นการล็อคอัตโนมัติเมื่อออกนอกพื้นที่ (พื้นที่รับสัญญาณ): หลังจากประตูทุกบานถูกปิด เมื่อตัวส่งสัญญาณรีโมทถูกนำออกจากพื้นที่รับสัญญาณของเสาอากาศกุญแจรีโมท ประตูทุกบานจะถูกล็อคโดยอัตโนมัติ
―  ฟังก์ชั่นหยุดการทำงานตัวส่งสัญญาณรีโมท: หากประตูถูกล็อคขณะที่ตัวส่งสัญญาณอยู่ภายในรถ ฟังก์ชั่นของตัวส่งสัญญาณรีโมทที่อยู่ภายในรถจะหยุดการทำงานชั่วคราว
―  ฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถ: หากประตูท้ายถูกปิดขณะที่ตัวส่งสัญญาณรีโมทอยู่ภายในรถ ประตูท้ายจะสามารถเปิดโดยการกดสวิตช์เปิดประตูท้าย
―  ฟังก์ชั่นการแจ้งเตือน: หากระบบทำงานไม่ถูกต้องซึ่งอาจทำให้เกิดปัญหาขึ้น ระบบจะเตือนผู้ใช้โดยการใช้ไฟแสดงสถานะ ไฟเตือน หน้าจอ LCD และเสียงเตือนในแผงหน้าปัด หรือโดยการใช้สัญญาณเตือนของกุญแจรีโมท

ฟังก์ชั่นแสดงสถานะแรงดันไฟแบตเตอรี่ตัวส่งสัญญาณรีโมทต่ำ

•  ฟังก์ชั่นนี้จะส่งสัญญาณไปยังผู้ใช้ว่าแรงดันไฟแบตเตอรี่ของตัวส่งสัญญาณรีโมทต่ำ

การล็อค/ปลดล็อคโดยใช้สวิตช์ขอการทำงานที่มือจับด้านนอก

หมายเหตุ
•  การล็อค/ปลดล็อคจะดำเนินการเมื่อกดสวิตช์ขอการทำงานบนมือจับด้านนอกขณะตรวจพบสภาวะต่อไปนี้ทั้งหมด
―  สวิตช์สตาร์ทอยู่ที่ OFF (LOCK)
―  ประตูรถทุกบานปิดอยู่
ตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณที่ด้านนอกรถ

1.  เมื่อสวิตช์ขอการทำงานที่ด้านคนขับหรือผู้โดยสารถูกกด สัญญาณการเปิดสวิตช์ขอการทำงานจะถูกส่งไปยังชุดเปิด/ปิดการทำงานผ่านชุดควบคุมด้านหน้าซ้าย

2.  ชุดเปิด/ปิดการทำงานจะส่งสัญญาณขอไปยังชุดควบคุมด้านหน้าซ้าย เพื่อตรวจสอบว่าตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณหรือไม่ ขึ้นอยู่กับสัญญาณการเปิดสวิตช์ขอการทำงาน

3.  ชุดควบคุมด้านหน้าซ้ายจะส่งสัญญาณขอจากเสาอากาศกุญแจรีโมทของประตูที่สวิตช์ขอการทำงานถูกกด และเสาอากาศกุญแจรีโมทภายในรถทั้งหมด

4.  ตัวส่งสัญญาณรีโมทจะได้รับสัญญาณขอการทำงานจากเสาอากาศด้านนอก และส่งข้อมูล ID ไปยังตัวรับสัญญาณกุญแจรีโมท

5.  ตัวรับสัญญาณกุญแจรีโมทจะส่งข้อมูล ID ที่ได้รับไปยังชุดเปิด/ปิดการทำงาน

6.  ชุดเปิด/ปิดการทำงานจะตรวจสอบข้อมูล ID และหากมีการระบุว่าตัวส่งสัญญาณรีโมทที่โปรแกรมไว้อยู่ด้านนอกรถ จะมีการส่งสัญญาณการล็อค/ปลดล็อคไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) และในเวลาเดียวกันจะส่งสัญญาณขอสัญญาณเตือนไปยังชุดควบคุมด้านหน้าซ้าย

7.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณล็อค/ปลดล็อค จะมีการสั่งการมอเตอร์แอ็คชิวเอเตอร์ล็อคประตูหน้า/หลังเพื่อล็อค/ปลดล็อคประตูหน้า/หลัง

หมายเหตุ
•  ระบบจะใช้เวลาประมาณ 1—2 วินาทีตั้งแต่เมื่อสวิตช์ขอการทำงานถูกกดจนกระทั่งการล็อค/ปลดล็อคเสร็จสมบูรณ์

8.  ในช่วงเวลาเดียวกับการล็อค/ปลดล็อคประตู ระบบจะส่งสัญญาณขอการกะพริบไฟฉุกเฉินไปยังแผงหน้าปัดเป็นสัญญาณระบบ CAN

9.  เมื่อชุดควบคุมด้านหน้าซ้ายได้รับสัญญาณขอเสียงสัญญาณเตือน จะมีการสั่งการเสียงสัญญาณเตือนของกุญแจรีโมทให้มีจำนวนครั้งดังต่อไปนี้:

―  เมื่อล็อค: สัญญาณเตือนกุญแจรีโมทดังหนึ่งครั้ง
―  เมื่อปลดล็อค: สัญญาณเตือนกุญแจรีโมทดังสองครั้ง

10.  แผงหน้าปัดจะส่งสัญญาณขอการกะพริบไฟฉุกเฉินไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) เป็นสัญญาณระบบ CAN

11.  เมื่อโมดูลควบคุมตัวถังด้านหน้า (FBCM) ได้รับสัญญาณขอการกะพริบไฟฉุกเฉิน จะมีการควบคุมไฟฉุกเฉินเพื่อให้กะพริบตามจำนวนครั้งดังต่อไปนี้เพื่อแจ้งว่าการล็อค/ปลดล็อคเสร็จสมบูรณ์

―  เมื่อล็อค: ไฟฉุกเฉินกะพริบหนึ่งครั้ง
―  เมื่อปลดล็อค: ไฟฉุกเฉินกะพริบสองครั้ง

หมายเหตุ
•  การล็อคจะดำเนินการเมื่อสวิตซ์ขอการทำงานบนประตูท้ายถูกกดขณะตรวจพบสภาวะต่อไปนี้ทั้งหมด
―  สวิตช์สตาร์ทอยู่ที่ OFF (LOCK)
―  ประตูรถทุกบานและประตูท้ายถูกปิด
―  ตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณที่ด้านนอกรถ

1.  เมื่อสวิตช์ขอการทำงานถูกกด สัญญาณการเปิดสวิตซ์ขอการทำงานจะถูกส่งไปยังชุดเปิด/ปิดการทำงานผ่านชุดควบคุมด้านหน้าซ้าย

2.  ชุดเปิด/ปิดการทำงานจะส่งสัญญาณขอไปยังชุดควบคุมด้านหน้าซ้าย เพื่อตรวจสอบว่าตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณหรือไม่ ขึ้นอยู่กับสัญญาณการเปิดสวิตช์ขอการทำงาน

3.  ชุดควบคุมด้านหน้าซ้ายจะส่งสัญญาณขอการทำงานจากเสาอากาศกุญแจรีโมทด้านนอก (ด้านหลัง) และเสาอากาศกุญแจรีโมททั้งหมดภายในรถ

4.  ตัวส่งสัญญาณรีโมทจะได้รับสัญญาณขอการทำงานจากเสาอากาศด้านนอก และส่งข้อมูล ID ไปยังตัวรับสัญญาณกุญแจรีโมท

6.  ชุดเปิด/ปิดการทำงานจะตรวจสอบข้อมูล ID และหากมีการระบุว่าตัวส่งสัญญาณรีโมทที่โปรแกรมไว้อยู่ด้านนอกรถ จะมีการส่งสัญญาณการล็อคไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) และในเวลาเดียวกันจะส่งสัญญาณขอสัญญาณเตือนไปยังชุดควบคุมด้านหน้าซ้าย

7.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณการล็อค จะมีการสั่งการมอเตอร์แอ็คชิวเอเตอร์ล็อคประตูหน้า/หลังเพื่อล็อคประตูหน้า/หลัง

หมายเหตุ
•  การล็อค/ปลดล็อคจะถูกดำเนินการเมื่อปุ่มล็อคของตัวส่งสัญญาณรีโมทถูกกดขณะตรวจพบสภาวะต่อไปนี้ทั้งหมด
―  สวิตช์สตาร์ทอยู่ที่ OFF (LOCK)
―  ประตูรถทุกบานปิดอยู่
―  ตัวส่งสัญญาณรีโมทอยู่ภายในช่วงที่สามารถสั่งการได้

1.  เมื่อปุ่มล็อคหรือปลดล็อคของตัวส่งสัญญาณถูกกด ตัวส่งสัญญาณรีโมทจะส่งข้อมูล ID

2.  เมื่อตัวรับสัญญาณกุญแจรีโมทได้รับข้อมูล ID จะส่งข้อมูลไปยังชุดเปิด/ปิดการทำงาน

3.  ชุดเปิด/ปิดการทำงานจะตรวจสอบข้อมูล ID และหากข้อมูล ID ตรงกัน จะมีการส่งสัญญาณการล็อค/ปลดล็อคไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) และในเวลาเดียวกันจะส่งสัญญาณขอสัญญาณเตือนไปยังชุดควบคุมด้านหน้าซ้าย

4.  โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะสั่งการมอเตอร์แอ็คชิวเอเตอร์ล็อคประตูหน้า/หลังเพื่อล็อค/ปลดล็อคประตูหน้า/หลัง

5.  ในช่วงเวลาเดียวกับการล็อค/ปลดล็อคประตู จะส่งสัญญาณขอการกะพริบไฟฉุกเฉินไปยังแผงหน้าปัดเป็นสัญญาณระบบ CAN

6.  ชุดควบคุมด้านหน้าซ้ายจะสั่งการสัญญาณเตือนกุญแจรีโมทให้มีจำนวนครั้งดังต่อไปนี้:

―  เมื่อล็อค: สัญญาณเตือนกุญแจรีโมทดังหนึ่งครั้ง
―  เมื่อปลดล็อค: สัญญาณเตือนกุญแจรีโมทดังสองครั้ง

7.  แผงหน้าปัดจะส่งสัญญาณขอการกะพริบไฟฉุกเฉินไปยังโมดูลควบคุมตัวถังด้านหน้า (FBCM) เป็นสัญญาณระบบ CAN

8.  โมดูลควบคุมตัวถังด้านหน้า (FBCM) จะควบคุมการกะพริบไฟฉุกเฉินให้มีจำนวนครั้งดังต่อไปนี้ เพื่อแจ้งว่าการล็อค/ปลดล็อคเสร็จสมบูรณ์

―  เมื่อล็อค: ไฟฉุกเฉินกะพริบหนึ่งครั้ง
―  เมื่อปลดล็อค: ไฟฉุกเฉินกะพริบสองครั้ง

1.  เมื่อสวิตช์เปิดประตูท้ายถูกกด สัญญาณเปิดประตูท้ายจะถูกส่งไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM)

2.  โมดูลควบคุมตัวถังด้านหลัง (RBCM) จะส่งสัญญาณเปิดประตูท้ายไปยังชุดเปิด/ปิดการทำงาน

3.  ชุดเปิด/ปิดการทำงานจะส่งสัญญาณขอไปยังชุดควบคุมด้านหน้าซ้าย เพื่อตรวจสอบว่าตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณหรือไม่ ขึ้นอยู่กับสัญญาณการเปิดประตูท้าย

4.  ชุดควบคุมด้านหน้าซ้ายจะส่งสัญญาณขอการทำงานโดยใช้เสาอากาศกุญแจรีโมทด้านในกันชนหลัง ขึ้นอยู่กับสัญญาณขอการทำงานจากชุดเปิด/ปิดการทำงาน

5.  ตัวส่งสัญญาณรีโมทจะได้รับสัญญาณขอการทำงานจากเสาอากาศ (ด้านนอก, ด้านหลัง) และส่งข้อมูล ID ไปยังตัวรับสัญญาณกุญแจรีโมท

6.  ตัวรับสัญญาณกุญแจรีโมทจะส่งข้อมูล ID ที่ได้รับไปยังชุดเปิด/ปิดการทำงาน

7.  ชุดเปิด/ปิดการทำงานจะตรวจสอบข้อมูล ID และหากข้อมูล ID ตรงกัน จะมีการส่งสัญญาณปลดประตูท้ายไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM)

8.  เมื่อโมดูลควบคุมตัวถังด้านหลัง (RBCM) ได้รับสัญญาณปลดประตูท้าย จะมีการสั่งการมอเตอร์แอ็คชิวเอเตอร์ล็อประตูท้ายเพื่อปลดประตูท้าย

การทำงานของฟังก์ชั่นการล็อคซ้ำอัตโนมัติ

•  หลังจากชุดเปิด/ปิดการทำงานส่งสัญญาณขอปลดล็อคไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) หากการสั่งการต่อไปนี้ไม่ถูกดำเนินการภายใน 30 วินาที*1 ระบบจะส่งสัญญาณขอการล็อคไปยังโมดูลควบคุมตัวถังด้านหลัง (RBCM) เพื่อล็อคประตูหน้า/หลัง
―  ประตูหรือประตูท้ายถูกเปิด
―  การล็อค/ปลดล็อคถูกดำเนินการโดยใช้เบ้าเสียบกุญแจประตูหน้าด้านคนขับ
―  การล็อค/ปลดล็อคถูกดำเนินการโดยใช้ปุ่มล็อคประตูหน้าด้านคนขับ
―  การล็อค/ปลดล็อคถูกดำเนินการโดยใช้ตัวส่งสัญญาณรีโมท
―  สวิตช์สตาร์ทอยู่ที่ ACC หรือ ON (เครื่องยนต์ไม่ทำงาน)
*1  :การตั้งค่าเวลาส่วนบุคคล


ฟังก์ชั่นการล็อคอัตโนมัติเมื่อออกนอกพื้นที่ (พื้นที่รับสัญญาณ)

1.  เมื่อประตูทุกบานและประตูท้ายถูกปิด หลังจากการเปิดประตูบานใดบานหนึ่งหรือประตูท้าย ชุดเปิด/ปิดการทำงานจะเข้าสู่โหมดสแตนด์บายการล็อคอัตโนมัติหากตรวจพบสภาวะต่อไปนี้ทั้งหมด

•  ไม่มีตัวส่งสัญญาณรีโมทอยู่ภายในรถ
•  ตัวส่งสัญญาณรีโมทอยู่ภายในพื้นที่การรับสัญญาณที่ด้านนอกรถ
•  สวิตช์สตาร์ทอยู่ OFF (LOCK)

2.  หลังจากเข้าสู่สถานะสแตนด์บาย หากตัวส่งสัญญาณรีโมทถูกนำออกนอกพื้นที่การรับสัญญาณของเสาอากาศกุญแจรีโมท และไม่สามารถรับข้อมูล ID จากตัวรับสัญญาณกุญแจรีโมท ชุดเปิด/ปิดการทำงานจะสั่งการให้แอ็คชิวเอเตอร์การล็อคทั้งหมดทำการล็อคประตู

หมายเหตุ
•  สัญญาณเตือนกุญแจรีโมทดังขึ้นหนึ่งครั้งเมื่อสลับเป็นโหมดสแตนด์บายและประตูถูกล็อค


การทำงานของฟังก์ชั่นหยุดการทำงานตัวส่งสัญญาณรีโมทชั่วคราว

เมื่อดำเนินการล็อคประตู
1.  หากได้รับสัญญาณล็อคขณะตรวจพบว่าตัวส่งสัญญาณรีโมทอยู่ภายในรถ ชุดเปิด/ปิดการทำงานจะส่งสัญญาณขอการทำงานโดยใช้เสาอากาศกุญแจรีโมทด้านนอก/ด้านในรถ
2.  เมื่อตัวส่งสัญญาณรีโมทภายในรถจะส่งข้อมูล ID ไปยังตัวรับสัญญาณกุญแจรีโมท และตัวรับสัญญาณกุญแจรีโมทจะส่งข้อมูล ID ที่ได้รับไปยังชุดเปิด/ปิดการทำงาน
3.  ชุดเปิด/ปิดการทำงานได้รับข้อมูล ID และหลังจากประมาณ 32 วินาที จะหยุดการทำงานของตัวส่งสัญญาณที่อยู่ภายในรถชั่วคราว และจะสั่งการสัญญาณเตือนกุญแจรีโมทให้ดังเป็นเวลา 10 วินาที
เมื่อประตูท้ายถูกปิด
1.  เมื่อประตูท้ายปิดอยู่ขณะตรวจพบสภาวะต่อไปนี้ทั้งหมด ชุดเปิด/ปิดการทำงานจะส่งสัญญาณขอการทำงานโดยใช้เสาอากาศกุญแจรีโมทภายในรถ
―  ประตูรถทุกบานถูกล็อค
―  ประตูท้ายเปิดอยู่
―  ชุดเปิด/ปิดการทำงานระบุว่าตัวส่งสัญญาณรีโมทอยู่ด้านในรถ
2.  เมื่อตัวส่งสัญญาณรีโมทภายในรถจะส่งข้อมูล ID ไปยังตัวรับสัญญาณกุญแจรีโมท และตัวรับสัญญาณกุญแจรีโมทจะส่งข้อมูล ID ที่ได้รับไปยังชุดเปิด/ปิดการทำงาน
3.  ชุดเปิด/ปิดการทำงานได้รับข้อมูล ID และหลังจากประมาณ 32 วินาที จะหยุดการทำงานของตัวส่งสัญญาณที่อยู่ภายในรถชั่วคราว
การยกเลิกการหยุดทำงานตัวส่งสัญญาณรีโมทชั่วคราว
•  ฟังก์ชั่นของตัวส่งสัญญาณรีโมทที่ถูกหยุดการทำงานชั่วคราวจะกลับมาทำงานเมื่อตรวจพบหนึ่งในสภาวะต่อไปนี้:
―  ตัวส่งสัญญาณรีโมทที่สามารถสตาร์ทเครื่องยนต์ถูกนำไปไว้ด้านในรถ
―  ปุ่มล็อค/ปลดล็อคบนตัวส่งสัญญาณรีโมทที่ถูกหยุดการทำงานชั่วคราวถูกกด


ฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถจะทำงาน

1.  เมื่อประตูท้ายถูกปิดขณะตรวจพบสภาวะต่อไปนี้ทั้งหมด ชุดเปิด/ปิดการทำงานจะสั่งการสัญญาณเตือนกุญแจรีโมท (ดู การทำงานของฟังก์ชั่นการแจ้งเตือน) และให้ฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถอยู่ในโหมดสแตนด์บาย
―  ประตูท้ายเปิดอยู่
―  ประตูรถทุกบานถูกปิดและล็อคอยู่
2.  เมื่อสวิตช์เปิดประตูท้ายถูกกดขณะฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถอยู่ในโหมดสแตนด์บาย ชุดเปิด/ปิดการทำงานจะตรวจสอบข้อมูล ID ของตัวส่งสัญญาณรีโมท และหากมีการระบุว่าตัวส่งสัญญาณรีโมทอยู่ภายในรถ ระบบจะสั่งการฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถให้ทำงานและส่งสัญญาณปลดคันล็อคไปยังชุดควบคุมตัวถังด้านหลัง (RBCM) เพื่อปลดคันล็อคประตูท้าย
•  เมื่อตรวจพบสภาวะต่อไปนี้ขณะฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถอยู่ในโหมดสแตนด์บาย ฟังก์ชั่นป้องกันการลืมตัวส่งสัญญาณรีโมทไว้ในรถจะถูกยกเลิก
―  ปุ่มล็อค/ปลดล็อคของตัวส่งสัญญาณรีโมทถูกกด
―  สวิตช์ขอการทำงานถูกกด
―  ประตูบานใดบานหนึ่งเปิดอยู่
―  รถวิ่งด้วยความเร็วเกินกว่า 10 km/h {6.2 mph}
―  ประตูท้ายถูกเปิดและปิด
