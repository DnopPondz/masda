การถอด / การติดตั้งสายพานขับเจนเนอร์เรเตอร์

ข้อควรระวัง
•  เพื่อป้องกันความเสียหายกับตัวปรับความตึงสายพานขับอัตโนมัติ อย่าใช้แรงบิดมากเกินไปหลังจากตัวปรับความตึงโซ่เคลื่อนที่จนสุด

1.  ถอดบังโคลนรถ (ดูที่ การถอด / การติดตั้งบังโคลนรถ)

2.  วางประแจบนส่วนที่เป็นหกเหลี่ยมบนตัวปรับความตึงสายพานขับอัตโนมัติและผ่อนแรงตึงโดยค่อยๆ หมุนตามทิศทางที่แสดงในรูปภาพ

3.  ถอดสายพานขับเจนเนอร์เรเตอร์

4.  ติดตั้งสายพานขับเจนเนอร์เรเตอร์

5.  หมุนพูลเลย์เพลาข้อเหวี่ยงตามเข็มนาฬิกาประมาณครึ่งรอบ และตรวจสอบว่าสายพานขับเจนเนอร์เรเตอร์ถูกติดตั้งไว้อย่างถูกต้อง

•  หากร่องของสายพานขับและร่องของพูลเลย์แต่ละตัวไม่ตรงกัน ให้ถอดสายพานขับและติดตั้งใหม่อีกครั้ง

6.   ตรวจสอบว่าชิ้นส่วนแสดงสถานะของส่วนที่เป็นหกเหลี่ยมบนตัวปรับความตึงสายพานขับอัตโนมัติอยู่ภายในช่วงปกติ (ดูที่ การตรวจสอบสายพานขับ [SKYACTIV-G 2.0])

7.  ติดตั้งบังโคลนรถ (ดูที่ การถอด / การติดตั้งบังโคลนรถ)

การถอด

1.  ถอดสายพานขับเจนเนอร์เรเตอร์ (ดูที่ การถอด / การติดตั้งสายพานขับเจนเนอร์เรเตอร์)

2.  ใช้ผ้าที่สะอาดกับสายพานขับปั๊มน้ำตามที่แสดงในรูปภาพ

3.  ดึงผ้าในทิศทางลูกศรและหมุนพูลเลย์เพลาข้อเหวี่ยงตามนาฬิกา

ข้อควรระวัง
•  อย่าหยุดหมุนพูลเลย์เพลาข้อเหวี่ยงขณะดำเนินการขั้นตอนการถอด (ขณะสอดผ้าอยู่ระหว่างสายพานขับและพูลเลย์ปั๊มน้ำ) เพื่อป้องกันไม่ให้เกิดความเสียหายกับสายพานขับ
หมายเหตุ
•  ร่องของสายพานขับอาจไถลออกจากร่องของพูลเลย์ปั๊มน้ำโดยการหมุนพูลเลย์เพลาข้อเหวี่ยงขณะดึงผ้าตามทิศทางลูกศร

4.  ถอดสายพานขับปั๊มน้ำ


การติดตั้ง

1.  วางสายพานขับปั๊มน้ำด้านหนึ่งลงในร่องของพูลเลย์เพลาข้อเหวี่ยง

2.  วางสายพานขับปั๊มน้ำอีกด้านหนึ่งลงในร่องของพูลเลย์ปั๊มน้ำที่มีขนาดเล็กกว่า

ข้อควรระวัง
•  ระวังอย่าให้นิ้วติดอยู่ระหว่างสายพานขับและพูลเลย์เมื่อดำเนินการขั้นตอนต่อไปนี้
•  การสวมถุงมืออาจมีผลทำให้นิ้วติดอยู่ระหว่างสายพานขับและพูลเลย์ อย่าสวมถุงมือผ้าเมื่อปฏิบัติตามขั้นตอน

3.  ดันสายพานขับขึ้นจากพูลเลย์ที่มีขนาดเล็กลงบนร่องของพูลเลย์ขนาดใหญ่ และหมุนพูลเลย์เพลาข้อเหวี่ยงตามเข็มนาฬิกา

ข้อควรระวัง
•  เพื่อป้องกันไม่ให้เกิดความเสียหายกับสายพานขับ อย่าหยุดหมุนพูลเลย์เพลาข้อเหวี่ยงขณะดันสายพานขับขึ้นจากความสูงที่ต่างกันของพูลเลย์ปั๊มน้ำลงในร่องของพูลเลย์

4.  หลังจากการติดตั้งสายพานขับ ให้หมุนพูลเลย์เพลาข้อเหวี่ยงตามเข็มนาฬิกาประมาณครึ่งรอบ และตรวจสอบว่าสายพานขับปั๊มน้ำถูกติดตั้งไว้อย่างถูกต้อง

•  หากร่องของสายพานขับและร่องของพูลเลย์ไม่ตรงกัน ให้ถอดสายพานขับและติดตั้งใหม่อีกครั้ง

5.  ติดตั้งสายพานขับเจนเนอร์เรเตอร์ (ดูที่ การถอด / การติดตั้งสายพานขับเจนเนอร์เรเตอร์)
