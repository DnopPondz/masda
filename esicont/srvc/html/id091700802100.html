การถอด / การติดตั้งแผ่นปิดด้านบนเสากลางรถ

1.  ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(2)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(3)  แผ่นปิดด้านล่างเสากลางรถ (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านล่างเสากลางรถ)
(4)  ฝาครอบสลักเกี่ยวตัวปรับ (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(5)  โบลท์ติดตั้งตัวยึดด้านบนของเข็มขัดนิรภัยของที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)

2.  พลิกยางกันซึมขึ้นบางส่วน

3.  จับพื้นที่แรเงาตามที่แสดงในรูปภาพ และดึงแผ่นปิดด้านบนเสากลางรถในทิศทางลูกศร (1) ขณะปลดคลิป A, สลัก B, สลัก C

4.  ถอดแผ่นปิดด้านบนเสากลางรถในทิศทางตามลูกศร (2) ที่แสดงไว้ในรูปภาพขณะปลดตัวนำ D

ข้อควรระวัง
•  หากถอดคลิป A สลัก B และสลัก C ออกอย่างรุนแรง อาจทำให้ตัวนำ D เสียหายได้ ให้ระมัดระวังในการถอดคลิป A สลัก B และสลัก C เพื่อไม่ทำให้ตัวนำ D เสียหาย

5.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
