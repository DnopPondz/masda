ายละเอียด

•  ระบบการวิเคราะห์ปัญหาบนรถประกอบด้วย การวิเคราะห์ปัญหาบนรถ, PID/DATA MONITOR และฟังก์ชันตรวจสอบการทำงานของ A/C
•  ฟังก์ชันการวิเคราะห์ปัญหาบนรถมีฟังก์ชันต่อไปนี้:
―  ฟังก์ชั่นการตรวจจับความบกพร่อง
•  ตรวจสอบความผิดพลาดในสัญญาณอินพุทและเอาท์พุท
―  ฟังก์ชันหน่วยความจำ
•  จัดเก็บความบกพร่องที่ตรวจพบ
―  ฟังก์ชั่นการระบุความบกพร่อง
•  แสดงความบกพร่องที่ตรวจพบ
•  ฟังก์ชันแสดงความบกพร่อง, PID/DATA MONITOR และฟังก์ชันตรวจสอบการทำงานของ A/C สามารถดำเนินการโดยการเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) เข้ากับ DLC-2

การทำงาน


ฟังก์ชั่นการตรวจจับความบกพร่อง

•  ฟังก์ชันการตรวจจับความบกพร่องตรวจพบความผิดพลาดในสัญญาณอินพุทและเอาท์พุท (สวิตช์สตาร์ทอยู่ที่ ON หรือเครื่องยนต์กำลังทำงาน)
•  หากตรวจพบความบกพร่อง รหัสวิเคราะห์ปัญหาจะถูกส่งไปยัง DLC-2 ผ่านทางฟังก์ชันการแสดงความบกพร่อง
•  ขณะเดียวกัน ผลการตรวจพบความบกพร่องจะถูกส่งไปยังฟังก์ชัน Fail-safe และหน่วยความจำ


ฟังก์ชั่นหน่วยความจำ

•  ฟังก์ชันหน่วยความจำจะจัดเก็บระบบที่มีสัญญาณบกพร่องซึ่งจะถูกตรวจพบว่ามีความบกพร่องโดยฟังก์ชันการตรวจจับความบกพร่อง
•  ข้อมูลความบกพร่องที่เก็บจะไม่ถูกลบออกแม้เมื่อสวิตช์สตาร์ทอยู่ที่ OFF (LOCK) หรือมีการแก้ไขความบกพร่องแล้วก็ตาม
•  ข้อมูลความบกพร่องที่เก็บไว้สามารถลบออกโดยการเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) เข้ากับ DLC-2 และใช้งาน


ฟังก์ชันการแสดงความบกพร่อง

•  ฟังก์ชันการแสดงความบกพร่องจะส่งความบกพร่องในปัจจุบันหรืออดีตไปยัง DLC-2 เป็นรหัสวิเคราะห์ปัญหา
•  รหัสวิเคราะห์ปัญหาสามารถอ่านโดยการเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) เข้ากับ DLC-2 และใช้งาน
โหมดแสดงความบกพร่อง
•  ชุดควบคุมอุณหภูมิตรวจพบความบกพร่องในปัจจุบันและอดีตในวงจรระบบควบคุม (การขาด/ลัดวงจร) แสดงรหัสวิเคราะห์ปัญหาที่แสดงในตารางบน Mazda Modular Diagnostic System (M-MDS)
•  ลบความบกพร่องในอดีตหลังจากซ่อมเสร็จเรียบร้อยแล้ว เนื่องจากหากเก็บความบกพร่องในอดีตไว้ ความบกพร่องจะยังคงอยู่แม้หลังจากที่มีการแก้ไขความบกพร่องเรียบร้อยแล้ว
•  ความบกพร่องในอดีตที่เก็บไว้สามารถลบออกโดยการเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) เข้ากับ DLC-2 และใช้งาน

•  โหมดตรวจสอบการทำงานของ A/C จะบังคับให้อุปกรณ์เอาท์พุททำงานตามที่แสดงในตารางการตรวจสอบการทำงาน ไม่ว่าจะมีการส่งสัญญาณอินพุทไปยังชุดควบคุมอุณหภูมิหรือไม่
•  ระหว่างการทำงานของโหมดตรวจสอบการทำงานของ A/C จะมีการแสดงข้อมูลบนหน้าจอและมีการติดสว่างของไฟแสดงสถานะบนสวิตช์ที่มีการควบคุมแบบอัตโนมัติ
•  ตรวจสอบชิ้นส่วนที่บกพร่องโดยการตรวจดู ฟังเสียงการทำงาน การวางมือที่ช่องลม หรือการตรวจสอบการเปลี่ยนแปลงตามที่ระบุในตารางตรวจสอบการทำงาน

หน้าจอแสดงผล Mazda Modular Diagnostic System (M-MDS)
