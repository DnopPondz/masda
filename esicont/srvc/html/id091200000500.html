1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  ใบปัดและก้านปัดน้ำฝนกระจกบังลมหน้า (ดูที่ การถอด / การติดตั้งใบปัดและก้านปัดน้ำฝนกระจกบังลมหน้า)
2.  แผงปิดช่องลมใต้กระจก (ดูที่ การถอด / การติดตั้งแผงปิดช่องลมใต้กระจก)
(3)  ฝาครอบ (ดูที่ การถอด / การติดตั้งกล้องตรวจจับด้านหน้า (FSC))
(4)  รางสายไฟกล้องตรวจจับด้านหน้า (FSC) (รุ่นที่มีระบบควบคุมไฟสูง (HBC)) (ดูที่ การถอด / การติดตั้งรางสายไฟกล้องตรวจจับด้านหน้า (FSC))
(5)  กล้องตรวจจับด้านหน้า (FSC) (รุ่นที่มีระบบควบคุมไฟสูง (HBC)) (ดูที่ การถอด / การติดตั้งกล้องตรวจจับด้านหน้า (FSC))
(6)  คลิปยึดกล้องตรวจจับด้านหน้า (FSC) (รุ่นที่มีระบบควบคุมไฟสูง (HBC)) (ดูที่ การถอด / การติดตั้งคลิปยึดกล้องตรวจจับด้านหน้า (FSC))
(7)  เซ็นเซอร์เลเซอร์ (รุ่นที่มีระบบสมาร์ทซิตี้เบรก (SCBS)) (ดูที่ การถอด / การติดตั้งเซ็นเซอร์เลเซอร์)
(8)  เซ็นเซอร์ตรวจจับหยดน้ำฝน (รุ่นที่มีระบบไฟส่องสว่าง / ที่ปัดน้ำฝนอัตโนมัติ) (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ตรวจจับหยดน้ำฝน)
(9)  กระจกปรับไฟฟ้าอัตโนมัติ (รุ่นที่มีกระจกปรับไฟฟ้าอัตโนมัติ) (ดูที่ การถอด / การติดตั้งกระจกปรับไฟฟ้าอัตโนมัติ)
(10)  กระจกมองหลัง (รุ่นที่ไม่มีกระจกปรับไฟฟ้าอัตโนมัติ) (ดูที่ การถอด / การติดตั้งกระจกมองหลัง)
(11)  แผ่นปิดเสา A (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา A)
(12)  ที่บังแดด (ดูที่ การถอด / การติดตั้งที่บังแดด)
(13)  ไฟส่องเอกสารด้านหน้า (ดูที่ การถอด / การติดตั้งไฟส่องเอกสาร)
(14)  มือจับเสา (ดูที่ การถอด / การติดตั้งมือจับเสา)

3.  ติดตั้งแผงปิดใต้หลังคาให้ห่างออกมา

คำเตือน
•  การใช้มือเปล่าจับลวดขึงอาจทำให้เกิดการบาดเจ็บ สวมถุงมือทุกครั้งที่ใช้ลวดขึง
ข้อควรระวัง
•  เนื่องจากเลนส์สำหรับตรวจจับระดับปริมาณน้ำฝน / การติดสว่างรวมอยู่กับกระจกบังลมหน้า ระบบไฟส่องสว่าง / ที่ปัดน้ำฝนอัตโนมัติอาจทำงานไม่ถูกต้องหากเลนส์ได้รับความเสียหาย เมื่อถอดกระจกบังลมหน้า ควรระมัดระวังอย่าทำให้เลนส์เสียหาย ถ้าเลนส์ได้รับความเสียหาย ให้เปลี่ยนกระจกบังลมหน้า
หมายเหตุ
•  ก่อนการถอดกระจกบังลมหน้าออกจากตัวรถ ให้ทำเครื่องหมายบอกตำแหน่งกระจกบังลมหน้าโดยการติดเทปเข้ากับกระจกบังลมหน้าและชิ้นส่วนตัวถัง

1.  หลีกเลี่ยงการใช้สลักที่ด้านในรถ ให้สอดลวดขึงที่ตัดให้มีความยาวเพียงพอ
