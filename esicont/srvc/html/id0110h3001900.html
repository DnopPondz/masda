เครื่องยนต์ Miller Cycle [SKYACTIV-G 2.0, SKYACTIV-G 2.5]

id0110h3001900


รายละเอียด

•  เครื่องยนต์ Miller Cycle เป็นเครื่องยนต์ประสิทธิภาพสูงซึ่งมีสมรรถนะในการประหยัดน้ำมันเชื้อเพลิงสูงด้วยการมีอัตราส่วนการขยายตัวสูง
•  เครื่องยนต์ Miller Cycle มีอัตราการขยายตัวสูงโดยไม่ต้องเพิ่มอัตราส่วนกำลังอัดที่มีประสิทธิภาพโดยการหน่วงจังหวะการปิดวาล์วไอดี

การทำงาน

1.  ไอดีจะถูกส่งเข้าไปในกระบอกสูบ

2.  วาล์วไอดีจะเปิดค้างหลังจากจังหวะดูดเสร็จสิ้นเพื่อบังคับให้อากาศในกระบอกสูบไหลกลับไปยังท่อไอดี

3.  ระยะชักกำลังอัดจะเริ่มต้นตั้งแต่ตำแหน่งที่วาล์วไอดีถูกปิดอยู่ (กำลังอัดเล็กน้อย)

4.  ส่วนผสมอากาศกับน้ำมันเชื้อเพลิงจะเผาไหม้และลูกสูบจะถูกดันลงไปยัง BDC (การขยายตัวจำนวนมาก)
