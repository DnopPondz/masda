การตรวจสอบค่าความต้านทาน

คำเตือน
•  เครื่องยนต์ที่ร้อนสามารถทำให้เกิดการบาดเจ็บรุนแรงจากการโดนลวกได้ ดับเครื่องยนต์และรอจนกว่าจะเย็นตัวลงก่อนถอดเซ็นเซอร์ ECT

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดแผ่นปิดช่องหัวเทียน (ดูที่ การถอด / การติดตั้งแผ่นปิดช่องหัวเทียน [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ถ่ายน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การเปลี่ยนถ่ายน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ดึงท่อไอระเหยน้ำมันเชื้อเพลิง No.1 ออกจากถังดักไอน้ำมันและวางไว้ด้านข้าง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (ECT) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

5.  ดึงท่อระบายไอน้ำมันเครื่องออกจากท่อยางอากาศและวางไว้ด้านข้าง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (ECT) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

6.  ถอดคลิปและหย่อนชุดสายไฟ (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (ECT) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

7.  ปลดขั้วต่อเซ็นเซอร์ ECT

8.  ถอดเซ็นเซอร์ ECT (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (ECT) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

9.  ตั้งเซ็นเซอร์ ECT ลงในน้ำพร้อมกับเทอร์โมมิเตอร์ และค่อยๆ ให้ความร้อนน้ำ
