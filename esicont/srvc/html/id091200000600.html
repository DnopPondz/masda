1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดส่วนบนประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนบนประตูท้าย)
(2)  แผงปิดด้านข้างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างประตูท้าย)
(3)  ช่องที่ประตูท้าย (ดูที่ การถอด / การติดตั้งช่องที่ประตูท้าย)
(4)  แผงปิดส่วนล่างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนล่างประตูท้าย)
(5)  สปอยเลอร์หลัง (ดูที่ การถอด / การติดตั้งสปอยเลอร์หลัง)
(6)  ใบปัดและก้านปัดน้ำฝนด้านหลัง (ดูที่ การถอด / การติดตั้งใบปัดและก้านปัดน้ำฝนด้านหลัง)

3.  ค่อยๆ ใช้ไขควงที่หุ้มด้วยเทปป้องกันดึงขั้วต่อสายรับ-ส่งสัญญาณ No.3 ในทิศทางตามลูกศร (2) ที่แสดงไว้ในรูปภาพขณะที่กดขั้วสายอากาศแบบฝังในกระจกในทิศทางตามลูกศร (1) ที่แสดงไว้ในรูปภาพ และปลดขั้วสายอากาศแบบฝังในกระจกออกจากขั้วต่อสายรับ-ส่งสัญญาณ No.3 (รุ่นที่มี RDS (ระบบข้อมูลวิทยุ))

5.  ถอดขั้วไส้หลอดในทิศทางตามลูกศรตามลำดับ (3), (4) ที่แสดงไว้ในรูปภาพ

•  ก่อนการถอดกระจกหลังออกจากตัวรถ ให้ทำเครื่องหมายบอกตำแหน่งกระจกหลังโดยการติดเทปเข้ากับกระจกและชิ้นส่วนตัวถัง

8.  พันปลายแต่ละข้างของลวดขึงรอบแท่งเหล็ก

หมายเหตุ
•  ใช้การตัดแบบต่อเนื่องเพื่อให้ชิ้นงานครอบคลุมความยาวทั้งหมดของลวดขึงอันเป็นการป้องกันการแตกหักระหว่างการให้ความร้อนเฉพาะที่
