แผงสวิตช์ฺ

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  กล่องคอนโซลหน้า (ดูที่ การถอด / การติดตั้งกล่องคอนโซลหน้า)
(2)  ปุ่มคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R])(ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R])
(3)  แผงคันเกียร์ (ดูที่ การถอด / การติดตั้งแผงคันเกียร์)
(4)  แผงด้านบน (ดูที่ การถอด / การติดตั้งแผงด้านบน)
(5)  สวิตช์คำสั่ง (ดูที่ การถอด / การติดตั้งสวิตช์คำสั่ง)
