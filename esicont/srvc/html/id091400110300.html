ข้อควรระวัง
•  ห้ามถอดโบลท์และสกรู ยกเว้นโบลท์ในขั้นตอนที่ 7 และ 8 ไม่เช่นนั้นอาจเกิดความเสียหายที่ภายในและไม่สามารถปลดล็อคพวงมาลัยได้ ซึ่งจะทำให้ไม่สามารถดำเนินการถอด / การติดตั้งชุดเปิด/ปิดการทำงาน
•  ถ้าไม่ทำการกำหนดค่าเมื่อเปลี่ยนชุดเปิด/ปิดการทำงานใหม่ ข้อมูลจำเพาะของรถจะไม่ถูกเก็บบันทึกไว้ในชุดเปิด/ปิดการทำงาน และระบบจะทำงานผิดปกติ
•  เมื่อทำการตั้งค่า จำเป็นต้องอ่านข้อมูลจำเพาะของรถจากชุดเปิด/ปิดการทำงานก่อนทำการเปลี่ยน ต่อ M-MDS เข้ากับรถ และทำการระบุรถก่อนทำการถอดชุดเปิด/ปิดการทำงาน ข้อมูลจำเพาะของรถจะถูกเก็บบันทึกไว้ใน M-MDS ชั่วคราว
หมายเหตุ
•  ชุดเปิด/ปิดการทำงานก่อนการเปลี่ยนจะเก็บบันทึกข้อมูลจำเพาะของรถ
•  ชุดเปิด/ปิดการทำงานใหม่จะไม่เก็บบันทึกข้อมูลจำเพาะใดๆ ของรถ
•  ถ้าไม่สามารถอ่านข้อมูลจำเพาะของรถจากชุดเปิด/ปิดการทำงานก่อนการเปลี่ยนได้ ให้ทำการกำหนดค่าโดยใช้ข้อมูลต้นฉบับ

1.  เมื่อเปลี่ยนชุดเปิด/ปิดการทำงาน ให้ดำเนินการกำหนดค่า (ดูที่ การกำหนดค่าชุดเปิด/ปิดการทำงาน (โดยใช้ฟังก์ชั่นการอ่าน/เขียนข้อมูล))

2.  บิดสวิตช์สตาร์ทไปที่ OFF

หมายเหตุ
•  หากบิดสวิตช์สตาร์ทจาก OFF ไปที่ ON (เครื่องยนต์ทำงานหรือไม่ทำงาน) ขณะที่ประตูด้านคนขับเปิด การทำงานของล็อคพวงมาลัยอาจถูกปลดชั่วคราว

3.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

4.  ถอดชิ้นส่วนต่อไปนี้:

(1)  โมดูลถุงลมนิรภัยด้านคนขับ (ดูที่ การถอด / การติดตั้งโมดูลถุงลมนิรภัยด้านคนขับ)
2)  พวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)
(3)  คล็อคสปริง (ดูที่ การถอด / การติดตั้งคล็อคสปริง)
(4)  ฝาครอบคอพวงมาลัย (ดูที่ การถอด / การติดตั้งฝาครอบคอพวงมาลัย)
(5)  สวิตช์ที่ปัดน้ำฝนและที่ฉีดล้างกระจก (ดูที่ การถอด / การติดตั้งสวิตช์ที่ปัดน้ำฝนและฉีดน้ำล้างกระจก)
(6)  สวิตช์ไฟ (ดูที่ การถอด / การติดตั้งสวิตช์ไฟ)
