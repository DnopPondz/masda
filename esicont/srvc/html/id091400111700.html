ตัวส่งสัญญาณรีโมท

จุดประสงค์:

•  เมื่อปุ่มล็อค/ปลดล็อคของตัวส่งสัญญาณรีโมทถูกกด ข้อมูล ID จะถูกส่งไปยังตัวรับสัญญาณกุญแจรีโมท

การทำงาน

•  เมื่อตัวส่งสัญญาณรีโมทจะส่งข้อมูล ID ไฟแสดงสถานะการทำงานจะติดสว่าง

โครงสร้าง

•  ตัวส่งสัญญาณรีโมทจะติดตั้งอยู่กับไฟแสดงสถานะการทำงาน
•  ตัวส่งสัญญาณรีโมทที่ติดตั้งอยู่ในอุปกรณ์การสื่อสารอิเล็กทรอนิกส์ (ชุดรับส่งสัญญาณ) จะเก็บรหัสอิเล็กทรอนิกส์เฉพาะ (หมายเลข ID กุญแจ)
•  ในกรณีที่ตัวส่งสัญญาณไม่สามารถใช้งานได้เนื่องจากไม่มีแบตเตอรี่ จะสามารถล็อค/ปลดล็อคประตูโดยใช้กุญแจสำรอง
