การวิเคราะห์ปัญหาบนรถ [ระบบ Bluetooth]

id0920001101z6


รายละเอียด

•  เพื่อให้การวิเคราะห์ความบกพร่องในระบบเครื่องเสียงทำได้โดยง่าย ได้มีการติดตั้งระบบ Bluetooth (ระบบโทรศัพท์แฮนด์ฟรี (HF/TEL)) ที่มีฟังก์ชั่นการวิเคราะห์ปัญหาบนรถและฟังก์ชั่นช่วยวิเคราะห์

การทำงาน

•  ฟังก์ชั่นการวิเคราะห์ปัญหาบนรถมีฟังก์ชั่นตรวจจับความบกพร่องซึ่งจะตรวจจับความบกพร่องทั้งหมดในชิ้นส่วนที่เกี่ยวข้องกับระบบเครื่องเสียง และฟังก์ชั่นการแสดงผลซึ่งจะบ่งชี้ความบกพร่องของระบบผ่านจอแสดงผลรหัสวิเคราะห์ปัญหา
•  วิธีการแสดงและการลบรหัสวิเคราะห์ปัญหามีสองวิธีคือ โดยการใช้ Mazda Modular Diagnostic System (M-MDS) และโดยวิธีที่ไม่ใช้ระบบดังกล่าว เนื่องจากรหัสวิเคราะห์ปัญหาจะแตกต่างกันไปโดยขึ้นอยู่กับว่าต้องใช้ระบบ M-MDS หรือไม่ ดังนั้นจึงจำเป็นต้องใช้ทั้งสองวิธี
•  ฟังก์ชั่นช่วยวิเคราะห์แสดงสภาวะการเชื่อมต่อและค่าที่กำหนดของชุด Bluetooth ในจอแสดงผลชุดเครื่องเสียง และตรวจสอบสภาวะการเชื่อมต่อและค่าที่กำหนดของชุด Bluetooth

เมื่อใช้ Mazda Modular Diagnostic System (M-MDS)
•  เมื่อดำเนินการตรวจสอบรหัสวิเคราะห์ปัญหาโดยใช้ Mazda Modular Diagnostic System (M-MDS) ระบบวิเคราะห์ความบกพร่องของระบบเครื่องเสียงและระบบ Bluetooth (ระบบโทรศัพท์แฮนด์ฟรี (HF/TEL) ) ถูกเริ่มต้นใช้งาน ดูที่คู่มือซ่อมสำหรับขั้นตอนการตรวจสอบรหัสวิเคราะห์ปัญหา

•  เมื่อระบบหรือส่วนประกอบที่เกี่ยวข้องทำงานผิดพลาดของกล่องควบคุม จะจัดเก็บรหัสวิเคราะห์ปัญหาของส่วนที่ทำงานบกพร่องไว้ในหน่วยความจำของกล่องควบคุม และสามารถเรียกดูข้อมูลที่จัดเก็บโดยใช้เครื่องมือสแกนได้เมื่อจำเป็น รหัสวิเคราะห์ปัญหาได้รับการกำหนดโดยใช้ตัวเลข 7 หลัก ซึ่งตัวเลขแต่ละหลักมีความหมายดังต่อไปนี้

ไบต์แสดงสถานะของรหัสวิเคราะห์ปัญหา

•  ไบต์แสดงสถานะคือรหัสสองหลัก (สองหลักหลังเครื่องหมายขีด (-)) หลังรหัสวิเคราะห์ปัญหา 7 หลัก

•  ไบต์แสดงสถานะคือรหัสที่แสดงรหัสที่ค้างอยู่, สถานะความบกพร่องปัจจุบัน/ในอดีต หรือสถานะไฟเตือน
•  ไบต์แสดงสถานะสามารถอ่านได้ด้วยการทดสอบ CMDTC ด้วยตนเองโดยใช้ Mazda Modular Diagnostic System (M-MDS)
•  สำหรับรายละเอียดเกี่ยวกับไบต์แสดงสถานะ ให้ดูที่คำอธิบายใน Mazda Modular Diagnostic System (M-MDS) เมื่ออ่านรหัสวิเคราะห์ปัญหา


ฟังก์ชั่นช่วยวิเคราะห์

•  แสดงสภาวะการเชื่อมต่อและค่าที่กำหนดของระบบ Bluetooth ในจอแสดงผลชุดเครื่องเสียง และตรวจสอบสภาวะการเชื่อมต่อและค่าที่กำหนดของระบบ Bluetooth ดูที่คู่มือซ่อมสำหรับขั้นตอนการเปิดใช้งานฟังก์ชั่นช่วยวิเคราะห์

ข้อมูลการเชื่อมต่อชุด HF

•  แสดงสภาวะการเชื่อมต่อในจอแสดงผลชุดเครื่องเสียงและตรวจสอบสถานะการเชื่อมต่อชุด Bluetooth


เวอร์ชั่นของซอฟต์แวร์ชุด HF

•  แสดงเวอร์ชั่นของซอฟต์แวร์ชุด Bluetooth ในชุดเครื่องเสียง และตรวจสอบค่าที่กำหนดของชุด Bluetooth
