วิธีการสั่นสะเทือน

•  ถ้าความบกพร่องเกิดขึ้นหรืออาการรุนแรงขึ้นในขณะขับขี่บนถนนขรุขระ หรือเมื่อเครื่องยนต์สั่น ให้ปฏิบัติตามขั้นตอนด้านล่าง
หมายเหตุ
•  มีเหตุผลหลายประการที่การสั่นของรถหรือเครื่องยนต์อาจทำให้เกิดไฟฟ้าขัดข้องได้ ตรวจเช็ครายการต่อไปนี้:
―  ขั้วต่อไม่แน่น
―  ชุดสายไฟไม่มีระยะหย่อน
―  สายไฟพาดข้ามขายึดหรือชิ้นส่วนที่เคลื่อนที่
―  เดินสายไฟใกล้กับชิ้นส่วนที่ร้อนมากเกินไป
•  ชุดสายไฟที่เดินสายไม่ถูกต้อง รัดไม่ถูกต้อง หรือหลุดหลวมสามารถทำให้ชุดสายไฟถูกหนีบระหว่างชิ้นส่วนได้
•  จุดต่อขั้วต่อ จุดของการสั่น และตำแหน่งที่ชุดสายไฟเดินผ่านผนังห้องเครื่องยนต์และแผงตัวถังเป็นพื้นที่หลักที่จะตรวจเช็ค

วิธีการตรวจสอบสายไฟหรือขั้วต่อสวิตช์

1.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

2.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

หมายเหตุ
•  ถ้าเครื่องยนต์สตาร์ทติดและเดินเครื่องอยู่ ให้ปฏิบัติตามขั้นตอนต่อไปนี้ในระหว่างเดินเบา

3.  เข้า PID สำหรับสวิตช์ที่ท่านกำลังตรวจสอบ

4.  เปิดสวิตช์

5.  ขยับชุดสายไฟหรือขั้วต่อแต่ละตัวในแนวตั้งและแนวนอนเบาๆ ในขณะที่ตรวจสอบ PID

•  ถ้าค่า PID ไม่คงที่ ให้ตรวจเช็คการต่อที่ไม่แน่น

3.  จัดเตรียมฟังก์ชั่นการควบคุมสถานะเอาท์พุทสำหรับแอ็คชิวเอเตอร์หรือรีเลย์ที่ท่านกำลังตรวจเช็ค

4.  ใช้นิ้วมือของท่านขยับแอ็คชิวเอเตอร์หรือรีเลย์เป็นเวลา 3 วินาที หลังจากฟังก์ชั่นการควบคุมสถานะเอาท์พุทถูกกระตุ้นการทำงาน

•  ถ้าได้ยินเสียงคลิกที่ไม่แน่นอน ให้ตรวจเช็คการต่อที่ไม่แน่นหรือแอ็คชิวเอเตอร์ที่ได้รับการติดตั้งที่ไม่ดี หรือทั้งสองอย่าง หรือรีเลย์

วิธีฉีดพ่นละอองน้ำ

ถ้าความบกพร่องเกิดขึ้นเฉพาะในระหว่างที่มีความชื้นสูง หรือสภาพที่มีฝนตก / ปกคลุมไปด้วยหิมะ ให้ปฏิบัติตามขั้นตอนต่อไปนี้:
ข้อควรระวัง
•  เปลี่ยนอุณหภูมิและความชื้นโดยทางอ้อมด้วยการฉีดพ่นน้ำไปที่ด้านหน้าของหม้อน้ำ
•  ถ้ามีน้ำรั่วเข้าในรถ การรั่วอาจทำให้โมดูลควบคุมเสียหายได้ เมื่อทดสอบรถด้วยปัญหาการรั่วของน้ำ ต้องใช้ความระมัดระวังเป็นพิเศษ

1.  ต่อ M-MDS เข้ากับ DLC-2 หากท่านกำลังตรวจสอบเซ็นเซอร์หรือสวิตช์

2.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

หมายเหตุ
•  ถ้าเครื่องยนต์สตาร์ทติดและเดินเครื่องอยู่ ให้ปฏิบัติตามขั้นตอนต่อไปนี้ที่รอบเดินเบา

3.  เข้า PID สำหรับเซ็นเซอร์หรือสวิตช์หากท่านกำลังตรวจสอบเซ็นเซอร์หรือสวิตช์ต่างๆ

4.  หากท่านกำลังตรวจสอบสวิตช์ ให้เปิดสวิตช์

5.  ฉีดพ่นน้ำไปยังตัวรถหรือขับรถผ่านเข้าไปในเครื่องล้างรถ

•  ถ้าค่า PID ไม่คงที่ หรือความบกพร่องเกิดขึ้น ให้ซ่อมหรือเปลี่ยนชิ้นส่วนใหม่ถ้าจำเป็น
