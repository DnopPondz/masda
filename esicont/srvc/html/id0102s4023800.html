รหัสวิเคราะห์ปัญหา P0238:00

สัญญาณอินพุทวงจรเซ็นเซอร์ MAP No.1 สูงกว่าที่กำหนด
สภาวะการตรวจพบ
•  PCM จะคอยตรวจสอบแรงดันไฟฟ้าอินพุทจากเซ็นเซอร์ MAP No.1 ถ้าแรงดันไฟฟ้าอินพุทที่ขั้ว 1BK ของ PCM สูงกว่า 4.20 V เป็นเวลา 10 วินาที PCM จะตัดสินว่าวงจรเซ็นเซอร์ MAP No.1 มีความบกพร่อง
สภาวะการตรวจสอบ
―  แรงดันไฟฟ้าแบตเตอรี่: 8—20 V
ข้อมูลสนับสนุนการวิเคราะห์ปัญหา
•  กรณีนี้คือ การตรวจสอบต่อเนื่อง (CCM)
•  ไฟเตือนตรวจเช็คเครื่องยนต์จะติดสว่างถ้า PCM ตรวจพบสภาวะความบกพร่องดังกล่าวด้านบนในระหว่างรอบการขับขี่แรก
•  มีข้อมูลสภาพรถยนต์ (FREEZE FRAME DATA) (โหมด 2) / ข้อมูล Snapshot
•  รหัสวิเคราะห์ปัญหาถูกเก็บไว้ในหน่วยความจำ PCM
ฟังก์ชั่น FAIL-SAFE
•  ยับยั้งการดับเครื่องยนต์โดยการใช้งานฟังก์ชั่น i-stop
•  PCM จำกัดการควบคุมระบบการรวมเครื่องยนต์-เกียร์
สาเหตุที่เป็นไปได้
•  เทอร์โบชาร์จเจอร์บกพร่อง
•  ขั้วต่อหรือขั้วของเซ็นเซอร์ MAP No.1 บกพร่อง
•  ขั้วต่อหรือขั้วของ PCM บกพร่อง
•  ชุดสายไฟระหว่างขั้ว C ของเซ็นเซอร์ MAP No.1 กับขั้ว 1BK ของ PCM ลัดวงจรกับแหล่งจ่ายไฟ
•  วงจรแหล่งจ่ายไฟเซ็นเซอร์ MAP No.1 กับวงจรสัญญาณลัดวงจรกัน
•  ชุดสายไฟระหว่างขั้วต่างๆ ต่อไปนี้ขาดวงจร:
―  ขั้ว C ของเซ็นเซอร์ MAP No.1—ขั้ว 1BK ของ PCM
―  ขั้ว B ของเซ็นเซอร์ MAP No.1—ขั้ว 1BI ของ PCM
•  เซ็นเซอร์ MAP No.1 บกพร่อง
•  PCM บกพร่อง
