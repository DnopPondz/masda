การตรวจสอบเซ็นเซอร์ ION [SKYACTIV-G 2.0]

id0140f4954700


การตรวจสอบรหัสวิเคราะห์ปัญหา

1.  ต่อ M-MDS เข้ากับ DLC-2

2.  ทำการตรวจสอบรหัสวิเคราะห์ปัญหาโดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0])

•  ถ้ารหัสวิเคราะห์ปัญหาใดๆ ที่เกี่ยวข้องกับคอล์ยจุดระเบิด/เซ็นเซอร์ ion ปรากฏขึ้น ให้ซ่อมตำแหน่งที่บกพร่องตามการค้นหาสาเหตุปัญหาของรหัสวิเคราะห์ปัญหานั้นๆ (ดูที่ ตารางรหัสวิเคราะห์ปัญหา [SKYACTIV-G 2.0])


การตรวจสอบด้วยสายตา

หมายเหตุ
•  เนื่องจากเซ็นเซอร์ ion รวมอยู่ในคอล์ยจุดระเบิด การเปลี่ยนเซ็นเซอร์ ion จะรวมถึงการเปลี่ยนคอล์ยจุดระเบิด / เซ็นเซอร์ ion

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดแผ่นปิดช่องหัวเทียน (ดูที่ การถอด / การติดตั้งแผ่นปิดช่องหัวเทียน [SKYACTIV-G 2.0])

3.  ถอดคอล์ยจุดระเบิด / เซ็นเซอร์ ion (ดูที่ การถอด / การติดตั้งคอล์ยจุดระเบิด / เซ็นเซอร์ ION [SKYACTIV-G 2.0])

4.  ตรวจสอบว่าไม่มีความเสียหายต่อคอล์ยจุดระเบิด / เซ็นเซอร์ ion และไม่มีการผุกร่อนและความเสียหายที่ขั้วต่อ

•  ถ้ามีความบกพร่อง ให้เปลี่ยนคอล์ยจุดระเบิด / เซ็นเซอร์ ion (ดูที่ การถอด / การติดตั้งคอล์ยจุดระเบิด / เซ็นเซอร์ ION [SKYACTIV-G 2.0])
