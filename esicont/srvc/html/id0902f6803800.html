หมายเหตุ
•  ซัพพลายเออร์ของชุดเครื่องเสียงอาจแตกต่างกันไป เมื่อต้องการให้ซัพพลายเออร์ (ศูนย์บริการ) ซ่อมหรือเปลี่ยนชิ้นส่วน ให้ระบุซัพพลายเออร์และกรอกแบบฟอร์มสั่งซ่อมชุดเครื่องเสียง (Audio Repair Order Form) โดยใช้ขั้นตอนต่อไปนี้

ขั้นตอนการยืนยันด้วยฉลากหรือตัวอักษรที่ระบุไว้

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดแผงตกแต่ง (ดูที่ การถอด / การติดตั้งแผงตกแต่ง)

3.  ถอดชุดเครื่องเสียง (ดูที่ การถอด / การติดตั้งชุดเครื่องเสียง)

4.  ยืนยันชื่อซัพพลายเออร์โดยดูจากฉลากที่ติดไว้บนชุดเครื่องเสียง

ชุดเครื่องเสียง (รุ่นที่มีจอ LCD แบบสี)

1.  บิดสวิตช์สตาร์ทไปที่ตำแหน่ง ACC หรือ ON (เครื่องยนต์ไม่ทำงาน)

2.  ตรวจสอบว่าเปิดใช้งานชุดเครื่องเสียงแล้ว หากปิดใช้งานเครื่องเสียงอยู่ ให้กดสวิตช์ POWER/VOLUME เพื่อเปิดใช้งาน

3.  กดสวิตช์ POWER/VOLUME และสวิตช์ SEEK (up) พร้อมกันเพื่อเริ่มต้นฟังก์ชั่นช่วยวิเคราะห์
