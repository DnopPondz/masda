หมายเหตุ
•  สวิตช์ i-stop OFF จะรวมอยู่กับสวิตช์แผงหน้าปัด
•  สำหรับการถอด / การติดตั้งสวิตช์แผงหน้าปัด ให้อ้างอิง “การถอด / การติดตั้งแผงสวิตช์” เนื่องจากสวิตช์แผงหน้าปัดติดตั้งอยู่กับแผงสวิตช์ (ดูที่ การถอด / การติดตั้งแผงสวิตช์)

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดสวิตช์แผงหน้าปัด (ดูที่ การถอด / การติดตั้งแผงสวิตช์)

3.  ตรวจสอบว่าความต้านทานระหว่างขั้ว B และ C ของสวิตช์แผงหน้าปัดเป็นไปตามที่ระบุในตารางหรือไม่

•  ถ้าสามารถตรวจสอบได้ว่าค่าความต้านทานเป็นไปตามที่ระบุในตาราง ให้ไปที่ขั้นตอนถัดไป
•  ถ้าไม่เป็นไปตามที่ระบุในตาราง ให้เปลี่ยนสวิตช์แผงหน้าปัด (ดูที่ การถอด / การติดตั้งแผงสวิตช์)

4.  จ่ายแรงดันไฟฟ้าขั้วบวกแบตเตอรี่ไปที่ขั้ว J ของสวิตช์แผงหน้าปัด และต่อขั้ว I ลงกราวด์

•  ถ้าไฟ LED ไม่ติดสว่าง ให้เปลี่ยนสวิตช์แผงหน้าปัด (ดูที่ การถอด / การติดตั้งแผงสวิตช์)
