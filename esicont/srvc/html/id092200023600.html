การตรวจสอบโมดูลควบคุมเซ็นเซอร์ช่วยจอด

id092200023600

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นกันรอยขีดข่วนด้านหน้าฝั่งผู้โดยสาร (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(2)  แผงปิดด้านข้างด้านหน้าฝั่งผู้โดยสาร (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างด้านหน้า)
(3)  ช่องเก็บของ (ดูที่ การถอด / การติดตั้งช่องเก็บของ)
(4)  แผงตกแต่ง (ดูที่ การถอด / การติดตั้งแผงตกแต่ง)
(5)  แผงด้านล่างด้านผู้โดยสาร (ดูที่ การถอด / การติดตั้งแผงด้านล่าง)

3.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

4.  ตรวจสอบว่าแรงดันไฟฟ้าของแต่ละขั้วเป็นไปตามที่ระบุไว้ในตารางแรงดันไฟฟ้าของขั้ว (อ้างอิง)

•  หากแรงดันไฟฟ้าไม่เป็นไปตามที่กำหนดในตารางแรงดันไฟฟ้าของขั้ว (ค่าอ้างอิง) ให้ตรวจสอบชิ้นส่วนในรายการที่ตรวจสอบ
―  หากระบบทำงานผิดปกติแม้ว่ารายการที่ตรวจสอบเป็นปกติก็ตาม ให้เปลี่ยนโมดูลควบคุมเซ็นเซอร์ช่วยจอดใหม่
