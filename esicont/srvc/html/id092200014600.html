การถอด / การติดตั้งโมดูลควบคุมระบบตรวจจับรถด้านหลัง

ข้อควรระวัง
•  หากไม่ทำการตั้งค่าเมื่อเปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) เป็นชุดใหม่ ข้อมูลจำเพาะของรถจะไม่ได้รับการบันทึกไว้ในโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) และระบบจะทำงานผิดปกติ
•  เมื่อทำการตั้งค่า จำเป็นต้องอ่านข้อมูลจำเพาะของรถจากโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ก่อนทำการเปลี่ยน ต่อ M-MDS เข้ากับรถ และทำการระบุรถก่อนถอดโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ข้อมูลจำเพาะของรถจะถูกเก็บบันทึกไว้ใน M-MDS ชั่วคราว
หมายเหตุ
•  โมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ก่อนการเปลี่ยนจะเก็บบันทึกข้อมูลจำเพาะของรถ
•  โมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ใหม่จะไม่เก็บบันทึกข้อมูลจำเพาะของรถใดๆ
•  หากไม่สามารถอ่านข้อมูลจำเพาะของรถจากโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ก่อนการเปลี่ยนได้ ให้ทำการตั้งค่าโดยใช้ข้อมูลต้นฉบับ

1.  เมื่อต้องเปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) ให้ดำเนินการตั้งค่าโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) (ดูที่ การตั้งค่าโมดูลควบคุมระบบตรวจจับรถด้านหลัง (โดยใช้ฟังก์ชั่นการอ่าน / เขียนข้อมูล))

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

หมายเหตุ
•  หากไม่สามารถทำการตั้งค่าได้โดยการอ่าน / เขียนข้อมูลจำเพาะของรถ ให้ทำการตั้งค่าโดยใช้ข้อมูลต้นฉบับหลังจากเปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) (ดูที่ การตั้งค่าโมดูลควบคุมระบบตรวจจับรถด้านหลัง (โดยใช้ข้อมูลต้นฉบับ)

11.  ทำตามขั้นตอนการปรับเรดาร์ระบบตรวจจับรถด้านหลัง (ดูที่ การปรับเรดาร์ระบบตรวจจับรถด้านหลัง)

ข้อควรระวัง
•  หากมีการเปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง การไม่ดำเนินการปรับเรดาร์ระบบตรวจจับรถด้านหลังจะทำให้มุมการยิงเรดาร์คาดเคลื่อนซึ่งอาจส่งผลให้ระบบไม่สามารถตรวจจับรถที่เข้ามาใกล้จากทางด้านหลังได้อย่างถูกต้อง หากมีการเปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง ให้ทำตามขั้นตอนการปรับเรดาร์ระบบตรวจจับรถด้านหลัง
