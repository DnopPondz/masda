การถอด / การติดตั้งปั๊มสุญญากาศ [SKYACTIV-D 2.2]

id0411008036s8

1.  ถอดฝาครอบเครื่องยนต์ (ดูที่ การถอด / การติดตั้งฝาครอบเครื่องยนต์ [SKYACTIV-D 2.2])

2.  ถอดแบตเตอรี่และถาดรองแบตเตอรี่ (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ใช้คีมหนีบถอดปลอกรัด และปลดท่อสุญญากาศออกจากปั๊มสุญญากาศ (ดูที่ การถอด / การติดตั้งท่อสุญญากาศ)

4.  ถอดชิ้นส่วนต่อไปนี้

(1)  ฝากรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(2)  ไส้กรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(3)  ท่ออากาศภายนอก (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(4)  เสื้อกรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(5)  ท่ออากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(6)  ท่ออากาศเข้า (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

5.  ถอดท่อยางอากาศออกของเทอร์โบชาร์จเจอร์พร้อมกับท่อทางอากาศเข้า (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

6.  ปลดขั้วต่อเซ็นเซอร์อุณหภูมิน้ำมันเชื้อเพลิง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])

7.  ปลดขั้วต่อเซ็นเซอร์ CMP (ดูที่ การตรวจสอบเซ็นเซอร์ตำแหน่งเพลาลูกเบี้ยว (CMP) [SKYACTIV-D 2.2])

8.  ถอดชุดสายไฟเซ็นเซอร์ CMP ออกจากเสื้อตัวล่าง

9.  ให้ปฏิบัติตาม “ข้อพึงระวังก่อนการซ่อมบำรุง” (ดูที่ ข้อพึงระวังก่อนการซ่อมบำรุง [SKYACTIV-D 2.2])

10.  ถอดโบลท์ติดตั้งชุดท่อน้ำ แล้วตั้งชุดท่อน้ำให้ห่างออกมา (ดูที่ การถอด / การติดตั้งปั๊มจ่ายน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])

11.  ถอดท่อน้ำมันเชื้อเพลิงไหลกลับ (ด้านปั๊มจ่ายน้ำมันเชื้อเพลิง) (ดูที่ การถอด / การติดตั้งปั๊มจ่ายน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])

12.  ถอดท่อน้ำมันเชื้อเพลิงไหลกลับ (ด้านหัวฉีดน้ำมันเชื้อเพลิง) (ดูที่ การถอด / การติดตั้งหัวฉีดน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])

13.  ถอดท่อน้ำมันเชื้อเพลิงไหลกลับ No.1 และท่อน้ำมันเชื้อเพลิงไหลกลับ No.2 (ดูที่ การถอด / การติดตั้งปั๊มจ่ายน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])

14.  ถอดโบลท์ติดตั้งเสื้อตัวล่าง แล้วตั้งเสื้อตัวล่างให้ห่างออกมา (ดูที่ การถอด / การติดตั้งเสื้อตัวล่าง [SKYACTIV-D 2.2])

15.  ถอดตามลำดับที่แสดงไว้ในตาราง

16.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

17.  ให้ปฏิบัติตาม “ข้อพึงระวังหลังการซ่อมบำรุง” (ดูที่ ข้อพึงระวังหลังการซ่อมบำรุง [SKYACTIV-D 2.2])

หมายเหตุการถอดปั๊มสุญญากาศ

1.  เมื่อถอดปั๊มสุญญากาศ ให้วางผ้าภายใต้พื้นผิวการติดตั้งปั๊มสุญญากาศ เพื่อมิให้น้ำมันเครื่องหกเลอะบนชิ้นส่วนอื่นๆ

2.  ถอดปั๊มสุญญากาศ


หมายเหตุการติดตั้งปั๊มสุญญากาศและโอริง

1.  ใช้มือหมุนปั๊มสุญญากาศ เพื่อถ่ายน้ำมันเครื่องที่เหลืออยู่ในปั๊มสุญญากาศออกให้หมด

2.  ล้างคราบน้ำมันบริเวณร่องติดตั้งและพื้นผิวติดตั้งโอริง

3.  ติดตั้งโอริงใหม่เข้ากับปั๊มสุญญากาศ

4.  ทาซีลแลนท์ซิลิโคนเคลือบบางๆ ดังแสดงในรูปภาพ
