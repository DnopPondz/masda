•  ชุดควบคุมอุณหภูมิก่อนการเปลี่ยนจะเก็บบันทึกข้อมูลจำเพาะของรถ
•  ชุดควบคุมอุณหภูมิใหม่จะไม่เก็บบันทึกข้อมูลจำเพาะใดๆ ของรถ
•  เมื่อบิดสวิตช์สตาร์ทไปที่ ON หลังจากเปลี่ยนชุดควบคุมอุณหภูมิ ชุดควบคุมอุณหภูมิจะอ่านข้อมูลจำเพาะของรถผ่านทางการส่งข้อมูลระบบ CAN จากแผงหน้าปัด และจัดเก็บข้อมูลจำเพาะของรถ

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  หัวคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R]
(2)  กล่องคอนโซลหน้า (ดูที่ การถอด / การติดตั้งกล่องคอนโซลหน้า)
(3)  แผงคันเกียร์ (ดูที่ การถอด / การติดตั้งแผงคันเกียร์)
(4)  แผงด้านบน (ดูที่ การถอด / การติดตั้งแผงด้านบน)
(5)  คอนโซลหลัง (ดูที่ การถอด / การติดตั้งคอนโซลด้านหลัง)
(6)  ผนังด้านข้าง (ดูที่ การถอด / การติดตั้งผนังด้านข้าง)
(7)  แผงตกแต่ง (ดูที่ การถอด / การติดตั้งแผงตกแต่ง)
(8)  คอนโซลหน้า (ดูที่ การถอด / การติดตั้งคอนโซลด้านหน้า)
