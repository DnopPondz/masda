การตรวจสอบด้วยสายตา

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ปลดขั้วต่อเซ็นเซอร์ MAF / เซ็นเซอร์ IAT No.1

3.  ถอดเซ็นเซอร์ MAF / เซ็นเซอร์ IAT No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ปริมาณอากาศ (MAF) / เซ็นเซอร์อุณหภูมิไอดี (IAT) NO.1 [SKYACTIV-D 2.2])

4.  ตรวจสอบดูเซ็นเซอร์ MAF เพื่อหาความบกพร่องดังต่อไปนี้:

•  ความเสียหาย การแตกร้าว ความสกปรก
•  ขั้วเซ็นเซอร์ที่ขึ้นสนิม
•  ขั้วเซ็นเซอร์ที่บิดงอ
―  ถ้ามีความบกพร่องใดๆ ให้ซ่อมหรือเปลี่ยนเซ็นเซอร์ MAF / เซ็นเซอร์ IAT No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ปริมาณอากาศ (MAF) / เซ็นเซอร์อุณหภูมิไอดี (IAT) NO.1 [SKYACTIV-D 2.2])


การตรวจสอบแรงดันไฟฟ้า

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดเซ็นเซอร์ MAF / เซ็นเซอร์ IAT No.1 โดยที่ไม่ต้องปลดขั้วต่อ (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ปริมาณอากาศ (MAF) / เซ็นเซอร์อุณหภูมิไอดี (IAT) NO.1 [SKYACTIV-D 2.2])

3.  ต่อสายขั้วลบแบตเตอรี่กลับเข้าที่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

4.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

5.  เนื่องจากอากาศค่อยๆ เข้าไปที่ส่วนตรวจจับ MAF ของเซ็นเซอร์ MAF / เซ็นเซอร์ IAT No.1 ให้วัดแรงดันไฟฟ้าที่ขั้ว C ของเซ็นเซอร์ MAF/IAT
