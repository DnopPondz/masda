การตรวจสอบความต่อเนื่อง

1.  ตรวจสอบว่าสวิตช์ CPP ได้รับการติดตั้งอย่างถูกต้องหรือไม่ (ดูที่ การถอด / การติดตั้งสวิตช์ตำแหน่งแป้นคลัตช์ [C66M-R, C66MX-R])

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

3.  ปลดขั้วต่อสวิตช์ CPP

4.  ตรวจสอบว่าความต่อเนื่องระหว่างขั้ว A และ B ของสวิตช์ CPP เมื่อเหยียบแป้นคลัตช์และเมื่อปล่อยแป้นคลัตช์เป็นไปตามตารางต่อไปนี้หรือไม่:
