การตรวจสอบเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2]

เซ็นเซอร์ MAP No.1


การตรวจสอบด้วยสายตา

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงช่องลมใต้กระจกบังลมหน้า (ดูที่ การถอด / การติดตั้งแผงช่องลมใต้กระจกบังลมหน้า)
(2)  กรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])
(3)  แบตเตอรี่และถาดรองแบตเตอรี่ (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-D 2.2])
(4)  ท่ออากาศและท่อยางกรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-D 2.2])

3.  ปลดขั้วต่อเซ็นเซอร์ MAP No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

4.  ถอดโบลท์ออกจากขายึด (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

5.  ถอดขายึด (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

6.  ถอดเซ็นเซอร์ MAP No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

7.  ตรวจสอบดูเซ็นเซอร์ MAP No.1 เพื่อหาความบกพร่องดังต่อไปนี้:

•  ความเสียหาย การแตกร้าว ความสกปรก
―  หากมีความบกพร่องใดๆ ให้ซ่อมหรือเปลี่ยนเซ็นเซอร์ MAP No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

8.  ตรวจสอบดูท่อสุญญากาศเพื่อหาความบกพร่องดังต่อไปนี้:

•  การเดินท่อไม่ถูกต้อง บิดงอ หรือการรั่วของท่ออากาศ
―  ถ้ามีความบกพร่องใดๆ ให้ซ่อมหรือเปลี่ยนชิ้นส่วนที่บกพร่องตามผลการตรวจสอบ


การตรวจสอบแรงดันไฟฟ้า

1.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

2.  วัดแรงดันไฟฟ้าที่ขั้ว C ของเซ็นเซอร์ MAP No.1

•  ถ้าแรงดันไฟฟ้าอยู่ภายในค่ากำหนด ให้ไปที่ขั้นตอนถัดไป
•  ถ้าแรงดันไฟฟ้าไม่อยู่ภายในค่ากำหนด ให้เปลี่ยนเซ็นเซอร์ MAP No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

3.  ต่อ M-MDS เข้ากับ DLC-2

4.  เข้ารายการตรวจสอบ PID/DATA TCA_CINP โดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

5.  สตาร์ทเครื่องยนต์

6.  วัดแรงดันไฟฟ้าที่ขั้ว C ของเซ็นเซอร์ MAP No.1 เมื่อ TCA_CINP PID เพิ่มขึ้น

•  ถ้าแรงดันไฟฟ้าไม่เพิ่มขึ้น ให้เปลี่ยนเซ็นเซอร์ MAP No.1 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])
