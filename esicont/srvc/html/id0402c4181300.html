ระบบการวิเคราะห์ปัญหาบนรถ [เซ็นเซอร์เลเซอร์]

id0402c4181300


รายละเอียด

•  ระบบการวิเคราะห์ปัญหาบนรถประกอบด้วยระบบตรวจจับความบกพร่อง ซึ่งคอยตรวจจับความบกพร่องที่เกิดขึ้นกับสัญญาณอินพุท/เอาท์พุทเมื่อบิดสวิตช์สตาร์ทไปที่ ON, ฟังก์ชันการตรวจสอบข้อมูล / PID ซึ่งคอยอ่านค่าสัญญาณอินพุท/เอาท์พุทที่ระบุ และฟังก์ชันโหมดคำสั่งการทำงานที่สามารถใช้ควบคุมส่วนที่ส่งสัญญาณเอาท์พุท (เช่น โซลินอยด์วาล์ว)
•  ขั้วต่อเชื่อมโยงข้อมูล 2 (DLC-2) ถูกนำมาใช้โดยการจัดกลุ่มขั้วต่อทั้งหมดที่ใช้สำหรับการวิเคราะห์ปัญหาและการตรวจจับ/ซ่อมแซมความบกพร่องรวมไว้ในตำแหน่งเดียวกัน เพื่อช่วยให้การซ่อมบำรุงง่ายยิ่งขึ้น ซึ่งการวิเคราะห์ปัญหาสามารถดำเนินการได้โดยการเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) เข้ากับ DLC-2
•  นอกจากการอ่านค่ารหัสวิเคราะห์ปัญหาแล้ว Mazda Modular Diagnostic System (M-MDS) ยังทำหน้าที่ลบรหัสวิเคราะห์ปัญหาโดยใช้หน้าจอแสดงผลของเครื่องมือทดสอบการวิเคราะห์ปัญหา และเข้าถึงฟังก์ชั่นต่างๆ ของการตรวจสอบข้อมูล / PID และโหมดคำสั่งการทำงาน ซึ่งจะช่วยเพิ่มประสิทธิภาพการวิเคราะห์ความบกพร่องและทำให้การซ่อมบำรุงง่ายยิ่งขึ้น

ฟังก์ชั่นการตรวจจับความบกพร่อง

•  ฟังก์ชั่นการตรวจจับความบกพร่องมีหน้าที่ตรวจสอบและแสดงผลความบกพร่องที่เกิดในระบบสัญญาณอินพุท/เอาท์พุทของเซ็นเซอร์เลเซอร์ เมื่อบิดสวิตช์สตาร์ทไปที่ ON
•  หากตรวจพบเซ็นเซอร์เลเซอร์บกพร่อง ไฟเตือนหลักจะติดสว่างและข้อความ “SCBS Inspection Required” จะแสดงในจอ TFT LCD สัญญาณเอาท์พุทรหัสวิเคราะห์ปัญหาสามารถส่งผ่าน CAN_H และ CAN_L ของ DLC-2 ได้ โดยใช้ฟังก์ชั่นการสื่อสารของเครื่องทดสอบภายนอก ในขณะเดียวกัน ผลการตรวจจับความบกพร่องจะถูกส่งไปยังฟังก์ชั่นหน่วยความจำและ fail-safe


ฟังก์ชั่นหน่วยความจำ

•  ฟังก์ชั่นหน่วยความจำทำหน้าที่จัดเก็บรหัสวิเคราะห์ปัญหาของความบกพร่องที่เกิดในระบบสัญญาณอินพุท/เอาท์พุท เมื่อใช้ฟังก์ชั่นนี้ ระบบจะยังไม่ลบรหัสวิเคราะห์ปัญหาที่จัดเก็บไว้ในทันที หลังจากที่บิดสวิตช์สตาร์ทไปที่ OFF ถึงแม้ว่าระบบสัญญาณความบกพร่องจะกลับสู่สถานะปกติแล้วก็ตาม
•  เนื่องจากเซ็นเซอร์เลเซอร์มีหน่วยความจำถาวรภายใน รหัสวิเคราะห์ปัญหาจะไม่ถูกลบออกไปแม้ว่าจะถอดแบตเตอรี่ออกก็ตาม ดังนั้น คุณจำเป็นต้องลบหน่วยความจำหลังจากดำเนินการซ่อมแซมเรียบร้อยแล้ว ดูที่คู่มือซ่อมสำหรับขั้นตอนการลบรหัสวิเคราะห์ปัญหา


คำอธิบายรหัส 7 หลักของรหัสวิเคราะห์ปัญหา

•  เมื่อระบบหรือส่วนประกอบที่เกี่ยวข้องทำงานผิดพลาด เซ็นเซอร์จะจัดเก็บรหัสวิเคราะห์ปัญหาของส่วนที่ทำงานบกพร่องไว้ในหน่วยความจำของเซ็นเซอร์ และสามารถเรียกดูข้อมูลที่จัดเก็บโดยใช้เครื่องมือสแกนได้เมื่อจำเป็น รหัสวิเคราะห์ปัญหาได้รับการกำหนดโดยใช้ตัวเลข 7 หลัก ซึ่งตัวเลขแต่ละหลักมีความหมายดังต่อไปนี้

การสื่อสารแบบอนุกรม
•  การสื่อสารแบบอนุกรม (การสื่อสารแบบสองทาง) ช่วยให้การรับและส่งข้อมูลจำนวนมากสามารถทำได้พร้อมกันในสายการสื่อสารเดียวกัน
•  เมื่อเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) กับ DLC-2 การรับและส่งข้อมูลการวิเคราะห์ปัญหาระหว่าง Mazda Modular Diagnostic System (M-MDS) และเซ็นเซอร์เลเซอร์จะสามารถทำได้โดยใช้ขั้ว CAN_H และ CAN_L (ภายใน DLC-2)
•  เซ็นเซอร์เลเซอร์จะรับสัญญาณคำสั่งของฟังก์ชันการตรวจจับความบกพร่องและฟังก์ชัน PID/DATA MONITOR จาก Mazda Modular Diagnostic System (M-MDS) และส่งรหัสวิเคราะห์ปัญหาและข้อมูลเกี่ยวกับสถานะการทำงานของชิ้นส่วนอินพุท/เอาท์พุทแต่ละรายการไปยัง Mazda Modular Diagnostic System (M-MDS)
