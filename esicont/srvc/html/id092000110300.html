การทำงาน

•  สำหรับการทำงานของชุดเครื่องเสียง ให้ดูที่ระบบเครื่องเสียง

การกำหนดส่วนบุคคลของผู้ใช้

1.  เมื่อกดสวิตช์บนแผงที่ชุดเครื่องเสียงหรือสวิตช์ SETUP บนสวิตช์คำสั่ง ระบบจะสลับไปยังโหมด SETUP
2.  เมื่อกดปุ่มรถบนหน้าจอโหมด SETUP ที่ปรากฏขึ้นในจอแสดงผลชุดเครื่องเสียง ชื่อระบบที่สามารถตั้งค่าได้จะปรากฏขึ้นในจอแสดงผลชุดเครื่องเสียง
3.  ชุดเครื่องเสียงจะส่งข้อมูลการเปลี่ยนแปลงคุณสมบัติการกำหนดส่วนบุคคลที่ผู้ใช้ตั้งค่าไว้สำหรับระบบควบคุมแต่ละระบบ (ชุดเปิด/ปิดการทำงาน โมดูลควบคุมตัวถังด้านหลัง (RBCM), FSC) ผ่านการสื่อสารระบบ CAN
4.  เมื่อระบบควบคุม (ชุดเปิด/ปิดการทำงาน โมดูลควบคุมตัวถังด้านหลัง (RBCM), FSC) แต่ละระบบได้รับข้อมูลการเปลี่ยนแปลงคุณสมบัติการกำหนดส่วนบุคคล การตั้งค่าคุณสมบัติการกำหนดส่วนบุคคลแต่ละรายการจะเปลี่ยนไปตามข้อมูลการเปลี่ยนแปลงดังกล่าว
หมายเหตุ
•  ดูที่คุณสมบัติการกำหนดส่วนบุคคลสำหรับรายละเอียดเกี่ยวกับวิธีการตั้งค่าคุณสมบัติการกำหนดส่วนบุคคล


Fail-safe

•  ไม่มีฟังก์ชั่นนี้
