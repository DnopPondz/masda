•  มีการใช้มอเตอร์หลังคาซันรูฟที่มี CPU ในตัว

เอียงขึ้น/เปิด/ปิด

•  เมื่อสวิตช์หลังคาซันรูฟ (ตำแหน่งเอียง) ทำงาน ซันรูฟจะเอียงขึ้น
•  เมื่อสวิตช์หลังคาซันรูฟ (ตำแหน่งเปิด) ทำงาน ซันรูฟจะเปิดออก
•  เมื่อสวิตช์หลังคาซันรูฟ (ตำแหน่งปิด) ทำงาน ซันรูฟจะเลื่อนปิดและเอียงลง

การทำงาน

•  การเปิด/ปิดแผ่นกระจกโดยใช้การเอียงและการเลื่อน
•  เมื่อบิดสวิตช์สตาร์ทไปที่ ON (เครื่องยนต์ไม่ทำงานหรือทำงาน) หลังคาซันรูฟจะทำงานโดยใช้สวิตช์หลังคาซันรูฟ
•  หากบิดสวิตช์สตาร์ทไปที่ตำแหน่ง OFF หรือ ACC ขณะที่หลังคาซัพรูฟกำลังทำงาน หลังคาซันรูฟจะหยุดทำงาน
•  การสั่งการด้วยการกดเพียงครั้งเดียวของสวิตช์เอียงขึ้นหรือเลื่อนเปิดจะทำให้ระบบทำงานโดยอัตโนมัติ
•  หากสวิตช์ใดๆ ถูกสั่งการระหว่างการทำงานอัตโนมัติ หลังคาซันรูฟจะหยุดทำงาน

การเอียงขึ้น

1.  เมื่อสวิตช์หลังคาซันรูฟถูกสั่งการสำหรับการเอียงขึ้น CPU จะได้รับสัญญาณการเอียงขึ้น (1)

2.  เมื่อ CPU ได้รับสัญญาณการเอียงขึ้น ระบบจะเปิดรีเลย์ No.2 (2)

3.  เมื่อรีเลย์ No.2 ถูกเปิด มอเตอร์หลังคาซันรูฟจะเริ่มทำการเอียงซันรูฟขึ้น (3)

4.  เมื่อผ่านไปประมาณ 1 s หลังจากมอเตอร์หลังคาซันรูฟเริ่มทำงาน การควบคุมตัวจับเวลาของ CPU จะปิดรีเลย์ซันรูฟ No.1 โดยอัตโนมัติเพื่อหยุดการเอียงขึ้น
