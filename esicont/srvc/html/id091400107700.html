ระบบยับยั้งการทำงานของเครื่องยนต์

id091400107700


รายละเอียด

•  ระบบยับยั้งการทำงานของเครื่องยนต์เป็นอุปกรณ์ป้องกันการขโมยรถซึ่งจะอนุญาตให้รีโมทที่ได้รับการลงโปรแกรมไว้กับรถก่อนหน้านี้เท่านั้นที่สามารถสตาร์ทเครื่องยนต์ได้
•  ชุดเปิด/ปิดการทำงานจะทำงานตามการป้องกันความบกพร่องของระบบยับยั้งการทำงานของเครื่องยนต์ (ดูที่ ชุดเปิด/ปิดการทำงาน)

การทำงาน

•  ชิ้นส่วนที่เกี่ยวข้องกับระบบยับยั้งการทำงานของเครื่องยนต์ที่โปรแกรมไว้ในระบบยับยั้งการทำงานของเครื่องยนต์จะมีหมายเลข ID ของกุญแจที่โปรแกรมไว้ และจะได้รับอนุญาตให้สตาร์ทเครื่องยนต์ได้เมื่อหมายเลข ID ของกุญแจตรงกัน
•  ชิ้นส่วนที่เกี่ยวข้องกับระบบยับยั้งการทำงานของเครื่องยนต์สามารถโปรแกรมโดยใช้ Mazda Modular Diagnostic System (M-MDS)
•  รถหนึ่งคันจะสามารถโปรแกรมตัวส่งสัญญาณรีโมทได้สูงสุด 6 ตัว
•  หากชิ้นส่วนต่อไปนี้ถูกเปลี่ยนจะต้องมีการโปรแกรมชิ้นส่วนในระบบยับยั้งการทำงานของเครื่องยนต์
―  ตัวส่งสัญญาณรีโมท (ชุดรับส่งสัญญาณ)

•  ขดลวดเสาอากาศจะถูกรวมอยู่กับปุ่มกดสตาร์ท
•  โดยนำส่วนด้านบนของตัวส่งสัญญาณรีโมทไปสัมผัสกับปุ่มกดสตาร์ท ขดลวดเสาอากาศจะอ่านรหัสกุญแจจากชุดรับส่งสัญญาณที่รวมอยู่กับตัวส่งสัญญาณรีโมท
