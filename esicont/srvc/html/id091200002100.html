id091200002100

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงด้านใน (ด้านคนขับ) (ดูที่ การถอด / การติดตั้งแผงด้านใน)
(2)  แผงประตูหน้า (ด้านคนขับ) (ดูที่ การถอด / การติดตั้งแผงประตูหน้า)
(3)  สวิตช์หลักของกระจกไฟฟ้า (ดูที่ การถอด / การติดตั้งสวิตช์หลักของกระจกไฟฟ้า)

3.  ต่อขั้วต่อสวิตช์หลักของกระจกไฟฟ้า

4.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / การต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

5.  วัดแรงดันไฟฟ้าแต่ละขั้ว

•  หากแรงดันไฟฟ้าไม่เป็นไปตามที่กำหนดในตารางแรงดันไฟฟ้าของขั้ว ให้ตรวจสอบชิ้นส่วนในรายการที่ตรวจสอบและชุดสายไฟที่เกี่ยวข้อง
―  หากระบบทำงานไม่ปกติแม้ว่ารายการที่ตรวจสอบหรือชุดสายไฟที่เกี่ยวข้องไม่มีข้อบกพร่องใดๆ ก็ตาม ให้เปลี่ยนสวิตช์หลักของกระจกไฟฟ้า
