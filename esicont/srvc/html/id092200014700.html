การตรวจสอบโมดูลควบคุมระบบตรวจจับรถด้านหลัง

id092200014700

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดไฟหลังรวม (ดูที่ การถอด / การติดตั้งไฟหลังรวม)

3.  ถอดกันชนหลัง (ดูที่ การถอด / การติดตั้งกันชนท้าย)

4.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

5.  ตรวจสอบว่าแรงดันไฟฟ้าของแต่ละขั้วเป็นไปตามที่ระบุไว้ในตารางแรงดันไฟฟ้าของขั้ว (อ้างอิง)

•  หากแรงดันไฟฟ้าไม่เป็นไปตามที่กำหนดในตารางแรงดันไฟฟ้าของขั้ว (ค่าอ้างอิง) ให้ตรวจสอบชิ้นส่วนในรายการที่ตรวจสอบ
―  หากระบบทำงานผิดปกติแม้ว่ารายการที่ตรวจสอบเป็นปกติก็ตาม ให้เปลี่ยนโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ขวา) หรือโมดูลควบคุมระบบตรวจจับรถด้านหลัง (ซ้าย)
