การสื่อสารระบบ CAN ผิดพลาด (HS CAN)
สภาวะการตรวจพบ
•  ความบกพร่องในสายบัสการสื่อสารระบบ CAN
ข้อมูลสนับสนุนการวิเคราะห์ปัญหา
•  กรณีนี้คือ การตรวจสอบต่อเนื่อง (อื่นๆ)
•  ไฟเตือนตรวจเช็คเครื่องยนต์ไม่ติดสว่าง
•  ไม่มีข้อมูลสภาพรถยนต์ (FREEZE FRAME DATA) (โหมด 2) / ข้อมูล Snapshot
•  รหัสวิเคราะห์ปัญหาถูกเก็บไว้ในหน่วยความจำ PCM
ฟังก์ชั่น FAIL-SAFE
—
สาเหตุที่เป็นไปได้
•  สายบัสการสื่อสารระบบ CAN บกพร่อง

การสื่อสาร CAN: การสื่อสารกับ TCM ผิดพลาด
สภาวะการตรวจพบ
•  การสื่อสารระหว่าง TCM ผิดพลาดต่อเนื่องเป็นเวลา 5 วินาที หรือมากกว่า
ข้อมูลสนับสนุนการวิเคราะห์ปัญหา
•  กรณีนี้คือ การตรวจสอบต่อเนื่อง (อื่นๆ)
•  ไฟเตือนตรวจเช็คเครื่องยนต์จะติดสว่างถ้า PCM ตรวจพบสภาวะความบกพร่องดังกล่าวด้านบนในระหว่างรอบการขับขี่แรก
•  มีข้อมูลสภาพรถยนต์ (FREEZE FRAME DATA) (โหมด 2) / ข้อมูล Snapshot
•  รหัสวิเคราะห์ปัญหาถูกเก็บไว้ในหน่วยความจำ PCM
ฟังก์ชั่น FAIL-SAFE
•  ควบคุมสัญญาณการหมุนของเทอร์ไบน์ที่ 0
•  ยับยั้งการขอทดรอบแรงบิด AT
สาเหตุที่เป็นไปได้
หมายเหตุ
•  ระหว่างการวิเคราะห์ความบกพร่อง ให้ตรวจสอบการตรวจจับความบกพร่องของรหัสวิเคราะห์ปัญหา U0074:00 หรือ U0115:00 ของ TCM ก่อน
•  สายการสื่อสารระบบ CAN ระหว่าง PCM กับ TCM บกพร่อง
•  TCM บกพร่อง
•  PCM บกพร่อง
