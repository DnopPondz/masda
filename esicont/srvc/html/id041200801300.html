1.  ถอดกล่องคอนโซลด้านหน้า (ดูที่ การถอด / การติดตั้งกล่องคอนโซลด้านหน้า)

2.  ถอดหัวคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R]

3.  ถอดแผงเปลี่ยนเกียร์ (ดูที่ การถอด / การติดตั้งแผงปิดแท่นคันเกียร์)

4.  ถอดแผงด้านบน (ดูที่ การถอด / การติดตั้งแผงปิดด้านบน)

5.  ถอดคอนโซลด้านหลัง (ดูที่ การถอด / การติดตั้งคอนโซลหลัง)

6.  คลายนัตปรับตั้ง

7.  ถอดฝาครอบใต้พื้นรถ No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้พื้นรถ)

8. ถอดตามลำดับที่ระบุไว้ในตาราง

9. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

10.  หลังจากการติดตั้ง ให้ตรวจสอบระยะดึงคันเบรกมือ (ดูที่ การตรวจสอบคันเบรกมือ)

ข้อมูลการถอดสายเบรกมือด้านหลัง

1.  ถอดสายเบรกมือด้านหลังออกจากสายเบรกมือด้านหน้า (ดูที่ การถอด / การติดตั้งสายเบรกมือด้านหน้า)

2.  ถอดสายเบรกมือด้านหลังออกจากคาลิปเปอร์ด้านหลัง

3.  ถอดสายเบรกมือด้านหลัง


หมายเหตุการติดตั้งสายเบรกมือด้านหลัง

1.  ติดตั้งสายเบรกมือด้านหลังเข้ากับคาลิปเปอร์หลัง

2.  ตรวจสอบว่าแท็บของปลายด้านนอกสายเบรกมือประกบกับขายึดแน่นหนาดีแล้ว
