1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  พลิกขอบบังโคลนหน้าขึ้นบางส่วน (ดูที่ การถอด / การติดตั้งบังโคลนด้านหน้า)

3.  วางยางกันโคลน (ด้านขวา) ไว้ด้านข้าง (ดูที่ การถอด / การติดตั้งยางกันโคลน)

4.  ถอดกันชนหน้า (ดูที่ การถอด / การติดตั้งกันชนหน้า)

5.  ปลดขั้วต่อมอเตอร์ฉีดน้ำล้างกระจกออกจากมอเตอร์ฉีดน้ำล้างกระจก

6.  ปลดท่อยางน้ำฉีดล้างกระจกบังลมหน้าออกจากมอเตอร์ฉีดน้ำล้างกระจก

7.  ปลดท่อยางน้ำฉีดล้างกระจกหลังออกจากมอเตอร์ฉีดน้ำล้างกระจก

คำเตือน
•  น้ำที่ฉีดพ่นลงบนแบตเตอรี่และสายกราวด์อาจทำให้เกิดการเสียชีวิตจากกระแสไฟฟ้า เมื่อต้องดำเนินการตรวจสอบมอเตอร์ฉีดน้ำล้างกระจก ให้เตรียมภาชนะเพื่อรองน้ำฉีดล้างกระจกและอย่าให้มีการฉีดพ่นน้ำฉีดล้างกระจก

8.  หลังจากต่อขั้ว A ของมอเตอร์ฉีดน้ำล้างกระจกเข้ากับแบตเตอรี่ขั้วบวกและต่อขั้ว B ลงกราวด์แล้ว ให้ตรวจสอบว่ามีการฉีดน้ำล้างกระจกจากด้านหน้าของมอเตอร์ฉีดน้ำล้างกระจกหรือไม่
