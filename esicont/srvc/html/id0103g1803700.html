การตรวจสอบการทำงานของระบบควบคุมเครื่องยนต์ [SKYACTIV-D 2.2]

id0103g1803700


การตรวจสอบการทำงานของชัตเตอร์วาล์วไอดี

1.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

2.  สตาร์ทเครื่องยนต์ และเดินเครื่องที่รอบเดินเบา

3.  เข้าและตรวจสอบ PID ของ ETC_DSD, TP และ MAP (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

4.  ตรวจสอบว่า MAP PID เพิ่มขึ้นและ TP PID เพิ่มขึ้นเป็นเส้นตรง ขณะที่เพิ่มองศาการเปิดชัตเตอร์วาล์วไอดีโดยใช้ฟังก์ชั่น Active Command Mode ของ ETC_DSD หรือไม่

•  ถ้าค่า PID ทั้งสองตัวไม่เพิ่มขึ้นตามองศาการเปิดชัตเตอร์วาล์วไอดี ให้ตรวจสอบโดยใช้ขั้นตอนต่อไปนี้
1.  ถอดชัตเตอร์วาล์วไอดีโดยที่ยังต่อขั้วต่ออยู่ (ดูที่ การถอด / การติดตั้งวาล์วเปิด-ปิดไอดี [SKYACTIV-D 2.2])
2.  ตรวจสอบว่าชัตเตอร์วาล์วไอดีเปิดและปิดขณะที่ทำงานโดยใช้ฟังก์ชั่น Active Command Mode ของ ETC_DSD หรือไม่
―  ถ้าไม่ผ่านการตรวจสอบ ให้ทดสอบอีกครั้งหลังจากทำความสะอาดชัตเตอร์วาล์วไอดีและตรวจสอบชุดสายไฟและขั้วต่อที่เกี่ยวข้องแล้ว
3.  ถ้าชัตเตอร์วาล์วไอดีไม่ทำงาน ให้เปลี่ยนชัตเตอร์วาล์วไอดี (ดูที่ การถอด / การติดตั้งวาล์วเปิด-ปิดไอดี [SKYACTIV-D 2.2])
•  ถ้า MAP PID เพิ่มขึ้นแต่ TP PID ไม่เพิ่มขึ้นเป็นเส้นตรงตามองศาการเปิดชัตเตอร์วาล์วไอดี ให้ตรวจสอบโดยใช้ขั้นตอนต่อไปนี้
1.  ตรวจสอบชุดสายไฟและขั้วต่อที่เกี่ยวข้องกับเซ็นเซอร์ตำแหน่งชัตเตอร์วาล์วไอดี (ดูที่ การตรวจสอบเซ็นเซอร์ตำแหน่งวาล์วเปิด-ปิดไอดี [SKYACTIV-D 2.2])
2.  ถอดชัตเตอร์วาล์วไอดีโดยที่ยังต่อขั้วต่ออยู่ (ดูที่ การถอด / การติดตั้งวาล์วเปิด-ปิดไอดี [SKYACTIV-D 2.2])
3.  ตรวจสอบว่ามีค่า TP PID เพิ่มขึ้นและลดลงเป็นเส้นตรงขณะที่ชัตเตอร์วาล์วไอดีได้รับการสั่งงานโดยใช้ฟังก์ชั่น Active Command Mode ของ ETC_DSD หรือไม่
―  ถ้า TP PID ไม่เพิ่มขึ้นและลดลงเป็นเส้นตรง ให้เปลี่ยนชัตเตอร์วาล์วไอดี (ดูที่ การถอด / การติดตั้งวาล์วเปิด-ปิดไอดี [SKYACTIV-D 2.2])
•  ถ้า TP PID เพิ่มขึ้นเป็นเส้นตรงแต่ MAP PID ไม่เพิ่มขึ้นเป็นเส้นตรงตามองศาการเปิดชัตเตอร์วาล์วไอดี ให้ตรวจสอบโดยใช้ขั้นตอนต่อไปนี้
1.  ตรวจสอบการรั่วของอากาศที่ชัตเตอร์วาล์วไอดีและฝาสูบ
2.  ถ้าเป็นปกติ ให้ตรวจสอบและเปลี่ยนเซ็นเซอร์ MAP (ดูที่ การตรวจสอบเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2]) (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])


การตรวจสอบการทำงานของระบบยับยั้งแรงขับเคลื่อน (Brake override system)

หมายเหตุ
•  ถ้าระบบยับยั้งแรงขับเคลื่อนทำงานตามปกติหลังจากทำการตรวจสอบดังต่อไปนี้ PCM จะตรวจพบรหัสวิเคราะห์ปัญหา P2299:00

1.  สตาร์ทเครื่องยนต์ และเดินเครื่องขณะเดินเบา

2.  ตรวจสอบว่าความเร็วรอบเครื่องยนต์อยู่ที่ประมาณ 1,000 rpmภายใต้สภาวะต่อไปนี้

•  เกียร์ว่าง
•  ความเร็วรอบเครื่องยนต์ที่ 875 rpm หรือมากกว่ารอบเดินเบา
•  เหยียบแป้นเบรก
―  ถ้าความเร็วรอบเครื่องยนต์อยู่ที่ประมาณ 1,000 rpm ให้ลบรหัสวิเคราะห์ปัญหาของ PCM โดยใช้ M-MDS (การทำงานของระบบปกติ) (ดูที่ ขั้นตอนหลังจากงานซ่อม [SKYACTIV-D 2.2])
―  ถ้าความเร็วรอบเครื่องยนต์ไม่อยู่ที่ประมาณ 1,000 rpm ให้ตรวจสอบชิ้นส่วนต่อไปนี้ แล้วซ่อมหรือเปลี่ยนชิ้นส่วนที่บกพร่อง:
•  เซ็นเซอร์ APP (ดูที่ การตรวจสอบเซ็นเซอร์ตำแหน่งแป้นคันเร่ง (APP) [SKYACTIV-D 2.2])
•  สวิตช์เกียร์ว่าง (MTX) (ดูที่ การตรวจสอบสวิตช์เกียร์ว่าง [SKYACTIV-D 2.2])
•  สวิตช์ CPP (MTX) (ดูที่ การตรวจสอบสวิตช์ตำแหน่งแป้นคลัตช์ (CPP) [SKYACTIV-D 2.2])
•  สวิตช์เบรก (สัญญาณ No.1) (ดูที่ การตรวจสอบสวิตช์ไฟเบรก)
•  สวิตช์เบรก (สัญญาณ No.2) (ดูที่ การตรวจสอบสวิตช์ไฟเบรก)


การตรวจสอบการทำงานของวาล์ว EGR

1.  ตรวจสอบว่าเซ็นเซอร์ MAP เป็นปกติ (ดูที่ การตรวจสอบเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

2.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

3.  สตาร์ทเครื่องยนต์

4.  เข้ารายการตรวจสอบ EGRP และ MAP ของ PID/DATA (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

5.  ตรวจสอบค่า MAP PID เมื่อรายการการจำลองสภาพปัญหา EGRP เพิ่มขึ้นจาก 0 % เป็น 100 %

•  ถ้าค่า MAP PID ไม่เปลี่ยนแปลง:
―  ไปที่ขั้นตอนถัดไป
•  ถ้าค่า MAP PID เปลี่ยนแปลง:
―  วาล์ว EGR เป็นปกติ

6.  สวิตช์สตาร์ทอยู่ที่ OFF

7.  ถอดวาล์ว EGR (ดูที่ การถอด / การติดตั้งวาล์ว EGR [SKYACTIV-D 2.2])

8.  ต่อขั้วต่อวาล์ว EGR/เซ็นเซอร์ตำแหน่งวาล์ว EGR

9.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

10.  ตรวจสอบการทำงานของวาล์ว EGR เมื่อรายการการจำลองสภาพปัญหา EGRP เพิ่มขึ้นจาก 0 % เป็น 100 %

•  ถ้าวาล์ว EGR ไม่ทำงาน:
―  ดำเนินการตรวจสอบความต้านทานวาล์ว EGR (ดูที่ การตรวจสอบวาล์ว EGR [SKYACTIV-D 2.2])
•  ถ้าวาล์ว EGR ทำงาน:
―  ตรวจสอบการอุดตันของท่อ EGR แล้วทำการซ่อมหรือเปลี่ยนชิ้นส่วนที่บกพร่องตามผลที่ได้จากการตรวจสอบ


การตรวจสอบการทำงานของวาล์วบายพาสของหม้อหล่อเย็น EGR

1.  ตรวจสอบว่าเซ็นเซอร์ MAP เป็นปกติ (ดูที่ การตรวจสอบเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) [SKYACTIV-D 2.2])

2.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

3.  สตาร์ทเครื่องยนต์

4.  เข้ารายการตรวจสอบ EGR_C_BP และ MAP ของ PID/DATA (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

5.  ตรวจสอบค่า MAP PID เมื่อรายการการจำลองสภาพปัญหา EGR_C_BP เพิ่มขึ้นจาก 0 % เป็น 100 %

•  ถ้าค่า MAP PID ไม่เปลี่ยนแปลง:
―  ไปที่ขั้นตอนถัดไป
•  ถ้าค่า MAP PID เปลี่ยนแปลง:
―  วาล์วบายพาสของหม้อหล่อเย็น EGR เป็นปกติ

6.  สวิตช์สตาร์ทอยู่ที่ OFF

7.  ถอดวาล์วบายพาสของหม้อหล่อเย็น EGR (ดูที่ การถอด / การติดตั้งวาล์วบายพาสระบบคูลเลอร์ EGR [SKYACTIV-D 2.2])

8.  ต่อขั้วต่อวาล์วบายพาสของหม้อหล่อเย็น EGR/เซ็นเซอร์ตำแหน่งวาล์วบายพาสของหม้อหล่อเย็น EGR

9.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

10.  ตรวจสอบการทำงานของวาล์วบายพาสของหม้อหล่อเย็น EGR เมื่อรายการการจำลองสภาพปัญหา EGR_C_BP เพิ่มขึ้นจาก 0 % เป็น 100 %

•  ถ้าวาล์วบายพาสของหม้อหล่อเย็น EGR ไม่ทำงาน:
―  ดำเนินการตรวจสอบความต้านทานวาล์วบายพาสของหม้อหล่อเย็น EGR (ดูที่ การตรวจสอบวาล์วบายพาสระบบคูลเลอร์ EGR [SKYACTIV-D 2.2])
•  ถ้าวาล์วบายพาสของหม้อหล่อเย็น EGR ทำงาน:
―  ตรวจสอบการอุดตันของท่อ EGR แล้วทำการซ่อมหรือเปลี่ยนชิ้นส่วนที่บกพร่องตามผลที่ได้จากการตรวจสอบ


การตรวจสอบการทำงานของหัวฉีดน้ำมันเชื้อเพลิง

1.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

2.  สตาร์ทเครื่องยนต์ และอุ่นเครื่องจนถึงอุณหภูมิการทำงานปกติ

3.  ทำการทดสอบตัวเองแบบ KOEO (ดูที่ การทดสอบตัวเองแบบ KOEO/KOER [SKYACTIV-D 2.2])

4.  ตรวจสอบว่าไม่มีรหัสวิเคราะห์ปัญหา P0201:00, P0202:00, P0203:00 และ/หรือ P0204:00 แสดงขึ้นโดยใช้การทดสอบตัวเองแบบ KOEO

•  ถ้ารหัสวิเคราะห์ปัญหา P0201:00, P0202:00, P0203:00 และ/หรือ P0204:00 แสดงขึ้น ให้ดำเนินการตามขั้นตอนการค้นหาสาเหตุปัญหาของรหัสวิเคราะห์ปัญหาดังกล่าว (ดูที่ ตารางรหัสวิเคราะห์ปัญหา [SKYACTIV-D 2.2])

5.  ตรวจสอบว่าความเร็วรอบเครื่องยนต์ลดลงหรือหยุดนิ่งเมื่อหัวฉีดน้ำมันเชื้อเพลิงของแต่ละสูบเปลี่ยนจากตำแหน่ง ON เป็น OFF โดยใช้ฟังก์ชั่น M-MDS Active Command Mode (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

•  ถ้าไม่ผ่านการตรวจสอบ ให้ตรวจสอบรายการต่อไปนี้ในสูบที่เกิดปัญหาดังกล่าว
―  หัวฉีดน้ำมันเชื้อเพลิง (ดูที่ การตรวจสอบหัวฉีดน้ำมันเชื้อเพลิง [SKYACTIV-D 2.2])
―  ชุดสายไฟสำหรับ PCM—หัวฉีดน้ำมันเชื้อเพลิง


การตรวจสอบระบบควบคุมตัดการทำงานของระบบปรับอากาศ

1.  สตาร์ทเครื่องยนต์

2.  เปิดสวิตช์ A/C และสวิตช์พัดลม

3.  ตรวจสอบว่าคลัตช์แม่เหล็กคอมเพรสเซอร์เครื่องปรับอากาศทำงานหรือไม่

•  ถ้าไม่มีการสั่งงาน ให้ไปที่การค้นหาสาเหตุปัญหา “A/C ทำงานไม่เพียงพอ NO.28”
(ดูที่ A/C ทำงานไม่เพียงพอ NO.28 [SKYACTIV-D 2.2])

4.  เปิดลิ้นเร่งสุด และตรวจสอบว่าคลัตช์แม่เหล็กคอมเพรสเซอร์เครื่องปรับอากาศไม่ทำงานเป็นเวลา 2—5 วินาที

•  ถ้ามีการสั่งงาน ให้ตรวจสอบรายการต่อไปนี้:
―  รีเลย์ A/C (ดูที่ การตรวจสอบรีเลย์)
―  ชุดสายไฟและขั้วต่อมีการขาดวงจรหรือลัดวงจรลงกราวด์ (ขั้ว 1C ของโมดูลควบคุมตัวถังด้านหน้า (FBCM) —รีเลย์ A/C—ขั้ว 2BE ของ PCM)
―  ชิ้นส่วนที่เกี่ยวข้องกับระบบปรับอากาศ
―  APP1, APP2 PID (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2]) (ดูที่ การตรวจสอบ PCM [SKYACTIV-D 2.2])


การตรวจสอบการทำงานของโซลินอยด์วาล์วน้ำมันเครื่อง

1.  ตรวจสอบว่าแรงดันน้ำมันเครื่องเป็นปกติ (ดูที่ การตรวจสอบแรงดันน้ำมัน [SKYACTIV-D 2.2])

2.  เชื่อมต่อ M-MDS เข้ากับ DLC-2

3. สตาร์ทเครื่องยนต์

4.  เข้ารายการตรวจสอบ OIL_P_DUTY, RPM และ ECT ของ PID/DATA (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

5.  เมื่อตรวจพบสภาวะต่อไปนี้ ให้ทำการตรวจสอบค่าเกจวัดแรงดันน้ำมันเครื่อง:

•  อุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (PID: ECT): 98 °C {208 °F} หรือ มากกว่า
•  ความเร็วรอบเครื่องยนต์: 3,000 rpm หรือมากกว่า
•  ค่า OIL_P_DUTY PID กำลังเปลี่ยนแปลง
―  ถ้าค่าเกจวัดแรงดันน้ำมันเครื่องไม่เปลี่ยนแปลง:
•  ตรวจสอบโซลินอยด์วาล์วน้ำมันเครื่อง (ดูที่ การตรวจสอบโซลินอยด์วาล์วน้ำมันเครื่อง [SKYACTIV-D 2.2])
―  ถ้าค่าเกจวัดแรงดันน้ำมันเครื่องเปลี่ยนแปลง:
•  โซลินอยด์วาล์วน้ำมันเครื่องเป็นปกติ
