คำเตือน
•  เครื่องยนต์ที่ร้อนอาจทำให้เกิดการไหม้ที่รุนแรง ดับเครื่องยนต์และรอจนกระทั่งเครื่องยนต์เย็นก่อนการซ่อมบำรุง
•  ไอน้ำมันเชื้อเพลิงเป็นสิ่งที่อันตราย ซึ่งสามารถลุกไหม้ได้ง่าย และทำให้เกิดการบาดเจ็บที่รุนแรงและความเสียหาย อย่าให้มีประกายไฟและเปลวไฟเข้าใกล้น้ำมันเชื้อเพลิง
•  การหกล้นและการรั่วของท่อทางเชื้อเพลิงเป็นสิ่งที่อันตราย น้ำมันเชื้อเพลิงสามารถลุกไหม้และทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตและเกิดความเสียหายได้ น้ำมันเชื้อเพลิงยังสามารถทำให้เกิดการระคายเคืองผิวหนังและดวงตา เพื่อป้องกันปัญหานี้ ให้ปฏิบัติตาม “ขั้นตอนเพื่อความปลอดภัยสำหรับท่อทางน้ำมันเชื้อเพลิง” อยู่เสมอ (ดูที่ ข้อควรระวังก่อนการซ่อมบำรุง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
ข้อควรระวัง
•  หากหมุนเพลาลูกเบี้ยวขณะที่ถอนโซ่ไทม์มิ่งออกและลูกสูบอยู่ที่ตำแหน่งศูนย์ตายบน อาจทำให้วาล์วชนกับลูกสูบและเครื่องยนต์เกิดความเสียหายได้ เมื่อทำการหมุนเพลาลูกเบี้ยวขณะที่ถอดโซ่ไทม์มิ่งออก ให้หมุนเพลาลูกเบี้ยวหลังจากลดระดับลูกสูบลงจากตำแหน่งศูนย์ตายบน

1.  ปลดสายขั้วลบของแบตเตอรี่ออก (ดูที่ การเชื่อมต่อ / การปลดสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การเชื่อมต่อ / การปลดสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (ไม่มีระบบ i-stop)])

2.  ถอดเพลทปิดช่องปลั๊ก (ดูที่ การถอด / การติดตั้งเพลทปิดช่องปลั๊ก [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ถอดคอยล์จุดระเบิด / เซ็นเซอร์ ION (ดูที่ การถอด / การติดตั้งคอยล์จุดระเบิด / เซ็นเซอร์ ION [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ถอดฝาครอบด้านล่างหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบด้านล่างหน้า No.2)

5.  ถอดบังโคลนรถ (ดูที่ การถอด / การติดตั้งบังโคลนรถ)

6.  ถอดสายพานขับ (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

7.  ถ่ายน้ำมันเครื่องออก (ดูที่ การเปลี่ยนน้ำมันเครื่อง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

8.  ถ่ายน้ำหล่อเย็นเครื่องยนต์ออก (ดูที่ การเปลี่ยนน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

9.  ถอดท่อร่วมไอดี (ดูที่ การถอด/การติดตั้งระบบไอดี [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

10.  วางท่อร่วมไอเสียเลี่ยงไปทางด้านหลังของรถ (ดูที่ การถอด / การติดตั้งระบบไอเสีย [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

11.  ปลดท่ออ่อนหม้อน้ำ (ด้านบน, ด้านล่าง)

12  ถอดปั๊มสุญญากาศ (ดูที่ การถอด / การติดตั้งปั๊มสุญญากาศ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

13.  ถอดปั๊มน้ำมันเชื้อเพลิงแรงดันสูงและตัวเรือนด้านหลัง (ดูที่ การถอด / การติดตั้งปั๊มเชื้อเพลิงแรงดันสูง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

14.  ปลดท่อยางฮีทเตอร์ออกจากช่องทางน้ำออกและท่อน้ำเข้า

15.  ถอดอ่างน้ำมัน (ดูที่ การถอด / การติดตั้งอ่างน้ำมัน [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

16.  ถอดโซ่ไทม์มิ่งและตัวกันโซ่ไทม์มิ่ง (ดูที่ การถอด / การติดตั้งโซ่ไทม์มิ่ง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

17.  ถอด OCV (ดูที่ การถอด / การติดตั้งวาล์วควบคุมน้ำมันเครื่อง (OCV) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

18.  ถอดตามลำดับที่ระบุไว้ในตาราง

19.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

20.  เติมน้ำมันเครื่องตามปริมาณและชนิดที่ระบุ (ดูที่ การเปลี่ยนน้ำมันเครื่อง [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

21.  เติมน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การเปลี่ยนน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

22.  สตาร์ทเครื่องยนต์ และตรวจสอบและปรับดังต่อไปนี้:

•  การรั่วของน้ำมันเครื่อง, น้ำหล่อเย็นเครื่องยนต์
•  การส่ายและการสัมผัสกันของพูลเลย์และสายพาน
•  จังหวะการจุดระเบิด ความเร็วรอบเดินเบาและส่วนผสมของรอบเดินเบา (ดูที่ การปรับแต่งเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
•  กำลังอัด (ดูที่ การตรวจสอบกำลังอัด [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
