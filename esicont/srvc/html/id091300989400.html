แบบแยก 6:4

1.  ถอดแผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)

2.  ถอดพนักพิงที่นั่งด้านหลัง (ดูที่ การถอด / การติดตั้งพนักพิงที่นั่งด้านหลัง)

3.  ถอดพนักพิงศีรษะ

4.  ถอดโครงพนักพิงที่นั่งด้านหลัง (ดูที่ การถอด / การติดตั้งโครงพนักพิงที่นั่งด้านหลัง)

5.  ถอดแหวน C ออกตามลำดับ (1), (2), (3) และ (4) ที่แสดงไว้ในรูปภาพ แล้วถอดขอบปิดพนักพิงที่นั่งด้านหลังออกจากแผ่นรองพนักพิงที่นั่งด้านหลัง

ข้อควรระวัง
•  หากนำแหวน C ที่ได้รับความเสียหายกลับมาใช้ใหม่ อาจไม่สามารถติดตั้งขอบปิดพนักพิงที่นั่งด้านหลังและแผ่นรองพนักพิงที่นั่งด้านหลังได้อย่างถูกต้อง เมื่อต้องถอดและติดตั้งแหวน C ควรระมัดระวังอย่าทำให้แหวนเสียหาย
•  หากแหวน C ได้รับความเสียหาย ให้เปลี่ยนเป็นอันใหม่

6.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด


แบบแยก 4:2:4

1.  ถอดแผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)

2.  ถอดที่นั่งด้านหลัง (ดูที่ การถอด / การติดตั้งเบาะหลัง)

3.  ถอดพนักพิงที่นั่งด้านหลัง (ดูที่ การถอด / การติดตั้งพนักพิงที่นั่งด้านหลัง)

4.  ถอดพนักพิงศีรษะ

5.  ถอดโครงพนักพิงที่นั่งด้านหลัง (ดูที่ การถอด / การติดตั้งโครงพนักพิงที่นั่งด้านหลัง)

6.  พลิกขอบปิดพนักพิงที่นั่งด้านหลังบางส่วนขึ้นจากแผ่นรองพนักพิงที่นั่งด้านหลัง ถอดแหวน C ตามลำดับ (1), (2), (3), (4), (5), (6) ที่แสดงไว้ในรูปภาพ (ด้านซ้าย/ด้านขวา)
