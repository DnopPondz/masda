1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดสวิตช์ BSM OFF (ดูที่ การถอด/การติดตั้งสวิตช์ปิดระบบตรวจสอบจุดบอด (BSM))

3.  ตรวจสอบว่าความต้านทานระหว่างขั้ว B และ C ของสวิตช์ BSM OFF เป็นไปตามที่ระบุไว้ในตาราง

•  หากไม่สามารถตรวจสอบความต้านทานตามที่ระบุในตารางได้ ให้เปลี่ยนสวิตช์แผงหน้าปัด

4.  จ่ายแรงดันไฟฟ้าขั้วบวกแบตเตอรี่ไปที่ขั้ว J ของสวิตช์ BSM OFF และต่อขั้ว I ลงกราวด์
