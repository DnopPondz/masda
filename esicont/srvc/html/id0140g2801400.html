การตรวจสอบเซ็นเซอร์ตำแหน่งเพลาลูกเบี้ยว (CMP) [SKYACTIV-G 2.0, SKYACTIV-G 2.5]

เซ็นเซอร์ CMP ไอดี


การตรวจสอบด้วยสายตา

ข้อควรระวัง
•  เมื่อเปลี่ยนเซ็นเซอร์ CMP ให้แน่ใจว่าไม่มีสิ่งแปลกปลอมบนเซ็นเซอร์ CMP เช่น เศษโลหะ ถ้าติดตั้งเซ็นเซอร์ CMP โดยที่มีสิ่งแปลกปลอม สัญญาณเอาท์พุทเซ็นเซอร์จะผิดปกติซึ่งเป็นผลมาจากการแกว่งไปมาของเส้นแรงแม่เหล็กและทำให้ประสิทธิภาพการควบคุมเครื่องยนต์ลดลง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดแผ่นปิดช่องหัวเทียน (ดูที่ การถอด / การติดตั้งแผ่นปิดช่องหัวเทียน [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

3.  ถอดท่อร่วมไอดี (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

4.  ปลดขั้วต่อเซ็นเซอร์ CMP ไอดี

5.  ถอดเซ็นเซอร์ CMP ไอดี (ดูที่ การถอด / การติดตั้งเซ็นเซอร์ตำแหน่งเพลาลูกเบี้ยว (CMP) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

6.  ตรวจสอบว่าไม่มีเศษโลหะบนเซ็นเซอร์ CMP ไอดี

•  ถ้ามีความบกพร่อง ให้ขจัดเศษโลหะที่ติดอยู่ออก

ข้อควรระวัง

•  ถ้าชุดสายไฟหรือขั้วต่อกันน้ำได้รับความเสียหาย น้ำที่ซึมผ่านเข้าไปในขั้วต่อจะทำให้เซ็นเซอร์เกิดความบกพร่องได้ การป้องกันกรณีนี้ ระมัดระวังอย่าให้ชุดสายไฟหรือขั้วต่อกันน้ำเสียหาย
