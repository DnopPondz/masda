การตรวจสอบโมดูลควบคุมตัวถังด้านหลัง (RBCM)

id094000002200

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
(2)  แผงปิดปลายห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดปลายห้องเก็บสัมภาระ)
(3)  แผ่นกันรอยขีดข่วนด้านหลัง (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(4)  แผงปิดด้านข้างกระโปรงหลัง (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างกระโปรงหลัง)

3.  ถอดโมดูลควบคุมตัวถังด้านหลัง (RBCM) ออกจากตัวถังขณะที่ขั้วต่อต่ออยู่ (ดูที่ การถอด / การติดตั้งโมดูลควบคุมตัวถังด้านหลัง (RBCM))

4.  ต่อสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

5.  ตรวจสอบว่าแรงดันไฟฟ้าของแต่ละขั้วเป็นไปตามที่ระบุไว้ใน ตารางแรงดันไฟฟ้าของขั้ว (อ้างอิง).

•  หากแรงดันไฟฟ้าไม่เป็นไปตามที่ระบุไว้ในตาราง ให้ตรวจสอบชิ้นส่วนในรายการตรวจสอบ
―  ถ้าระบบไม่ทำงานตามปกติแม้ว่ารายการการตรวจสอบจะปกติ ให้ระบุตำแหน่งของความบกพร่องตามคำแนะนำจากระบบการวิเคราะห์ปัญหาบนรถและการค้นหาสาเหตุปัญหา และทำการซ่อม
