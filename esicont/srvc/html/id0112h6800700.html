คำเตือน
•  ห้ามเปิดฝาปิดระบบหล่อเย็นหรือคลายปลั๊กถ่ายน้ำหม้อน้ำในขณะที่เครื่องยนต์ทำงานหรือเมื่อเครื่องยนต์และหม้อน้ำมีความร้อน น้ำหล่อเย็นและไอน้ำที่มีความร้อนอาจพ่นออกมาและทำให้ได้รับบาดเจ็บรุนแรง ซึ่งอาจทำให้เครื่องยนต์และระบบหล่อเย็นเสียหายได้ด้วยเช่นกัน
•  ดับเครื่องยนต์และรอจนกว่าจะเย็นตัวลง ให้ระมัดระวังอย่างมากเมื่อถอดฝาปิด ให้พันผ้าหนาๆ รอบๆ ฝาปิด และหมุนทวนเข็มนาฬิกาไปที่ตำแหน่งหยุดขั้นแรก ก้าวถอยหลังในขณะที่แรงดันพ่นออกมา
•  เมื่อท่านแน่ใจว่าแรงดันทั้งหมดออกไปแล้ว ให้กดฝาปิดลงโดยใช้ผ้า หมุนฝาปิด แล้วถอดออก
หมายเหตุ
•  อย่าถอดแยกปั๊มน้ำเนื่องจากเป็นชุดที่มีความเที่ยงตรงแม่นยำ
•  ถอด / ติดตั้งพูลเลย์ปั๊มน้ำและปั๊มน้ำออกจากด้านล่างของรถ

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดฝาครอบใต้เครื่องด้านหน้า No.2 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.2)

3.  ถอดแผ่นบังโคลน (ด้านขวา) (ดูที่ การถอด / การติดตั้งแผ่นบังโคลน)

4.  ถ่ายน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การเปลี่ยนถ่ายน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

5.  ถอดสายพานขับปั๊มน้ำขณะที่สายพานขับเจนเนอร์เรเตอร์ยังคงติดตั้งอยู่ และวางไว้ให้ห่าง (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

6. ถอดตามลำดับที่ระบุไว้ในตาราง

7. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

8.  เติมน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การเปลี่ยนถ่ายน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

9.  ตรวจสอบการรั่วของน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การตรวจสอบการรั่วของน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
