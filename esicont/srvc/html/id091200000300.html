การถอด / การติดตั้งกระจกประตูหลัง

id091200000300

1.  ลดกระจกประตูหลังลงจนสุด

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดประตูหลัง (ดูที่ การถอด / การติดตั้งแผงปิดประตูหลัง)
(2)  กระจกหูช้างประตูหลัง (ดูที่ การถอด / การติดตั้งกระจกหูช้างประตูหลัง)
(3)  ลำโพงประตูหลัง (ดูที่ การถอด / การติดตั้งลำโพงประตูหลัง)
(4)  มอเตอร์กระจกไฟฟ้าด้านหลัง (ดูที่ การถอด / การติดตั้งมอเตอร์กระจกไฟฟ้า)

4.  กดที่กึ่งกลางของฝาครอบช่องบริการในทิศทางตามลูกศร (1) ที่แสดงในรูปภาพ

5.  ถอดฝาครอบช่องบริการโดยการจับยึดแถบเอาไว้ แล้วพลิกขึ้นบางส่วนในทิศทางตามลูกศร (2) ที่แสดงในรูปภาพ

6.  ถอดโบลท์

ข้อควรระวัง
•  หากถอดโบลท์ออกโดยที่ไม่ได้รองรับกระจกประตูหลัง กระจกประตูหลังอาจหล่นและเกิดความเสียหาย ขณะถอดโบลท์ ให้สอดมือเข้าไปในรูติดตั้งลำโพงประตูหลังเพื่อรองรับกระจกประตูหลัง

7.  ยกกระจกประตูหลังขึ้น แล้วเอียงกระจกดังกล่าวขณะทำการถอด

8.  ยกกระจกประตูหลังขึ้น แล้วเอียงในทิศทางตามลูกศร (3) ที่แสดงไว้ในรูปภาพ

9. ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
