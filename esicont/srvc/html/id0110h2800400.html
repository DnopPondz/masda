การถอด / การติดตั้งเครื่องยนต์ [SKYACTIV-G 2.0]

id0110h2800400

คำเตือน
•  เครื่องยนต์ที่ร้อนอาจทำให้เกิดการไหม้ที่รุนแรง ดับเครื่องยนต์และรอจนกระทั่งเครื่องยนต์เย็นก่อนการซ่อมบำรุง
•  ไอน้ำมันเชื้อเพลิงเป็นสิ่งที่อันตราย ซึ่งสามารถลุกไหม้ได้ง่าย และทำให้เกิดการบาดเจ็บที่รุนแรงและความเสียหาย อย่าให้มีประกายไฟและเปลวไฟเข้าใกล้น้ำมันเชื้อเพลิง
•  การหกล้นและการรั่วของท่อทางเชื้อเพลิงเป็นสิ่งที่อันตราย น้ำมันเชื้อเพลิงสามารถลุกไหม้และทำให้เกิดการบาดเจ็บรุนแรงหรือเสียชีวิตและเกิดความเสียหายได้ น้ำมันเชื้อเพลิงยังสามารถทำให้เกิดการระคายเคืองผิวหนังและดวงตา เพื่อป้องกันปัญหานี้ ให้ปฏิบัติตาม “ขั้นตอนเพื่อความปลอดภัยสำหรับท่อทางน้ำมันเชื้อเพลิง” อยู่เสมอ (ดูที่ ข้อพึงระวังก่อนการซ่อมบำรุง [SKYACTIV-G 2.0])
ข้อควรระวัง
•  ยึดพวงมาลัยโดยใช้เทปหรือสายไฟเพื่อป้องกันเพลาบังคับเลี้ยวไม่ให้หมุนหลังจากปลดเพลาบังคับเลี้ยวออก หากพวงมาลัยหมุนหลังจากปลดเพลาบังคับเลี้ยว แร็กพวงมาลัยและก้านต่อออก ชิ้นส่วนภายในของคล็อคสปริงอาจเสียหาย
•  การใช้แรงมากเกินไป (แรงเท่ากับ 100 N {10.2 kgf, 22.5 lbf} หรือมากกว่า) กับมอเตอร์ / ตัวขับไทม์มิ่งวาล์วแปรผันแบบไฟฟ้าอาจทำให้เกิดความบกพร่อง เมื่อทำการซ่อมบำรุง ให้ระวังอย่าใช้แรงมากเกินไปกับมอเตอร์ / ตัวขับไทม์มิ่งวาล์วแปรผันแบบไฟฟ้าโดยใช้เครื่องมือหรือชิ้นส่วนอื่นๆ
หมายเหตุ
•  ดำเนินการถอด / การติดตั้งส่วนประกอบเครื่องยนต์และชุดเกียร์จากด้านล่างรถ

1.  ปลดสายขั้วลบของแบตเตอรี่ออก (ดูที่ การถอด/ต่อสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0]) (ดูที่ การถอด/ต่อสายแบตเตอรี่ขั้วลบ [SKYACTIV-G 2.0 (รุ่นที่ไม่มีระบบ i-stop)])

2.  ถอดเพลทปิดช่องปลั๊ก (ดูที่ การถอด / การติดตั้งเพลทปิดช่องปลั๊ก [SKYACTIV-G 2.0])

3.  ถอดไส้กรองอากาศ ท่อยางอากาศ และท่ออากาศภายนอกออกเป็นชุดเดียวกัน (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-G 2.0])

4.  ถอดโบลท์ นัต และขั้วต่อที่แสดงในรูปภาพ และวางชุดสายไฟไว้ที่ด้านข้าง

5.  ถอดถาดแบตเตอรี่และส่วนประกอบ PCM (ดูที่ การถอด / การติดตั้งแบตเตอรี่ [SKYACTIV-G 2.0])

6.  ถอดฝาครอบด้านล่างหน้า No.1 และ No.2 (ดูที่ การถอด / การติดตั้งฝาครอบด้านล่างหน้า No.1) (ดูที่ การถอด / การติดตั้งฝาครอบด้านล่างหน้า No.2)

7.  ถอดบังโคลนรถ (ดูที่ การถอด / การติดตั้งบังโคลนรถ)

8.  ถ่ายน้ำหล่อเย็นเครื่องยนต์ออก (ดูที่ การเปลี่ยนถ่ายน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0])

9.  ถ่ายน้ำมันเกียร์ (MTX) หรือ ATF (ATX) (ดูที่ การเปลี่ยนถ่ายน้ำมันเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การเปลี่ยนถ่ายน้ำมันเกียร์อัตโนมัติ (ATF) [FW6A-EL, FW6AX-EL])

10.  ถอดล้อหน้าและยางออก (ดูที่ ขั้นตอนทั่วไป (ระบบกันสะเทือน))

11.  ปลดสายคันเกียร์ (ATX) (ดูที่ การถอด / การติดตั้งไกลไกการเปลี่ยนเกียร์อัตโนมัติ)

12.  ปลดสายควบคุม (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R])

13.  ถอดชุดปั๊มคลัตช์ตัวล่างโดยที่ยังคงมีการต่อท่ออยู่ (MTX) (ดูที่ การถอด / การติดตั้งชุดปั๊มคลัตช์ตัวล่าง [C66M-R, C66MX-R])

14.  ปลดท่อสุญญากาศเบรก (ดูที่ การถอด / การติดตั้งท่อสุญญากาศ)

15.  ปลดท่อไอระเหยน้ำมันเชื้อเพลิง (ดูที่ การถอด / การติดตั้งโซลินอยด์วาล์วควบคุมการดูดไอระเหยน้ำมันเชื้อเพลิง [SKYACTIV-G 2.0])

16.  ปลดท่อน้ำมันเชื้อเพลิง (ดูที่ การถอด / การติดตั้งขั้วต่อสวมเร็ว [SKYACTIV-G 2.0])

17.  ปลดท่อยางด้านล่างหม้อน้ำ (ดูที่ การถอด / การติดตั้งหม้อน้ำ [SKYACTIV-G 2.0])

18.  ปลดท่อยางด้านบนหม้อน้ำ (ดูที่ การถอด / การติดตั้งมอเตอร์พัดลมระบายความร้อน [SKYACTIV-G 2.0])

19.  ปลดท่อยางฮีทเตอร์ (ดูที่ การถอด / การติดตั้งชุดระบบปรับอากาศ)

20.  ถอดสายพานขับเจนเนอร์เรเตอร์ (ดูที่ การถอด / การติดตั้งสายพานขับ [SKYACTIV-G 2.0])

21.  ถอดคอมเพรสเซอร์ A/C โดยยังคงเชื่อมต่อท่อยางคูลเลอร์อยู่ และยึดไว้ด้วยลวดหรือเชือกเพื่อไม่ให้ท่อขวางทางการทำงาน (ดูที่ การถอด / การติดตั้งคอมเพรสเซอร์ระบบปรับอากาศ)

22.  ถอดเพลางกลาง (4WD) (ดูที่ การถอด / การติดตั้งเพลากลาง)

23.  ถอดนัตติดตั้ง TWC (ด้านท่อร่วมไอเสีย) และยึด TWC ด้วยลวดหรือเชือกเพื่อไม่ให้ขวางการทำงาน (ดูที่ การถอด / การติดตั้งระบบไอเสีย [SKYACTIV-G 2.0])

24.  ปลดเพลาขับ (ซ้าย) ออกจากด้านชุดเกียร์และวางเพลาขับ (ซ้าย) ออกให้พ้นทาง (ดูที่ การถอด / การติดตั้งเพลาขับหน้า)

25.  ปลดเพลาขับ (ขวา) ออกจากด้านชุดเกียร์และวางเพลาขับ (ขวา) ออกให้พ้นทาง (2WD) (ดูที่ การถอด / การติดตั้งเพลาขับด้านหน้า)

26.  ถอดเพลาขับ (ขวา) (4WD) (ดูที่ การถอด / การติดตั้งเพลาขับด้านหน้า)

28.  ถอดตามลำดับที่ระบุไว้ในตาราง

29.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด

30.  เติมน้ำมันเกียร์ (MTX) หรือ ATF (ATX) (ดูที่ การเปลี่ยนถ่ายน้ำมันเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การเปลี่ยนถ่ายน้ำมันเกียร์อัตโนมัติ (ATF) [FW6A-EL, FW6AX-EL])

31.  เติมน้ำหล่อเย็นเครื่องยนต์ (ดูที่ การเปลี่ยนถ่ายน้ำหล่อเย็นเครื่องยนต์ [SKYACTIV-G 2.0])

32.  สตาร์ทเครื่องยนต์ และตรวจสอบและปรับดังต่อไปนี้:

•  การตั้งศูนย์ล้อหน้า (ดูที่ การตั้งศูนย์ล้อหน้า)
•  การรั่วของน้ำมันเครื่อง, น้ำหล่อเย็นเครื่องยนต์, ATF (ATX), น้ำมันเกียร์ (MTX) และน้ำมันเชื้อเพลิง
•  การส่ายและการสัมผัสกันของพูลเลย์และสายพาน
•  จังหวะการจุดระเบิด ความเร็วรอบเดินเบาและส่วนผสมของรอบเดินเบา (ดูที่ การปรับแต่งเครื่องยนต์ [SKYACTIV-G 2.0])
•  การทำงานของอุปกรณ์เสริมที่ทำงานตามเครื่องยนต์
