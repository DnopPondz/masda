การตรวจสอบการเชื่อมโยงล็อคประตู

1.  ดำเนินการเตรียมกระจกประตูหน้า (ดูที่ การถอด / การติดตั้งกระจกประตูหน้า)

2.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

3.  ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นปิดด้านใน (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านใน)
(2)  แผงปิดประตูหน้า (ดูที่ การถอด / การติดตั้งแผงปิดประตูหน้า)
(3)  กระจกประตูหน้า (ดูที่ การถอด / การติดตั้งกระจกประตูหน้า)
(4)  แผงโมดูลประตูหน้า (ดูที่ การถอด / การติดตั้งแผงโมดูลประตูหน้า)
(5)  เบ้าเสียบกุญแจประตูหน้า (ดูที่ การถอด / การติดตั้งเบ้าเสียบกุญแจประตูหน้า)
(6)  แอ็คชิวเอเตอร์ล็อคและล็อคประตูหน้า (ดูที่ การถอด / การติดตั้งแอ็คชิวเอเตอร์ล็อคและล็อคประตูหน้า)

4.  เมื่อสายสวิตช์เชื่อมโยงล็อคประตูถูกดึงหรือกดเข้าไป ให้ตรวจสอบว่าความต่อเนื่องเป็นไปตามที่ระบุในตาราง

การตรวจสอบการเชื่อมโยงล็อคประตูหลัง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดประตูหลัง (ดูที่ การถอด / การติดตั้งแผงปิดประตูหลัง)
(2)  มือจับหลังด้านใน (ดูที่ การถอด / การติดตั้งมือจับด้านใน)
(3)  แอ็คชิวเอเตอร์ล็อคและล็อคประตูหลัง (ดูที่ การถอด / การติดตั้งแอ็คชิวเอเตอร์ล็อคและล็อคประตูหลัง)

3.  หากตรวจสอบสภาวะล็อคประตู ให้ดึงสายสวิตช์เชื่อมโยงล็อคประตูหรือกดเข้าไป

4.  ตรวจสอบว่าความต่อเนื่องเป็นไปตามที่ระบุไว้ในตาราง
