หมายเหตุ
•  เซ็นเซอร์อุณหภูมิน้ำมันเครื่องและเซ็นเซอร์แรงดันน้ำมันเครื่องยนต์ไม่สามารถถอดแยกได้ เมื่อเปลี่ยนเซ็นเซอร์อุณหภูมิน้ำมันเครื่องหรือเซ็นเซอร์แรงดันน้ำมันเครื่อง ให้เปลี่ยนเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง

1.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงาน)

2.  วัดแรงดันไฟฟ้าที่ขั้ว D ของเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง

•  ถ้าแรงดันไฟฟ้าอยู่ภายในค่ากำหนด ให้ไปที่ขั้นตอนถัดไป
•  ถ้าแรงดันไฟฟ้าไม่อยู่ภายในค่ากำหนด ให้เปลี่ยนเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง [SKYACTIV-D 2.2])
ข้อมูลจำเพาะ
Approx. 0.5 V

3.  ต่อ M-MDS เข้ากับ DLC-2

4.  สตาร์ทเครื่องยนต์

5.  เข้ารายการตรวจสอบ PID/DATA EOP โดยใช้ M-MDS (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-D 2.2])

6.  วัดแรงดันไฟฟ้าที่ขั้ว D ของเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่องเมื่อ EOP PID เพิ่มขึ้น

•  ถ้าแรงดันไฟฟ้าไม่เพิ่มขึ้น ให้เปลี่ยนเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิน้ำมันเครื่อง / เซ็นเซอร์แรงดันน้ำมันเครื่อง [SKYACTIV-D 2.2])
