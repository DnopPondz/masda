การตรวจสอบช่องเสียบอุปกรณ์เสริม

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  คอนโซลตัวบน (ดูที่ การถอดแยก / การประกอบคอนโซลหลัง)
(2)  ช่องเสียบอุปกรณ์เสริม/USB (ดูที่ การถอด / การติดตั้งช่องเสียบอุปกรณ์เสริม/USB)

3.  เสียบปลั๊กที่วางจำหน่ายโดยทั่วไปเข้ากับชุดอุปกรณ์อินพุทจากภายนอก

การตรวจสอบช่องเสียบ USB

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  คอนโซลตัวบน (ดูที่ การถอดแยก / การประกอบคอนโซลหลัง)
(2)  ช่องเสียบอุปกรณ์เสริม/USB (ดูที่ การถอด / การติดตั้งช่องเสียบอุปกรณ์เสริม/USB)

3.  สอดสลักเข้าไปยังตำแหน่ง UA', UB', UC' และ UD' ที่แสดงไว้ในรูปภาพจากด้านของบริเวณเสียบอุปกรณ์ USB สำหรับชุดอุปกรณ์อินพุทจากภายนอก และตรวจสอบความต่อเนื่องที่ด้านเสียบและด้านขั้วของอุปกรณ์ USB

4.  ตรวจสอบว่าความต่อเนื่องที่ด้านเสียบและด้านขั้วของอุปกรณ์ USB เป็นไปตามที่ระบุไว้ในตารางหรือไม่

หมายเหตุ
•  ขณะตรวจสอบด้านเสียบอุปกรณ์ USB ให้ใช้คลิปหนีบกระดาษหรือสลักหรือเข็มเล่มบางที่คล้ายกันสัมผัสกับส่วนดังกล่าวโดยไม่เสียบเครื่องทดสอบเข้ากับขั้วต่างๆ โดยตรง
