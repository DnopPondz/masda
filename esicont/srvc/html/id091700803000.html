การถอด / การติดตั้งแผ่นปิดพื้นรถ

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(2)  แผงปิดด้านข้างด้านหน้า (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างด้านหน้า)
(3)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(4)  แผ่นปิดเสา B ด้านล่าง (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา B ด้านล่าง)
(5)  เบาะนั่งด้านหลัง (ดูที่ การถอด / การติดตั้งเบาะนั่งด้านหลัง)
(6)  ที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งที่นั่งด้านหน้า)
(7)  ตัวยึดด้านล่างของเข็มขัดนิรภัยที่นั่งด้านหน้า (ไม่มีเข็มขัดนิรภัยระบบดึงกลับอัตโนมัติแบบรัดหน้าตัก) (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(8)  เข็มขัดนิรภัยระบบดึงกลับอัตโนมัติแบบรัดหน้าตัก (มีเข็มขัดนิรภัยระบบดึงกลับอัตโนมัติแบบรัดหน้าตัก) (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(9)  หัวคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R])
(10)  กล่องคอนโซลหน้า (ดูที่ การถอด / การติดตั้งกล่องคอนโซลหน้า)
(11)  แผงคันเกียร์ (ดูที่ การถอด / การติดตั้งแผงคันเกียร์)
(12)  แผงด้านบน (ดูที่ การถอด / การติดตั้งแผงด้านบน)
(13)  คอนโซลหลัง (ดูที่ การถอด / การติดตั้งคอนโซลด้านหลัง)
(14)  ผนังด้านข้าง (ดูที่ การถอด / การติดตั้งผนังด้านข้าง)
(15)  คอนโซลหน้า (ดูที่ การถอด / การติดตั้งคอนโซลด้านหน้า)
(16)  โมดูลควบคุม SAS (ดูที่ การถอด / การติดตั้งโมดูลควบคุม SAS)
(17) เบรกมือ (ดูที่ การถอด / การติดตั้งเบรกมือ)
18)  ฝาครอบข้อต่อ (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)
