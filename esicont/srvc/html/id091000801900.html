ข้อควรระวัง
•  เนื่องจากดำเนินการซ่อมบำรุงขณะฝากระโปรงหน้าเปิด ให้กันฝากระโปรงโดยใช้ท่อนไม้เพื่อป้องกันไม่ให้ตกลงมา

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงซีลส่วนบน (ดูที่ การถอด / การติดตั้งแผงซีลส่วนบน)
(2)  กันชนหน้า (ดูที่ การถอด / การติดตั้งกันชนหน้า)
(3)  ไฟหน้ารวม (ดูที่ การถอด / การติดตั้งไฟหน้ารวม)
(4)  คานส่วนบนกระบังลม (ดูที่ การถอด / การติดตั้งคานส่วนบนกระบังลม)
(5)  แผ่นยึด (ดูที่ การถอด / การติดตั้งแผ่นยึด)
(6)  โครงแบบดูดซับแรง (ดูที่ การถอด / การติดตั้งโครงแบบดูดซับแรง)
(7)  ฝาครอบใต้เครื่องด้านหน้า No.1 (ดูที่ การถอด / การติดตั้งฝาครอบใต้เครื่องด้านหน้า No.1)
(8)  แผ่นเสริมแรงด้านล่างกันชน (ดูที่ การถอด / การติดตั้งแผ่นเสริมแรงกันชน)

3.  รองรับแผงกระบังลมโดยใช้แม่แรง

•  เนื่องจากแผงกระบังลมถูกติดตั้งเข้ากับเหล็กค้ำกันชน ให้รองรับแผงกระบังลมโดยใช้แม่แรงก่อนที่จะถอดเหล็กค้ำกันชนเพื่อป้องกันไม่ให้มีการใช้แรงมากเกินไปกับแผงกระบังลม

(1)  เหล็กค้ำกันชนหน้า (ดูที่ การถอด / การติดตั้งเหล็กค้ำกันชนหน้า)
(2)  แผ่นซีล (ดูที่ การถอด / การติดตั้งแผ่นซีล)
(3)  ตัวล็อคฝากระโปรงหน้า (ดูที่ การถอด / การติดตั้งตัวล็อคและก้านปลดกระโปรงหน้า)
(4)  เซ็นเซอร์โซนการชน (ดูที่ การถอด / การติดตั้งเซ็นเซอร์โซนการชน)
(5)  เซ็นเซอร์อุณหภูมิภายนอก (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิภายนอก [ระบบปรับอากาศแบบอัตโนมัติ]) (ดูที่ การถอด / การติดตั้งเซ็นเซอร์อุณหภูมิภายนอก [ระบบปรับอากาศแบบธรรมดา])
(6)  แตร (ดูที่ การถอด / การติดตั้งแตร)
(7)  ฝาครอบกรองอากาศ (ดูที่ การถอด / การติดตั้งระบบไอดี [SKYACTIV-G 2.0, SKYACTIV-G 2.5])

ข้อควรระวัง
•  การถอดแผงกระบังลมโดยไม่มีการรองรับอาจทำให้แผงกระบังลมหล่นลงมาและทำให้เกิดการบาดเจ็บที่รุนแรง ปฏิบัติตามขั้นตอนเสมอ โดยต้องมีช่างผู้ช่วยอย่างน้อยหนึ่งคนเพื่อป้องกันไม่ให้แผงกระบังลมหล่นลงมา
