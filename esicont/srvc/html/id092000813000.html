การตรวจสอบสายรับ-ส่งสัญญาณ NO.3

id092000813000

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(2)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(3)  แผ่นปิดด้านล่างเสากลางรถ (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านล่างเสากลางรถ)
(4)  แผ่นปรับเข็มขัดนิรภัยของที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(5)  โบลท์ยึดขอเกี่ยวด้านบนของเข็มขัดนิรภัยของที่นั่งด้านหน้า (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(6)  แผ่นปิดด้านบนเสา B (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านบนเสา B)
(7)  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
(8)  แผงปิดปลายห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดปลายห้องเก็บสัมภาระ)
(9)  แผงปิดประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดประตูท้าย)
(10)  แผ่นปิดเสา C (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา C)
(11)  แผงปิดส่วนบนประตูท้าย (รุ่นที่มี RDS (ระบบข้อมูลวิทยุ)) (ดูที่ การถอด / การติดตั้งแผงปิดส่วนบนประตูท้าย)

3.  ถอดที่จับภายในรถของผู้โดยสารด้านหลัง (ดูที่ การถอด / การติดตั้งที่จับภายในรถ)

4.  พลิกแผงปิดใต้หลังคาขึ้นบางส่วน

หมายเหตุ
•  หากพลิกแผงปิดใต้หลังคาขึ้นมากเกินไป แผงปิดใต้หลังคาเป็นรอยพับ ระมัดระวังอย่าพลิกแผงปิดใต้หลังคาขึ้นมากเกินไป

5.  ปลดสายรับ-ส่งสัญญาณ No.2

6.  ปลดขั้วต่อเสาอากาศบนหลังคา

7.  ปลดสายอากาศแบบฝังในกระจก (รุ่นที่มี RDS (ระบบข้อมูลวิทยุ))

8.  ตรวจสอบว่าความต่อเนื่องระหว่างขั้วต่างๆ ของสายรับ-ส่งสัญญาณ No.3 เป็นไปตามที่ระบุไว้ในตารางหรือไม่
