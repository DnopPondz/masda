การตรวจสอบแอ็คชิวเอเตอร์ล็อคประตูท้าย

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดส่วนบนประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนบนประตูท้าย)
(2)  แผงปิดด้านข้างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างประตูท้าย)
(3)  แผงปิดส่วนล่างประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดส่วนล่างประตูท้าย)
(4)  แอ็คชิวเอเตอร์ล็อคและล็อคประตูท้าย (ดูที่ การถอด / การติดตั้งแอ็คชิวเอเตอร์ล็อคและล็อคประตูท้าย)

3.  กดคันล็อคของแอ็คชิวเอเตอร์ล็อคและล็อคประตูท้ายเข้าไปโดยใช้ไขควงปากแบน และปรับไปที่สภาวะล็อค
