หมายเหตุ
•  โมดูลควบคุม AFS ก่อนการเปลี่ยนจะเก็บบันทึกข้อมูลจำเพาะของรถ
•  โมดูลควบคุม AFS ใหม่จะไม่เก็บบันทึกข้อมูลจำเพาะของรถใดๆ
•  เมื่อบิดสวิตช์สตาร์ท ON หลังจากเปลี่ยนโมดูลควบคุม AFS แล้ว โมดูลควบคุม AFS จะอ่านข้อมูลจำเพาะของรถที่ส่งผ่านการสื่อสารระบบ CAN จากแผงหน้าปัดโดยใช้ฟังก์ชันการตั้งค่าอัตโนมัติและบันทึกข้อมูลดังกล่าว
•  ชุดควบคุม AFS จะบันทึกความสูงของรถขณะที่ไม่มีการใช้ที่นั่ง
•  ชุดควบคุม AFS ใหม่จะไม่บันทึกความสูงของรถขณะที่ไม่มีการใช้ที่นั่ง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2.  ถอดแผ่นกันรอยขีดข่วนด้านหน้า (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)

3.  ถอดแผ่นกันรอยขีดข่วนด้านหน้า (ด้านซ้าย) (ดูที่ การถอด / การติดตั้งแผงปิดข้างด้านหน้า)

4.  ขณะกดแถบยึดคลิปในทิศทางตามลูกศร (1) ที่แสดงไว้ในรูปภาพ ให้กดคลิปในทิศทางตามลูกศร (2) เพื่อปลดแถบยึดคลิปและตัวยึด
