การตรวจสอบแรงดันไฟฟ้า

1.  ต่อ M-MDS เข้ากับ DLC-2

2.  บิดสวิตช์สตาร์ท ON (เครื่องยนต์ไม่ทำงานหรือทำงาน)

3.  ตรวจสอบว่าแรงดันไฟฟ้าเอาท์พุทเซ็นเซอร์ MAP (PID: MAP) อยู่ภายในค่ากำหนดหรือไม่ (ดูที่ การทดสอบการวิเคราะห์ปัญหาบนรถ [SKYACTIV-G 2.0]) (ดูที่ การตรวจสอบ PCM [SKYACTIV-G 2.0])

•  ถ้าไม่เป็นไปตามที่กำหนด ให้เปลี่ยนเซ็นเซอร์ MAP / เซ็นเซอร์ IAT No.2 (ดูที่ การถอด / การติดตั้งเซ็นเซอร์แรงดันสัมบูรณ์ท่อร่วมไอดี (MAP) / เซ็นเซอร์อุณหภูมิไอดี (IAT) NO.2 [SKYACTIV-G 2.0])
