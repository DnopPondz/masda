ไฟส่องเอกสารด้านหลัง

1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  โครงรอยต่อหลังคาซันรูฟ (รถรุ่นที่มีหลังคาซันรูฟ) (ดูที่ การถอด / การติดตั้งชุดหลังคาซันรูฟ)
(2)  แผ่นปิดเสา A (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา A)
(3)  ที่บังแดด (ดูที่ การถอด / การติดตั้งที่บังแดด)
(4)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(5)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(6)  แผ่นปิดเสา B ด้านล่าง (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา B ด้านล่าง)
(7)  ปรับฝาครอบตัวยึด (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(8)  โบลท์ติดตั้งตัวยึดด้านบนของเข็มขัดนิรภัย (ดูที่ การถอด / การติดตั้งเข็มขัดนิรภัยของที่นั่งด้านหน้า)
(9)  แผ่นปิดด้านบนเสา B (ดูที่ การถอด / การติดตั้งแผ่นปิดด้านบนเสา B)
(10)  มือจับเสา (ดูที่ การถอด / การติดตั้งมือจับเสา)
(11)  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
(12)  แผงปิดปลายห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดปลายห้องเก็บสัมภาระ)
(13)  แผงปิดประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดประตูท้าย)
(14)  แผ่นปิดเสา D (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา D)
(15)  แผ่นปิดเสา C (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา C)
(16)  แผงปิดใต้หลังคา (ดูที่ การถอด / การติดตั้งแผงปิดใต้หลังคา)

ข้อควรระวัง
•  หากถอดไฟส่องเอกสารด้านหลังออกจากขายึด ไฟส่องเอกสารด้านหลังอาจหลุดหล่นและได้รับความเสียหายได้ ขณะถอดไฟส่องเอกสารด้านหลังออกจากขายึด ให้ใช้มือจับไฟส่องเอกสารด้านหลังที่ด้านเลนส์ไว้ขณะซ่อมบำรุง

6.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด
