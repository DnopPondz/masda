1.  ปลดสายขั้วลบแบตเตอรี่ (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-G 2.0, SKYACTIV-G 2.5 (รุ่นที่ไม่มีระบบ i-stop)]) (ดูที่ การปลด / ต่อสายขั้วลบแบตเตอรี่ [SKYACTIV-D 2.2])

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  แผงปิดห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดห้องเก็บสัมภาระ)
(2)  แผงปิดปลายห้องเก็บสัมภาระ (ดูที่ การถอด / การติดตั้งแผงปิดปลายห้องเก็บสัมภาระ)
(3)  แผ่นกันรอยขีดข่วนด้านหลัง (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหลัง)
(4)  แผงปิดประตูท้าย (ดูที่ การถอด / การติดตั้งแผงปิดประตูท้าย)

3.  ปลดสายรีโมทบนด้านก้านจับรีโมทออกจากสายรีโมทบนด้านที่นั่งด้านหลัง

4.  ถอดสายรีโมทบนด้านก้านจับรีโมทออกจากคลิป

5.  ดึงสายรีโมทบนด้านก้านจับรีโมทออกจากสายรัดในทิศทางตามลูกศรที่แสดงไว้ในรูปภาพ แล้วถอดก้านจับรีโมทออก

6.  ติดตั้งแบบย้อนลำดับขั้นตอนการถอด (ดูที่ หมายเหตุการติดตั้งก้านจับรีโมท)

•  ห้ามใช้คลิป B เนื่องจากมีการใช้งานคลิปดังกล่าวขณะที่ผลิตรถยนต์ สายรีโมทถูกเดินสายและติดตั้งไว้ใกล้กับคลิป B อย่างไรก็ตาม หากสายรีโมทติดตั้งไว้กับคลิป B สายรีโมทอาจสัมผัสกับตัวถังรถขณะที่ใช้งานก้านจับรีโมทหรือขณะขับรถ ซึ่งอาจทำให้เกิดเสียงผิดปกติ ดังนั้น จึงไม่ควรเดินสายรีโมทเข้ากับคลิป B

สายรีโมท

ข้อควรระวัง
หากไม่ได้เดินสายและติดตั้งสายรีโมทอย่างถูกต้อง อาจทำให้พนักพิงที่นั่งด้านหลังทำงานผิดพลาด มีเสียงผิดปกติ หรือทำให้สายรีโมทได้รับความเสียหาย ติดตั้งสายรีโมทตามขั้นตอนต่อไปนี้

1.  ติดตั้งสายรีโมทตามขั้นตอนต่อไปนี้

(1)  เดินสายรีโมท (ตรงกลาง) ข้างใต้บานพับด้านหลังตามที่แสดงไว้ในรูป
(2)  เดินสายรีโมท (ซ้าย/ขวา) ผ่านขอเกี่ยวระบบ ISO FIX จากด้านบนลงด้านล่างตามที่แสดงไว้ในรูปภาพ
(3)  วางที่ครอบสายไว้บนพรมหลังเบาะนั่งด้านหลัง

2.  หลังจากติดตั้งสายรีโมทแล้ว ให้โยกพนักพิงที่นั่งด้านหลังไปมาเพื่อตรวจสอบว่าพนักพิงที่นั่งด้านหลังล็อคเข้ากับตัวเกี่ยวสลักพนักพิงที่นั่งด้านหลังอย่างแน่นหนาแล้ว

•  หากการล็อคผิดปกติ ให้ปรับตำแหน่งที่ติดตั้งสายรีโมทในที่ครอบสาย
