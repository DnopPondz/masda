รายละเอียด

•  ฟังก์ชันการสื่อสารของเครื่องทดสอบภายนอกจะเปิดการสื่อสารข้อมูลการวิเคราะห์ปัญหา (การแสดงผลรหัสวิเคราะห์ปัญหา, การแสดงผลสัญญาณอินพุท/เอาท์พุท) ระหว่างโมดูลควบคุม (CM) EPS กับเครื่องทดสอบภายนอก

การสื่อสารแบบอนุกรม

•  การสื่อสารแบบอนุกรม (การสื่อสารแบบสองทาง) ช่วยให้การรับและส่งข้อมูลจำนวนมากสามารถทำได้พร้อมกันในสายการสื่อสารเดียวกัน
•  การเชื่อมต่อ Mazda Modular Diagnostic System (M-MDS) กับ DLC-2 จะสามารถส่งและรับข้อมูลการวิเคราะห์ระหว่าง Mazda Modular Diagnostic System (M-MDS) กับ EPS CM ผ่านทางสายระบบ CAN
•  EPS CM จะรับสัญญาณคำสั่งของฟังก์ชันการตรวจจับความบกพร่องและฟังก์ชัน PID/DATA MONITOR จาก Mazda Modular Diagnostic System (M-MDS) และส่งรหัสวิเคราะห์ปัญหาและข้อมูลเกี่ยวกับสถานะการทำงานของชิ้นส่วนอินพุท/เอาท์พุทแต่ละรายการไปยัง Mazda Modular Diagnostic System (M-MDS)
