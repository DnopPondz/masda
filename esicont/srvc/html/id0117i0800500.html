การทำงานหลังจากเปลี่ยนแบตเตอรี่ (รุ่นที่มีระบบ i-stop)

หมายเหตุ
•  ถ้าอัตราการไหลในการชาร์จไฟ / จ่ายไฟแบตเตอรี่ (จำนวนครั้งที่แบตเตอรี่สามารถทำงานได้) เกินค่ากำหนด รหัสวิเคราะห์ปัญหาจะถูกบันทึกและไฟเตือนระบบ i-stop (สีเหลืองอำพัน) จะกะพริบ ถ้าอัตราการไหลในการชาร์จไฟ / จ่ายไฟแบตเตอรี่ (จำนวนครั้งที่แบตเตอรี่สามารถทำงานได้) เกินค่ากำหนด ให้เปลี่ยนแบตเตอรี่ใหม่
•  เมื่อเปลี่ยนแบตเตอรี่ใหม่ ให้ปฏิบัติตามขั้นตอนดังต่อไปนี้: เมื่อเปลี่ยน PCM พร้อมกัน ให้ทำการตั้งค่า PCM ก่อน (ดูที่ การถอด / การติดตั้ง PCM [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การตั้งค่า PCM (โดยใช้ฟังก์ชั่นการอ่าน / เขียนข้อมูล) [SKYACTIV-G 2.0, SKYACTIV-G 2.5]) (ดูที่ การตั้งค่า PCM (โดยใช้ข้อมูลต้นฉบับ) [SKYACTIV-G 2.0, SKYACTIV-G 2.5])
