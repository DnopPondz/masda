การถอด

1.  ปลดสายขั้วลบแบตเตอรี่

2. ถอดชิ้นส่วนต่อไปนี้:

(1)  ใบปัดและก้านปัดน้ำฝนกระจกบังลมหน้า (ดูที่ การถอด / การติดตั้งใบปัดและก้านปัดน้ำฝนกระจกบังลมหน้า)
2.  แผงปิดช่องลมใต้กระจก (ดูที่ การถอด / การติดตั้งแผงปิดช่องลมใต้กระจก)
(3)  มอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้า (ดูที่ การถอด / การติดตั้งก้านต่อและมอเตอร์ที่ปัดน้ำฝนกระจกบังลมหน้า)
(4)  แผ่นกันรอยขีดข่วนด้านหน้า (ดูที่ การถอด / การติดตั้งแผ่นกันรอยขีดข่วนด้านหน้า)
(5)  แผงปิดด้านข้างด้านหน้า (ดูที่ การถอด / การติดตั้งแผงปิดด้านข้างด้านหน้า)
(6)  ช่องเก็บของ (ดูที่ การถอด / การติดตั้งช่องเก็บของ)
(7)  ฝาครอบใต้แผงหน้าปัด (ดูที่ การถอด / การติดตั้งฝาครอบใต้แผงหน้าปัด)
(8)  แผงตกแต่ง (ดูที่ การถอด / การติดตั้งแผงตกแต่ง)
(9)  แผงตรงกลาง (ดูที่ การถอด / การติดตั้งแผงตรงกลาง)
(10)  ชุดเครื่องเสียง (รุ่นที่มีระบบเครื่องเสียง) (ดูที่ การถอด / การติดตั้งชุดเครื่องเสียง)
(11)  กล่องคอนโซลหน้า (ดูที่ การถอด / การติดตั้งกล่องคอนโซลหน้า)
(12)  หัวคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R]
(13)  แผงคันเกียร์ (ดูที่ การถอด / การติดตั้งแผงคันเกียร์)
(14)  แผงด้านบน (ดูที่ การถอด / การติดตั้งแผงด้านบน)
(15)  คอนโซลหลัง (ดูที่ การถอด / การติดตั้งคอนโซลด้านหลัง)
(16)  ผนังด้านข้าง (ดูที่ การถอด / การติดตั้งผนังด้านข้าง)
(17)  คอนโซลหน้า (ดูที่ การถอด / การติดตั้งคอนโซลด้านหน้า)
(18)  ส่วนประกอบคันเกียร์ (MTX) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [C66M-R, C66MX-R]) (ดูที่ การถอด / การติดตั้งกลไกการเปลี่ยนเกียร์ธรรมดา [D66M-R, D66MX-R]
(19)  ส่วนประกอบคันเลือกเกียร์ (ATX) (ดูที่ การถอด / การติดตั้งไกลไกการเปลี่ยนเกียร์อัตโนมัติ)
(20)  ตัวปลดฝากระโปรง (ดูที่ การถอด / การติดตั้งล็อคฝากระโปรงหน้าและตัวปลดฝากระโปรง)
(21)  แผงด้านล่าง (ดูที่ การถอด / การติดตั้งแผงด้านล่าง)
(22)  ชุดระบบนำทางของรถ (ดูที่ การถอด / การติดตั้งชุดระบบนำทางของรถ)
(23)  ชุดควบคุมอุณหภูมิ (ดูที่ การถอด / การติดตั้งชุดควบคุมอุณหภูมิ [ระบบปรับอากาศแบบอัตโนมัติ]) (ดูที่ การถอด / การติดตั้งชุดควบคุมอุณหภูมิ [ระบบปรับอากาศแบบธรรมดา])
(24)  ฝาครอบคอพวงมาลัย (ดูที่ การถอด / การติดตั้งฝาครอบคอพวงมาลัย)
(25)  โมดูลถุงลมนิรภัยด้านคนขับ (ดูที่ การถอด / การติดตั้งโมดูลถุงลมนิรภัยด้านคนขับ)
26)  พวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)
27)  เพลาแกนพวงมาลัย (ดูที่ การถอด / การติดตั้งคอพวงมาลัยและพวงมาลัย)
(28)  แผงสวิตช์ (ดูที่ การถอด / การติดตั้งแผงสวิตช์)
(29)  ซุ้มมาตรวัด (ดูที่ การถอด / การติดตั้งซุ้มมาตรวัด)
(30)  แผงหน้าปัด (ดูที่ การถอด / การติดตั้งแผงหน้าปัด)
(31)  แผ่นปิดเสา A (ดูที่ การถอด / การติดตั้งแผ่นปิดเสา A)

การติดตั้ง

1.  ใช้มือหมุนเพลทนัตตามทิศทางของลูกศรที่แสดงในรูปภาพจนกว่าจะเข้าที่

ข้อควรระวัง
•  หลังจากถอดแผงหน้าปัด เพลทนัตของจุดเชื่อมคานขวางจะยกขึ้น หากถอดแผงหน้าปัดออกโดยที่เพลทนัตยกขึ้น เพลทนัตอาจสัมผัสถูกตัวถังรถ ซึ่งอาจทำให้ฟิลม์เคลือบงอหรือเสียหายได้ และการติดตั้งแผงหน้าปัดจะทำได้ยากขึ้น ก่อนที่จะใส่แผงหน้าปัดลงในห้องโดยสาร ให้ปรับเพลทนัตของจุดเชื่อมต่อคานขวาง
•  หากขันเพลทนัตแน่นจนสุดขณะติดตั้งโบลท์ F และ G เพลทนัตจะขยับไม่ได้ ซึ่งจะทำให้แถบยึดจุดเชื่อมต่อคานขวางและตัวถังของรถเสียหายได้ หลังจากติดตั้งเพลทนัตเข้าที่แล้ว ให้ตรวจสอบว่าสามารถใช้มือหมุนได้หรือไม่
หมายเหตุ
•  หากเพลทนัตไม่สามารถใช้มือหมุนได้ ให้คลายเพลทนัตโดยใช้ประแจหกเหลี่ยม

2.  ใส่แผงหน้าปัดจากประตูหน้าด้านคนขับหรือประตูหน้าด้านผู้โดยสารที่เปิดอยู่

ข้อควรระวัง
•  หากแถบยึดจุดเชื่อมต่อคานขวางหลุดออกจากแผงหน้าปัดขณะใส่แผงหน้าปัดเข้าไปในห้องโดยสาร จุดเชื่อมต่อคานขวางอาจเสียหายได้ ก่อนที่จะใส่แผงหน้าปัดเข้าไปในห้องโดยสาร ให้ตรวจสอบว่าแถบยึดจุดเชื่อมต่อคานขวางติดอยู่กับแผงหน้าปัดดีแล้ว
