คำเตือน
•  ควรสวมแว่นป้องกันทุกครั้งที่ใช้อากาศอัด มิฉะนั้น อนุภาคสิ่งสกปรกที่ฟุ้งกระจายจากอากาศอัดอาจเข้าตา

ข้อควรระวังในการซ่อมบำรุง


การถอดแยก

1.  ทำความสะอาดด้านนอกของเกียร์โดยใช้ไอน้ำและ/หรือสารขจัดจาระบีก่อนถอดแยก

2.  ในการป้องกันไม่ให้ฝุ่นเข้าสู่เกียร์ ให้ดำเนินการขั้นตอนในสภาพแวดล้อมที่ปราศจากฝุ่น

3.  ทำความสะอาดชิ้นส่วนที่ถอดแยกโดยใช้สารขจัดจาระบีและทำให้แห้งโดยใช้อากาศอัด

4.  ทำความสะอาดรูน้ำมันและร่องน้ำมันโดยใช้อากาศอัดและตรวจสอบว่าไม่มีการอุดตัน

5.  สวมแว่นนิรภัยทุกครั้งเมื่อใช้อากาศอัด

6.  ใช้ค้อนพลาสติกเมื่อถอดแยกชิ้นส่วนสำหรับต่อของส่วนที่เป็นอัลลอยแบบเบาของเสื้อเกียร์

7.  จัดเรียงและจัดเก็บชิ้นส่วนที่ถอดแยกเพื่อปกป้องให้พ้นจากสิ่งสกปรก

8.  เมื่อใช้ปากกาจับชิ้นงาน ต้องยึดชิ้นส่วนประกอบให้แน่นทุกครั้งโดยใช้แผ่นอะลูมิเนียมเพื่อป้องกันไม่ให้ชิ้นส่วนประกอบได้รับความเสียหาย

9.  ถอดแยกพร้อมกับตรวจสอบความเสียหาย, รอยแตก, การเสียรูป, รอยขีดข่วนและลักษณะการประกอบของแต่ละชิ้นส่วน


การทำความสะอาด

1.  ล้างด้านนอกของเกียร์ให้สะอาดหมดจดโดยใช้ไอน้ำหรือสารขจัดจาระบีก่อนถอดแยก

2.  ทำความสะอาดชิ้นส่วนที่ถอดออกโดยใช้สารขจัดจาระบีและทำให้แห้งโดยใช้อากาศอัด ทำความสะอาดรูน้ำมันและช่องทางน้ำมันโดยใช้อากาศอัด


การประกอบ

1.  ก่อนการประกอบ ให้ตรวจสอบว่าแต่ละชิ้นส่วนสะอาดและแห้งแล้ว

2.  ทาน้ำมันที่ระบุ (SAE 75W-80, API service GL-4) อย่างเพียงพอบนชิ้นส่วนที่เคลื่อนที่และพื้นผิวที่เลื่อนได้ขณะทำการประกอบชิ้นส่วน

3.  สำหรับชิ้นส่วนต่างๆ เช่น ปะเก็น สปริง หรือสลักซึ่งกำหนดให้มีการเปลี่ยนทุกครั้งที่ถอดแยก ให้ใช้ชิ้นส่วนใหม่เสมอ

4.  เช็ดซีลแลนท์ซิลิโคนที่ค้างอยู่ออกให้หมดก่อนทาซีลแลนท์ซิลิโคนใหม่

5.  หลังจากทาซีลแลนท์ซิลิโคนแล้ว ให้ติดตั้งเสื้อเกียร์ก่อนที่ซีลแลนท์ซิลิโคนที่ทาลงไปจะเริ่มแข็งตัว จากนั้นรอเป็นเวลา 30 นาทีหรือมากกว่านั้นก่อนเติมน้ำมันหล่อลื่นลงไป

6.  เมื่อใช้ปากกาจับชิ้นงาน ให้ยึดส่วนประกอบโดยใช้เพลทอะลูมิเนียมเสมอ เพื่อป้องกันไม่ให้ส่วนประกอบเสียหาย
